## 📝 Summary

OpenVerb is an open-source project that provides offline voice dictation capabilities for macOS, leveraging a single model to improve efficiency. This innovative solution eliminates the need for two separate models typically required for dictation, making it a more streamlined and lightweight option. By utilizing a unified model, OpenVerb offers a more seamless voice-to-text experience.
