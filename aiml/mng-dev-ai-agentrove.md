## 📝 Summary

Here's a brief summary for the project:

Agentrove is an open-source platform that enables users to host Claude Code and Codex, AI-powered tools, on their own devices. This self-hosted application offers a comprehensive workspace with features like chat, editor, terminal, and version control tools, allowing users to work independently and flexibly. With customizable agent workflows and sandboxed environments, Agentrove offers a secure and adaptable solution for AI development and collaboration.
