## 📝 Summary

RSDKModding’s **RSDKv3-Decompilation** is a comprehensive reverse‑engineering project that reconstructs the source code of the 2011 Sonic CD remake and the Retro Engine (version 3) it runs on. By providing a full decompiled codebase, the repository enables developers and modders to study, modify, and extend the game’s mechanics, rendering pipeline, and asset handling with a modern, readable C++/C# structure.
