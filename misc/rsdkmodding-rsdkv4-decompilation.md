## 📝 Summary

### English

RSDKModding/RSDKv4-Decompilation is an open‑source project that fully decompiles the Retro Engine version 4 used in the 2013 releases of Sonic 1 and Sonic 2. It reconstructs the original game code into readable C/C++ source, enabling developers and modders to study, modify, and extend the games while preserving the original engine’s functionality.
