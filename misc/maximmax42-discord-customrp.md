## 📝 Summary

Here's a brief summary for the Discord-CustomRP project:

Discord-CustomRP is an open-source project that allows users to customize their Discord Rich Presence, providing a simple way to manage playing status. This project supports Windows 7 and above, with features including automatic startup and a tray icon for easy access. It also offers multi-language support, making it accessible to users worldwide.
