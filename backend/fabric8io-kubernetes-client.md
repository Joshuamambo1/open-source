## 📝 Summary

fabric8io/kubernetes-client is an open‑source Java library that provides a fluent, type‑safe API for interacting with Kubernetes and OpenShift clusters. It abstracts the raw REST endpoints into idiomatic Java objects, enabling developers to programmatically manage resources, deploy applications, and automate DevOps workflows directly from Java code. The project is widely used for building backend services, CI/CD tools, and infrastructure automation within the cloud‑native ecosystem.
