## 📝 Summary

Here's a brief summary for the 0xdea/rhabdomancer project:

Rhabdomancer is an open-source vulnerability research assistant that scans binary files to identify calls to potentially insecure API functions. This tool helps security researchers and developers locate and address potential vulnerabilities in software applications. By automating the process of tracking API usage, Rhabdomancer streamlines the vulnerability discovery and mitigation process.
