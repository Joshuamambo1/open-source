## 📝 Summary

Edconv is an open-source, user-friendly graphical user interface (GUI) for FFmpeg, a powerful video and audio processing tool. This project provides an accessible and intuitive way to manipulate and convert multimedia files, making it easier for users to perform complex tasks without the need for in-depth technical knowledge. Edconv streamlines the FFmpeg experience, offering a modern and user-friendly interface for a wide range of multimedia tasks.
