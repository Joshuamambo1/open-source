## 📝 Summary

This project, gfazioli/octoscope, is an open-source terminal dashboard for GitHub that provides a real-time overview of key metrics, including followers, stars, pull requests, and issues. The dashboard automatically refreshes, ensuring users stay up-to-date on their GitHub account activity without needing to constantly reload the page. It's a handy tool for developers and GitHub users who want to quickly monitor their account's performance.
