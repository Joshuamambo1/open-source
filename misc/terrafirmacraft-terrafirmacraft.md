## 📝 Summary

TerraFirmaCraft is an open-source project that aims to reimagine Minecraft's Survival Mode with a focus on realism and authenticity. By introducing more realistic mechanics, such as hunger and thirst systems, and removing some of the game's more forgiving aspects, TerraFirmaCraft provides a more challenging and immersive survival experience. This project is perfect for players who crave a more authentic Minecraft experience.
