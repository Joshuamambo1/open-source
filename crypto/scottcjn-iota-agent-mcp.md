## 📝 Summary

The iota-agent-mcp project is an open-source implementation of a MCP (Model Context Protocol) server for the IOTA blockchain. This server provides 20 tools for seamless integration with AI agents, enabling features such as wallet management, Move CLI capabilities, and on-chain query functionality. By leveraging MCP, developers can unlock advanced AI-driven applications on the IOTA network.
