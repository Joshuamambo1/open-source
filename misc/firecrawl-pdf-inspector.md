## 📝 Summary

The firecrawl/pdf-inspector library is an open-source, high-performance Rust tool designed for inspecting, classifying, and extracting text from PDF files. It utilizes advanced algorithms to differentiate between scanned and text-based PDFs, empowering developers to make informed routing decisions. This feature-rich library streamlines PDF analysis tasks, making it an ideal solution for various applications.
