## 📝 Summary

The 'lazytrivy' project is an open-source tool that simplifies vulnerability scanning by automating the process, making it easier for developers to identify potential security risks in their code. Developed by Owen Rumney, this project leverages the Trivy vulnerability scanner and aims to streamline the scanning process, saving time and effort. It is designed to seamlessly integrate with development workflows, enhancing security and reducing the risk of vulnerabilities.
