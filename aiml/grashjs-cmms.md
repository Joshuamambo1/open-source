## 📝 Summary

Grashjs / cmms is a free, open‑source Computerized Maintenance Management System (CMMS) that you can host yourself for both web and mobile use. It provides a full‑featured platform for enterprises to schedule, track, and manage maintenance activities, assets, and work orders across teams. Built with modern frontend and mobile technologies, the project also incorporates AI/ML components to enhance predictive maintenance and workflow automation.
