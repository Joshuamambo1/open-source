## 📝 Summary

Here's a brief summary of the project:

"Open-codesign is an open-source, alternative to Claude Design that enables users to import their Claude Code or Codex API key with just one click. This platform supports multiple models, including Claude, GPT, Gemini, Kimi, GLM, and Ollama, allowing users to create prototype designs, slides, and PDFs based on their prompts. The project prioritizes user data control through a BYOK (bring your own key) and local-first approach, adhering to an MIT license."
