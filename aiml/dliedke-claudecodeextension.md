## 📝 Summary

Here's a brief summary of the open-source project:

ClaudeCodeExtension is a Visual Studio .NET extension that enhances productivity by providing an intuitive interface for various AI-powered coding tools, including Claude Code CLI, OpenAI Codex, and more. It enables users to work with multi-line prompts and attachments, such as images and files, for seamless coding experiences. This extension is designed to simplify the workflow for developers using AI-driven coding assistants.
