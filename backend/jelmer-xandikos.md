## 📝 Summary

Xandikos is a CalDAV/CardDAV server built upon Git, providing users with a scalable and version-controlled solution for managing calendar and contact data. This open-source project enables users to store and synchronize their calendar and contact information across multiple devices while maintaining a transparent and auditable Git history. By leveraging Git, Xandikos offers robust version control and collaboration features for shared calendars and address books.
