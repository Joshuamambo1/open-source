## 📝 Summary

Ant Design CLI is an open-source command-line tool that provides offline access to Ant Design component knowledge, project analysis, and migration guidance. It empowers developers to query component information, analyze project usage, and navigate Ant Design upgrades without requiring an internet connection. This tool streamlines the development process by offering a comprehensive set of features at the command line.
