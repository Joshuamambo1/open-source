## 📝 Summary

Here's a brief summary for the lance0/ttl project:

ttl is a fast and modern traceroute tool that provides real-time graphical user interface (TUI) output, including per-hop statistics, ASN/geo lookup, and ECMP detection. It also supports MPLS label parsing, making it a comprehensive alternative to traditional traceroute and mtr tools. With its open-source nature, ttl offers a customizable and extensible solution for network diagnostic and troubleshooting tasks.
