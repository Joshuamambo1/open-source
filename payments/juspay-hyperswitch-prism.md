## 📝 Summary

Hyperswitch-Prism is an open-source library that provides a unified interface for integrating multiple payment processors, simplifying the payment processing workflow and enabling easy scalability to support various payment gateways. With this library, developers can write few lines of code to connect to multiple payment processors, streamlining their payment processing infrastructure. This library is ideal for applications requiring seamless payment integration across various payment gateways.
