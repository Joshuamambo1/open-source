**项目简介（2‑3 句话）**  
SoloDawn（huanchong-99/SoloDawn）是基于 GitCortex 的 AI 编码平台，能够从社交媒体对话中直接生成可投产的软件代码。它通过多模态 AI 编码代理（Claude Code、Gemini CLI、Codex 等）并行工作，配备三层质量门控，实现“无需 IDE、无需终端、全自动”的代码交付。

**价值**  
- **快速赋能 AI 能力**：无需从零搭建模型堆栈，即可在现有项目中嵌入对话式代码生成、RAG（检索增强生成）或多代理工作流。  
- **降低研发成本**：自动化解决歧义、架构设计和代码实现，显著缩短原型迭代周期。  
- **质量保障**：三层质量门（语义校验、单元测试、静态分析）在生成代码前即进行过滤，提升交付可靠性。

**典型接入方式**  
1. **阅读 README 并完成依赖安装**（Rust 环境、对应 AI 提供商的 API Key）。  
2. **在项目中引入 SoloDawn SDK**，通过简洁的配置文件声明要使用的 AI 代理和质量门策略。  
3. **编写触发脚本或 webhook**，将社交媒体对话（如 Slack、Twitter）发送给 SoloDawn，平台自动返回经过质量门审查的代码并可直接提交至 Git。  
4. **先做小范围 PoC**：在内部测试分支上跑一次完整流程，验证生成代码的质量和安全性后再推广到主线。
