## 📝 Summary

### English

ssh‑vault is an open‑source command‑line tool that lets you encrypt and decrypt files or arbitrary data using the public keys of SSH identities. By leveraging existing SSH key pairs, it provides a simple, password‑less way to secure secrets and share encrypted payloads with any user who possesses the corresponding private key. The project is lightweight, written in Go, and targets security‑focused workflows where SSH keys are already in use.
