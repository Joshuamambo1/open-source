## 📝 Summary

Testssl.sh is an open-source tool that allows users to check a server's service on any port for the support of various TLS ciphers. This command-line interface tool provides detailed information on the server's TLS configuration, enabling developers and security professionals to identify potential vulnerabilities and strengthen their server's security posture. By running testssl.sh, users can ensure the secure exchange of data between their servers and clients.
