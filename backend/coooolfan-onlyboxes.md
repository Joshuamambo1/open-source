## 📝 Summary

### English

Coooolfan/onlyboxes is an open‑source code execution sandbox platform designed for individuals and small teams, providing a secure, isolated environment to run arbitrary code snippets across multiple programming languages. Built with a lightweight backend and integrated with a Minimal Container Platform (MCP), it streamlines setup, resource management, and permission controls, making sandboxing accessible without heavyweight infrastructure.
