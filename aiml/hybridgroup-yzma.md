## 📝 Summary

Here's a 2-3 sentence summary for the open-source project:

Yzma is an open-source project that enables Go applications to leverage local inference using hardware acceleration by directly integrating with llama.cpp. This integration allows developers to harness the power of LLaMA, a large language model, within their Go applications. By doing so, Yzma empowers users to utilize AI-driven intelligence in a more efficient and localized manner.
