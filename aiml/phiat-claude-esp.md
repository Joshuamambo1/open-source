## 📝 Summary

Here is a brief summary for the open-source project phiat/claude-esp:

phiat/claude-esp is an open-source project that allows real-time streaming of Claude Code's hidden output to a separate terminal in real-time. This includes thinking processes, tool calls, and subagents, providing users with a more transparent and interactive experience. By enabling this feature, developers can gain a deeper understanding of how Claude Code operates and make more informed decisions.
