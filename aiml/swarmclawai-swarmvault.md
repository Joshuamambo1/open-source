## 📝 Summary

Swarmvault is an open-source, local-first knowledge base compiler that utilizes a Reinforcement Augmented Generator (RAG) to create a persistent markdown wiki, knowledge graph, and hybrid search system. Inspired by Karpathy's LLM Wiki, this project aims to turn raw research into a dynamic and compound knowledge base over time. It is designed to work with multiple MCP servers, including Claude Code, Codex, OpenCode, and OpenClaw.
