## 📝 Summary

Linguist is an open-source language detection tool that helps accurately identify the programming languages used in a repository. If you've encountered incorrect language reporting in your GitHub repository, this project invites you to contribute by submitting a pull request to improve its language detection capabilities. By doing so, you can help enhance the accuracy of language identification for the open-source community.
