## 📝 Summary

The ai-rules project is an open-source governance framework that aims to mitigate "Architectural Decay" in AI-driven development by enforcing project boundaries, UI libraries, and design patterns. This framework is designed to work with AI agents such as Cursor, Windsurf, and Copilot, ensuring that they respect project requirements. By implementing ai-rules, developers can maintain a cohesive and well-structured architecture in their AI-driven projects.
