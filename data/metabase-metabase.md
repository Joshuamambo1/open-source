**价值**

2. **数据源连接**：在 Metabase UI 中添加数据库（PostgreSQL、MySQL、ClickHouse、BigQuery 等）或通过 JDBC/ODBC 连接；也可使用 API 将嵌入式报表集成到内部系统。  
3. **小规模验证**：先在测试环境中创建一个简单的仪表盘或查询，确认权限、网络和查询性能，然后逐步扩展到完整的分析管线。
