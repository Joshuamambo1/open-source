## 📝 Summary

Here's a brief summary of the project:

The thesis-skills repository is an open-source project designed to facilitate a deterministic workflow for academic papers, focusing on Word to LaTeX conversion, citation integrity, and compliance with Chinese language standards. It also includes tools for formatting validation and identifying specific university rules and guidelines. This project aims to streamline the writing process for academic papers in Chinese.
