## 📝 Summary

Tootik is an open-source, federated social network that allows users to connect with others across different platforms in a text-based interface. Built with a Gemini frontend, Tootik provides a simple and accessible way to share ideas, thoughts, and experiences with others. This project combines frontend and backend functionality to create a decentralized community where users can engage in discussions and share content freely.
