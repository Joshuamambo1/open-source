## 📝 Summary

The ytsub project is an open-source, text-based YouTube client that allows users to interact with the platform in a subscription-only mode, providing an alternative to traditional YouTube functionality. This project focuses on front-end development and utilizes a terminal user interface (TUI) to enhance user experience. By utilizing a subscription-based model, ytsub offers users a unique and exclusive way to engage with YouTube content.
