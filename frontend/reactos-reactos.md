## 📝 Summary

### English

ReactOS is an open‑source operating system designed to be binary compatible with Microsoft Windows, allowing it to run Windows applications and drivers natively. It re‑creates the Windows architecture from the ground up, providing a free alternative for both desktop and server environments. The project includes both frontend components (user interface and shell) and backend subsystems (kernel, drivers, and system libraries).

### Русский

ReactOS — это открытая операционная система, совместимая с Windows, позволяющая запускать привычные Windows‑приложения и драйверы без лицензий Microsoft. Проект объединяет как фронтенд‑компоненты (графический интерфейс, пользовательский опыт), так и бекенд‑части (ядро, системные службы), предоставляя полностью свободную альтернативу Windows. Благодаря активному сообществу разработчиков ReactOS постоянно развивается и улучшает совместимость с программным обеспечением Windows.
