1lab is an open-source project that provides a comprehensive, cross-linked reference resource for mathematics in Homotopy Type Theory. This project formalizes mathematical concepts and structures within the framework of Homotopy Type Theory, allowing for a deeper understanding and exploration of the subject. It aims to serve as a valuable resource for researchers and students working in this area.
