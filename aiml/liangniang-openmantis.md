## 📝 Summary

Here's a brief summary for the open-source project:

OpenMantis is a lightweight, multi-platform AI agent chat framework that enables the connection of multiple Large Language Models (LLMs) to popular chat platforms such as Feishu, WeCom, and QQ. This framework provides a suite of composable tools, including long-term memory, scheduling, and browser automation, making it an ideal solution for building robust and intelligent chatbots. By leveraging OpenMantis, developers can create sophisticated AI-powered chat experiences with ease.
