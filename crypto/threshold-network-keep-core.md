The threshold-network/keep-core project is an open-source initiative that provides the core smart contracts and reference client for the Threshold Network. This decentralized network aims to enable secure, private, and scalable transactions through the use of advanced cryptography and threshold signatures. By open-sourcing the core components, the project facilitates community-driven development and experimentation with the Threshold Network's innovative technology.
