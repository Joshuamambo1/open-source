## 📝 Summary

Here's a brief summary of the project:

Claude Code Organizer is an open-source dashboard that simplifies the management of Claude Code memories, configurations, and MCP servers. Its features include a security scanner to detect tool poisoning, a context token budget tracker, duplicate cleanup functionality, and scope management capabilities. This tool aims to streamline the development process for AI/ML and Backend projects, ensuring efficient and secure operations.
