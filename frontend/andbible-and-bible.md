## 📝 Summary

### English

AndBible is an open‑source Android application designed for personal Bible study, offering a clean, responsive frontend optimized for mobile devices. It provides features such as multiple Bible translations, parallel viewing, bookmarking, note‑taking, and search, all with offline access and a customizable reading experience.
