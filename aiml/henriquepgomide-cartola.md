## 📝 Summary

Here's a brief summary for the open-source project:

The henriquepgomide/caRtola project is an open-source repository that extracts data from the CartolaFC API, performs exploratory data analysis, and trains predictive models using R and Python. Focused on the period from 2014 to 2022, the project aims to provide insights into the popular Brazilian fantasy football game. Its codebase serves as a learning resource for data munging, analysis, and machine learning techniques.
