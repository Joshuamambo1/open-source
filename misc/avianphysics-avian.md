## 📝 Summary

The avian physics engine is an open-source, ECS-driven (Entity-Component-System) physics engine designed for the Bevy game engine, supporting both 2D and 3D physics simulations. This project aims to provide a powerful and flexible physics solution for game developers, leveraging the Bevy ecosystem. By utilizing the ECS architecture, avian promises high-performance and efficient physics simulations.
