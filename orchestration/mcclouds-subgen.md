## 📝 Summary

McCloudS/subgen is an open‑source automation tool that leverages the OpenAI Whisper model to generate subtitles for media libraries managed by Jellyfin, Plex, Emby, Tautulli, or Bazarr. By integrating directly with these platforms, it orchestrates AI‑driven transcription workflows, delivering accurate, language‑agnostic subtitles without manual effort. The project combines media orchestration with modern machine‑learning capabilities for seamless, automated subtitle creation.
