## 📝 Summary

This open-source project provides a hands-on workshop for building a multi-agent AI system from scratch. It includes code, slides, and video resources to guide users through the process of creating a Deep Research Agent and a Writing Workflow served as MCP (Multi-Agent Communication Protocol) servers. The project enables users to learn about AI orchestration and multi-agent systems through a practical, step-by-step approach.
