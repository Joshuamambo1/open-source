## 📝 Summary

SimBA (Simple Behavioral Analysis) is an open-source pipeline and graphical user interface (GUI) designed for developing supervised behavioral classifiers. This project enables researchers to streamline their analysis workflow, simplifying the process of training and evaluating machine learning models for behavioral data. By offering a user-friendly interface, SimBA aims to make behavioral analysis more accessible and efficient.
