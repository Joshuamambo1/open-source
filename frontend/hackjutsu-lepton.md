**价值**  
- **提升开发效率**：通过统一管理、复用 UI 代码片段，显著缩短产品界面的开发周期。  
- **降低维护成本**：组件化的片段库让 UI 变更可以一次更新、全局生效，避免重复劳动。  
- **跨平台一致性**：同一套片段可在 macOS、Windows 与 Linux 环境下无缝使用，保证团队协作的统一性。

**典型接入方式**  
1. **API / SDK**：项目提供 JavaScript SDK，直接在前端项目中 `import` 相应函数，即可调用片段管理的增删改查接口。  
2. **CLI**：通过全局安装的 `lepton` 命令行工具，可在本地或 CI/CD 流程中同步、发布、搜索片段。  
3. **语言元数据**：Lepton 会为每个片段生成语言/框架元数据（如 React、Vue、Svelte），便于 IDE 自动补全和文档生成。
