## 📝 Summary

The iShape-Rust/iOverlay project provides a set of open-source tools for performing boolean operations on 2D polygons. It supports various polygon operations, including intersection, union, difference, and XOR, as well as self-intersections, making it a valuable resource for developers working with geometric shapes. This library is designed to be versatile and adaptable for use in a wide range of development projects.
