## 📝 Summary

Here's a brief summary for the open-source project "rigaya/NVEnc":

This project, "NVEnc", is an open-source performance experiment focused on evaluating the high-speed encoding capabilities of NVENC. It allows users to test and compare the encoding performance of NVENC under various conditions. By providing a platform for performance benchmarking, the project aims to help developers optimize and improve NVENC encoding efficiency.
