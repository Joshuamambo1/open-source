## 📝 Summary

Y2Z/monolith is an open-source command-line interface (CLI) tool and library that enables users to save complete web pages as a single HTML file. This allows developers to easily capture and preserve the structure, content, and visual appearance of web pages for further analysis, testing, or archival purposes. By providing a monolithic representation of web pages, Y2Z/monolith streamlines the process of web page preservation and inspection.
