## 📝 Summary

djipco/webmidi is an open‑source library that simplifies interaction with the Web MIDI API, providing intuitive functions such as `playNote`, `sendPitchBend`, and others for sending MIDI messages. It also offers straightforward event listeners (e.g., `noteon`, `pitchbend`, `controlchange`) to handle incoming MIDI data, making it easy to build both frontend and backend applications that control musical instruments.
