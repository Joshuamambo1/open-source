## 📝 Summary

The complgen project allows users to generate declarative bash, fish, zsh, and pwsh completions without writing custom shell scripts. This open-source tool streamlines completion definition, making it easier to integrate with existing projects and simplify user interactions. By leveraging declarative configurations, complgen reduces the complexity of shell completion setup.
