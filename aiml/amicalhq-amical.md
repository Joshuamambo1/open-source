## 📝 Summary

The amical project is an open-source AI-powered dictation app that enables users to type up to three times faster without needing a keyboard. This local-first application operates offline, utilizing open-source models for fast and accurate results. It is designed to be a free and reliable alternative for users seeking efficient and keyboard-free text input solutions.
