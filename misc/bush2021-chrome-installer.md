## 📝 Summary

This open-source project, "chrome_installer", utilizes GitHub Actions to automatically fetch the Google Chrome offline installer. This convenient tool simplifies the process of obtaining the offline installer, allowing users to easily download and install Chrome without relying on online installers. The project is designed to provide a hassle-free experience for users looking to install Chrome without an active internet connection.
