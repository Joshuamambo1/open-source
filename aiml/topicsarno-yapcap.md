## 📝 Summary

Here's a brief summary for the yapcap project:

yapcap is an open-source, native Linux application that provides real-time monitoring of local usage limits for Codex, Claude Code, and Cursor in the COSMIC panel. This project aims to offer a user-friendly interface for managing AI/ML-related resources on Linux systems. By using yapcap, users can easily track and optimize their usage limits for improved performance.
