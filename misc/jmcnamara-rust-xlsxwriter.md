## 📝 Summary

### English

The **rust_xlsxwriter** library provides a pure‑Rust API for generating Microsoft Excel XLSX files without requiring any external dependencies. It supports creating worksheets, writing data types, formatting cells, adding formulas, charts, and other common spreadsheet features, making it a lightweight solution for Rust applications that need to produce Excel reports programmatically.
