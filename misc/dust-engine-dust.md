## 📝 Summary

The Dust project is an open-source research initiative that explores the frontiers of voxel geometry and interactive experiences. Built on a foundation of Rust, Vulkan, and Bevy, it features an experimental real-time global illumination renderer, allowing for cutting-edge visualizations and simulations. This project aims to push the boundaries of what is possible in real-time rendering and voxel-based interactions.
