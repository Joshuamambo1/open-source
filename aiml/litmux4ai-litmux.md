## 📝 Summary

The litmux project is an open-source framework for unit testing artificial intelligence (AI) models. It enables users to create and compare test prompts, helping to optimize model performance and reduce costs associated with AI model development. By providing a standardized testing environment, litmux facilitates the evaluation and improvement of AI models.
