## 📝 Summary

### English

Luxtorpeda is an open‑source Steam Play compatibility layer that enables Windows games to run on Linux by leveraging their native Linux engines when available, rather than relying on full emulation or translation. It automatically detects and launches the Linux version of a game, providing smoother performance and better integration with the Steam ecosystem.

### Русский

Luxtorpeda — это open‑source утилита для Steam Play, позволяющая запускать Windows‑игры на Linux, используя их оригинальные нативные движки вместо эмуляции через Proton. Проект автоматически подбирает и интегрирует подходящий Linux‑движок (например, Unity, Unreal, Godot), обеспечивая более стабильную и производительную работу игр. Подходит для любой платформы, где доступен Steam, и распространяется под лицензией MIT.
