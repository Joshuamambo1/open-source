### English

Overte is an open‑source platform for creating, hosting, and exploring immersive virtual worlds. It provides a web‑based, real‑time 3D environment where users can build interactive spaces, script behaviors, and connect with others across the globe. The project aims to foster a decentralized, customizable metaverse that anyone can freely modify and extend.
