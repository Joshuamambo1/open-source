## 📝 Summary

The deadsy/sdfx project is an open-source, simple CAD (Computer-Aided Design) package that leverages the power of signed distance functions to create and manipulate 3D models. This innovative tool allows users to define and edit complex shapes using mathematical functions, enabling a unique and expressive way of designing and modeling. By streamlining the design process, sdfx empowers users to create sophisticated CAD models with ease and flexibility.
