## 📝 Summary

Here's a 2-3 sentence summary of the open-source project:

The telegram-proxy-collector is an open-source project that automatically collects MTProto proxy servers with a unique filtering system. It separates high-quality "golden" proxies (Fake-TLS for Yandex, VK, and GosUslugi) from regular ones, and updates the list every hour. This project facilitates censorship circumvention in regions with restricted internet access, such as Russia.
