## 📝 Summary

COLMAP is an open‑source library for automatic 3‑D reconstruction from unordered image collections. It implements state‑of‑the‑art Structure‑from‑Motion pipelines and dense Multi‑View Stereo algorithms, providing tools for feature extraction, matching, camera pose estimation, and dense point‑cloud generation. The project is widely used in computer‑vision research and applications that require high‑quality 3‑D models.
