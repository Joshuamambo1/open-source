## 📝 Summary

Rare is an open-source alternative to the Epic Games Launcher, built using the Legendary framework. This project allows users to easily manage their games and access content from the Epic Games Store. By providing a free and customizable solution, Rare aims to offer users a more personal and flexible gaming experience.
