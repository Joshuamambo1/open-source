## 📝 Summary

### English

Cockpit provides a modern, web‑based graphical interface for managing Linux servers, allowing administrators to monitor system performance, configure services, and troubleshoot issues directly from a browser. It combines a lightweight backend with real‑time data visualisation, making server administration accessible without requiring deep command‑line expertise.
