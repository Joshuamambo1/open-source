## 📝 Summary

The **zzet/gortex** project is a high‑performance code‑intelligence engine that parses repositories into an in‑memory knowledge graph, then serves the data through a CLI, MCP server, and web UI. Designed for AI coding agents, it delivers only the essential context—one `get_editing_context` call replaces 5‑10 file reads—reducing token usage by roughly 94%. It spans AI/ML, frontend, and backend components, offering fast, low‑overhead access to repository insights.
