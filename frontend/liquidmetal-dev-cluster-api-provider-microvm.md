## 📝 Summary

Here's a brief summary for the project:

The Cluster API provider for microvm is an open-source project that enables the creation of microvm clusters using Cloud Hypervisor or Firecracker. This provider allows users to manage and orchestrate microvm clusters with ease, making it a valuable tool for DevOps and infrastructure teams. By leveraging the Cluster API, users can deploy and manage microvm clusters in a scalable and automated manner.
