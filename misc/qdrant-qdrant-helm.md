## 📝 Summary

Qdrant/Helm is an open-source project that provides a Helm chart for deploying the Qdrant vector database. This chart simplifies the installation and management of Qdrant on Kubernetes environments, making it easier to integrate this powerful database into various applications. With Qdrant/Helm, users can quickly set up and scale their Qdrant instances.
