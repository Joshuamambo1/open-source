## 📝 Summary

Here's a brief summary for the open-source project:

Opencode-antigravity-auth is an open-source project that enables Opencode to authenticate against Google's Antigravity IDE using OAuth. This integration allows users to leverage Antigravity's rate limits and access advanced models like gemini-3-pro and claude-opus-4-5-thinking using their Google credentials. This project facilitates seamless collaboration between Opencode and Antigravity, empowering developers to unlock the full potential of Antigravity's AI capabilities.
