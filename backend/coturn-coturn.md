## 📝 Summary

Coturn is an open-source TURN (Traversal Using Relays around NAT) server project designed to facilitate communication between peer-to-peer applications by bypassing network address translation (NAT) limitations. It provides a scalable and reliable solution for real-time communication, enabling users to establish connections across different networks and devices. Coturn can be used in various applications, including VoIP, video conferencing, and online gaming.
