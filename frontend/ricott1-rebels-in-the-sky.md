## 📝 Summary

Rebels‑in‑the‑Sky is an open‑source, peer‑to‑peer terminal game where players assume the roles of space‑pirate basketball teams battling across a procedurally generated galaxy. Built with a lightweight frontend, the game synchronizes player actions in real time over a decentralized network, delivering fast, text‑based multiplayer action without a central server. It showcases creative use of terminal graphics and networking libraries for an engaging, cross‑platform gaming experience.
