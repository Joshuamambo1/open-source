## 📝 Summary

Delegua is an open-source programming language fully written in Portuguese, built on top of TypeScript and designed to be system and device independent. This language allows for support of multiple dialects, catering to the needs of Portuguese-speaking developers. Delegua aims to provide a versatile and accessible development experience for users.
