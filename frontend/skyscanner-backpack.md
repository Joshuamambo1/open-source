### English

Skyscanner’s **Backpack** is an open‑source design system and component library built for web applications. It provides a cohesive set of reusable UI elements, style guidelines, and accessibility best practices that enable developers and designers to create consistent, high‑quality interfaces quickly. The project is maintained by Skyscanner and aims to streamline front‑end development while preserving the brand’s visual identity.
