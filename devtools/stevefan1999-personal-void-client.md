## 📝 Summary

This open-source project aims to port the partially deobfuscated version of Old School Runescape to use LWJGL 3, a popular Java game development library. The goal is to create a modernized version of the classic game, utilizing the latest LWJGL 3 features. This project showcases the community's efforts to revitalize a beloved game while advancing its underlying technology.
