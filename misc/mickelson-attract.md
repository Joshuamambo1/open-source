## 📝 Summary

The mickelson/attract project is an open-source, graphical front-end designed to seamlessly integrate with command line emulators, providing a user-friendly interface that shields the underlying operating system from view. This innovative tool is optimized for control via joystick or gamepad, offering a more immersive and engaging experience for gamers and emulator enthusiasts alike. By bridging the gap between emulators and gamepads, attract unlocks a new level of accessibility and enjoyment for retro gaming enthusiasts.
