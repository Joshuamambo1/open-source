tinyauth (tinyauthapp/tinyauth) is a lightweight, open‑source authentication and authorization server designed for simplicity and minimal footprint. It provides essential login, token issuance, and permission checks via a clean, easy‑to‑integrate backend API, making it ideal for microservices, prototypes, or any application that needs a tiny yet secure auth solution.
