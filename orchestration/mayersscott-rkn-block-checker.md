## 📝 Summary

### English

MayersScott/rkn‑block‑checker is an open‑source diagnostic tool that systematically tests connectivity to RKN/TSPU services across each network layer—starting with DNS resolution, then TCP reachability, TLS handshakes, and finally HTTP requests. By pinpointing exactly where a block occurs, it helps developers and network operators quickly identify and troubleshoot censorship or filtering issues. The project is positioned as a lightweight orchestration/DevTools utility for automated, layer‑by‑layer internet block detection.
