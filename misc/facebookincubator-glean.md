## 📝 Summary

Glean is an open-source project that provides a system for collecting, deriving, and working with facts about source code. It offers a unified framework for extracting and processing metadata from software projects, enabling developers to easily integrate and analyze this information. This project aims to standardize the collection and usage of source code facts, making it a valuable tool for software development and maintenance.
