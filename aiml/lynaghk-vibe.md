## 📝 Summary

**vibe** is an open‑source tool that lets you spin up lightweight Linux virtual machines on macOS, providing an isolated sandbox for running and testing large language model (LLM) agents. By automating VM creation, networking, and cleanup, it simplifies development workflows that require a secure, reproducible Linux environment without leaving the macOS host.
