## 📝 Summary

A minimalist web server written entirely in x86‑64 assembly, showcasing low‑level programming techniques while handling HTTP requests, routing, and static file serving. The project serves as both a technical experiment and a personal “meaning‑making” exercise, demonstrating that even complex backend functionality can be built from the ground up without high‑level languages or frameworks. It’s useful for developers interested in operating‑system concepts, performance tuning, and the quirks of writing network code directly in assembly.
