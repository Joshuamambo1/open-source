## 📝 Summary

JsonAsAsset is a robust Unreal Engine plugin that streamlines the import of game assets directly from FModel, converting them into native Unreal assets with minimal effort. It supports a wide range of asset types—including meshes, textures, and materials—while preserving original metadata and ensuring compatibility with UE workflows. This open‑source tool simplifies modding and content creation for developers and hobbyists alike.
