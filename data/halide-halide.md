## 📝 Summary

### English

Halide is a domain‑specific language and accompanying compiler framework designed for high‑performance, portable data‑parallel computation, especially in image processing and computational photography. It separates algorithm definition from scheduling, allowing developers to write clear, mathematically expressive code while fine‑tuning execution strategies for CPUs, GPUs, and other accelerators. The open‑source project provides a robust C++ API, runtime libraries, and tooling to generate optimized native code across diverse hardware platforms.

### Русский

Он отделяет алгоритм от расписания (schedule), позволяя оптимизировать код под разные архитектуры (CPU, GPU, DSP) без изменения самой логики.
