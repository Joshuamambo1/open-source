## 📝 Summary

### English

75lb/handbrake-js is an open‑source Node.js library that provides a simple, programmatic interface to the HandBrake video transcoding engine. It enables developers to encode, convert, and transcode video files directly from JavaScript, handling common options such as format selection, quality settings, and audio/video tracks. The package abstracts the underlying command‑line tool, making video processing tasks easy to integrate into server‑side applications.
