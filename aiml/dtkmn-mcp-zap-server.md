## 📝 Summary

The dtkmn/mcp-zap-server project is an open-source, Spring Boot application that integrates OWASP ZAP with the Model Context Protocol (MCP) framework. This integration enables MCP-compatible AI agents to orchestrate ZAP actions, such as scanning, importing OpenAPI specs, and generating reports. The result is a seamless and automated penetration testing workflow.
