## 📝 Summary

Axolotl is an open-source, AI-powered quality assurance (QA) agent for Visual Studio Code (VS Code). It analyzes code changes, generates test plans, runs real browser tests, and provides evidence-backed merge verdicts to ensure high-quality code updates. This innovative tool streamlines the QA process, leveraging artificial intelligence to optimize testing efficiency and accuracy.
