## 📝 Summary

The timewall project brings dynamic HEIF wallpapers, a feature previously exclusive to Apple devices, to GNU/Linux systems. It provides a way to experience dynamic wallpapers on Linux, offering a seamless and immersive desktop experience. This project empowers users to customize their desktop with visually engaging and interactive wallpapers.
