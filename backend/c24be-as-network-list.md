## 📝 Summary

C24Be/AS_Network_List is an open‑source tool that automatically generates blacklists of IP subnets associated with Russian government agencies. By fetching and processing authoritative routing data, it creates up‑to‑date network lists that can be integrated into firewalls, proxies, or other server‑side security solutions to block traffic from these entities. The project is focused on backend integration and aims to simplify the deployment of region‑specific network blocks.
