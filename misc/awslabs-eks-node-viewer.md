## 📝 Summary

The EKS Node Viewer is an open-source project that allows users to visualize and monitor AWS Elastic Kubernetes Service (EKS) node metrics and logs in a centralized and intuitive manner. This project provides a simple and user-friendly interface for cluster administrators to monitor node performance, diagnose issues, and optimize their EKS cluster resources. By leveraging this tool, users can gain valuable insights into their EKS node operations and make data-driven decisions to improve their cluster's overall health and efficiency.
