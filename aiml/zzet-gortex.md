## 📝 Summary

The zzet/gortex project is an open-source, high-performance code intelligence engine that leverages an in-memory knowledge graph to provide efficient and extensive code insights. This cutting-edge engine enables AI coding agents to access the necessary information via a CLI, MCP Server, or web UI, significantly reducing token usage by up to 94%. By streamlining code intelligence, gortex accelerates the development process and enhances coding experiences.
