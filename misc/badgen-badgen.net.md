## 📝 Summary

badgen/badgen.net is a lightweight, high‑performance service for generating dynamic SVG badges (e.g., build status, version, coverage) on the fly. It provides a simple HTTP API that can be self‑hosted, allowing developers to create custom, cache‑friendly badges without relying on third‑party providers. The project is open‑source, MIT‑licensed, and suited for any web or CI/CD workflow that needs fast, reproducible badge images.
