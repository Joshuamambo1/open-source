## 📝 Summary

prism‑coder is an open‑source, HIPAA‑compliant cognitive platform that runs a 7‑billion‑parameter LLM entirely on‑device, combining Hebbian learning, ACT‑R‑style spreading activation, and persistent memory to create a “mind palace” for AI agents. It supports adversarial evaluation, a multi‑agent Hivemind architecture, and a visual dashboard, all orchestrated without the need for external API keys.
