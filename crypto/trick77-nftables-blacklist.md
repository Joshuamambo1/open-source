## 📝 Summary

### English

**trick77/nftables-blacklist** is a lightweight Bash utility that automates the creation of nftables rules to block extensive IP address lists sourced from public blacklists. By parsing these lists and generating the appropriate nftables set and drop statements, the script enables quick, repeatable deployment of network‑level IP bans—useful for protecting cryptocurrency services and other internet‑facing applications from malicious traffic.
