## 📝 Summary

### English

The **ecki/net-tools** repository provides a collection of classic Linux networking utilities such as `ifconfig`, `netstat`, `route`, `arp`, and related tools. These command‑line programs enable users to configure interfaces, inspect routing tables, monitor network connections, and perform low‑level network diagnostics on Linux systems. The project is maintained as an open‑source, community‑driven alternative to newer networking stacks.
