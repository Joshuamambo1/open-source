## 📝 Summary

Here's a brief summary for the project:

sandboxed.sh is a self-hosted orchestrator designed to manage AI autonomous agents in isolated Linux workspaces. It allows users to run Claude Code and Open Code within secure environments, leveraging a git repository to manage skills, configurations, and encrypted secrets. This open-source project streamlines the process of developing and deploying AI agents with enhanced security and organization.
