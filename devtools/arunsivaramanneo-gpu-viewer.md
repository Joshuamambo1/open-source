## 📝 Summary

GPU-Viewer is an open-source project that provides a user-friendly interface to retrieve detailed information about graphics processing units (GPUs) on Linux systems. It serves as a front-end to existing tools like glxinfo, vulkaninfo, clinfo, and es2_info, presenting their insights in a more accessible format. This project is categorized under DevTools and aims to make GPU-related data more easily understandable for developers and users alike.
