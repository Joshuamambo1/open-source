## 📝 Summary

Here's a brief summary for the open-source project:

OfficeCLI Skills is an open-source project that enables seamless integration of AI agents like Claude Code and Codex with Microsoft Office workflows. It allows users to create and orchestrate PPTX, DOCX, XLSX, and report workflows using OfficeCLI, leveraging the capabilities of AI/ML technology. This project empowers developers to automate and streamline office document creation and management tasks.
