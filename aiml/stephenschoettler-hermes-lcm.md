## 📝 Summary

**hermes‑lcm** is an open‑source, lossless context‑management plugin for the Hermes Agent that builds a directed‑acyclic‑graph (DAG) of conversational context, guaranteeing that no message is ever dropped or overwritten. By persisting each interaction as a node in the DAG, it enables reliable, reproducible state tracking for AI/ML applications that require complete, auditable histories. The plugin integrates seamlessly with Hermes, providing developers a robust foundation for building context‑aware agents and chatbots.
