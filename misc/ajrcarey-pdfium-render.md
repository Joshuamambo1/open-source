## 📝 Summary

This project, ajrcarey/pdfium-render, provides a high-level, idiomatic Rust wrapper around Pdfium, enabling easy integration of PDF rendering capabilities into Rust applications. By leveraging Pdfium, a powerful C++ PDF library used by the Google Chromium project, developers can create robust and feature-rich PDF rendering solutions with minimal overhead. This Rust wrapper streamlines the process of working with Pdfium, making it more accessible and efficient.
