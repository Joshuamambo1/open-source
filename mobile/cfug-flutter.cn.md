## 📝 Summary

### English

Flutter.cn is an open‑source initiative that translates the official Flutter documentation into Chinese, making the framework more accessible to Chinese‑speaking developers. The project provides a structured translation plan and collaboration guidelines through its wiki, allowing contributors to help localize tutorials, API references, and best‑practice guides. It aims to foster a thriving Flutter community in China by keeping the translated content up‑to‑date with the official releases.
