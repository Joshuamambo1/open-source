## 📝 Summary

### English

DmitryTsepelev/ar_lazy_preload is an open‑source Ruby gem that adds transparent lazy‑loading capabilities to ActiveRecord associations, allowing related records to be fetched only when they are actually accessed. By intercepting association calls and deferring database queries, it reduces unnecessary eager loads and improves performance in Rails applications. Although categorized under AI/ML, the library is a general‑purpose tool for optimizing data access in any Ruby on Rails project.
