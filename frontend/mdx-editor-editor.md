## 📝 Summary

mdx‑editor/editor is a lightweight, open‑source React component that provides a rich‑text editing experience for Markdown content. It combines a WYSIWYG interface with live preview, supporting common formatting, code blocks, tables, and extensible plugins while preserving the original MDX syntax. Ideal for frontend applications, it simplifies integrating a full‑featured Markdown editor into any React project.
