## 📝 Summary

DataRecce/recce is an open-source project that provides a data-validation toolkit to enhance the review process of PRs (pull requests) in dbt, a popular data build tool. This toolkit enables users to efficiently validate and verify data transformations, ensuring accuracy and reliability in their data pipelines. By integrating with dbt, recce streamlines data validation, making it easier to catch errors and improve data quality.
