## 📝 Summary

This open-source project utilizes the Z3 SMT (Satisfiability Modulo Theories) solver in WASM (WebAssembly) to deterministically verify AWS costs. By leveraging the power of SMT, the project aims to provide precise and reliable cost calculations for AWS resources. This innovative approach enables developers to trust the accuracy of their AWS cost estimations and make informed decisions about resource allocation.
