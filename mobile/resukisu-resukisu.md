## 📝 Summary

Here's a 2-3 sentence summary for the project:

ReSukiSU is an open-source kernel-based Android root solution that provides a flexible and customizable approach to rooting Android devices. Forked from SukiSU, this project currently supports kernels from Android 3.4 to 6.12, offering a wide range of compatibility options for developers and users alike. By leveraging a kernel-based approach, ReSukiSU aims to enhance the security and functionality of Android devices.
