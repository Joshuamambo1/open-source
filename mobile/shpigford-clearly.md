## 📝 Summary

Here's a brief summary of the Clearly project:

Clearly is an open-source Markdown editor and knowledge base that provides a seamless writing experience on Mac, iOS, and iPadOS devices. This powerful tool allows users to create and manage their content with ease, utilizing Markdown syntax for a distraction-free writing environment. With Clearly, users can focus on crafting their ideas without worrying about formatting.
