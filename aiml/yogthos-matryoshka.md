## 📝 Summary

Matryoshka is an open-source MCP (Master Control Protocol) server designed for efficient analysis of large documents. It utilizes REPL (Read-Eval-Print Loop) state to optimize token usage, making it suitable for applications involving extensive document processing. This project enables the development of scalable and token-efficient AI/ML workflows.
