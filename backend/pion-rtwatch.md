## 📝 Summary

Here's a brief summary of the pion/rtwatch project:

The pion/rtwatch project allows users to watch videos with friends in real-time using WebRTC, eliminating the need for manual pausing and seeking. By delegating these tasks to the server, users can enjoy a seamless viewing experience. This open-source backend solution is designed to facilitate collaborative video watching over the web.
