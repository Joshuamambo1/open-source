## 📝 Summary

TweetClaw (Xquik‑dev/tweetclaw) is an open‑source integration that lets you interact with Twitter—posting tweets, replying, liking, retweeting, following users, sending DMs, and more—via the structured Xquik API. It exposes 99 agent‑callable endpoints, making it easy to automate Twitter actions from both frontend and backend services, and is positioned for AI/ML workflows that need programmatic social‑media control.
