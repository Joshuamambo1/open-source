## 📝 Summary

The agent-rules project is an open-source repository providing rules and instructions for agentic coding tools. It aims to standardize and streamline the interactions between these tools, enabling developers to work more efficiently and effectively. By providing a unified set of rules, agent-rules supports the development of various tools such as Cursor, Claude CLI, Gemini CLI, and more.
