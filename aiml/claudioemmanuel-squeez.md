## 📝 Summary

Here's a summary of the project:

The "squeez" project is an open-source, hook-based token compressor that optimizes performance for five AI CLI hosts, achieving up to 95% bash compression. It features a signature-mode for code reads, cross-call deduplication, and a self-teaching protocol, making it an efficient tool for developers. This lightweight solution has zero runtime dependencies, making it easy to integrate into existing workflows.
