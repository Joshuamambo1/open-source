## 📝 Summary

The skills-manager project is an open-source, lightweight desktop application designed to streamline the management of AI agent skills across multiple coding tools. It allows users to organize, synchronize, and access skills from over 15+ tools, including Cursor, Claude Code, Codex, and Copilot. This tool aims to simplify the workflow of developers working with AI-powered coding assistants.
