## 📝 Summary

Frigate is an open-source NVR (Network Video Recorder) system that combines IP camera footage with local object detection capabilities in real-time. This AI/ML-powered solution allows users to monitor and track objects within their camera feeds, enhancing overall security and surveillance. By integrating object detection with IP cameras, Frigate provides a robust and customizable platform for real-time monitoring and alerts.
