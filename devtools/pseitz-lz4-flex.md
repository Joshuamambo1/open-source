## 📝 Summary

PSeitz/lz4_flex is a pure‑Rust library that provides high‑performance LZ4 compression and decompression, aiming to be the fastest implementation of the algorithm in the Rust ecosystem. It offers a safe, zero‑dependency API compatible with the official LZ4 spec, making it ideal for developers who need fast, memory‑efficient data compression without relying on native code.
