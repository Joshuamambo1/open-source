## 📝 Summary

OpenCoworkAI’s **open‑codesign** is a lightweight, open‑source alternative to Claude Design that lets users generate prototypes, slide decks, or PDFs directly from prompts. With a single click you can import any Claude, Codex, or other LLM API key (Claude, GPT, Gemini, Kimi, GLM, Ollama, etc.), run locally or bring your own model (BYOK), and keep all data on‑device under an MIT license. It provides both frontend and backend components for a seamless, privacy‑first AI‑powered design workflow.
