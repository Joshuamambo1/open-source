## 📝 Summary

Here's a brief summary of the project:

NipaPlay-Reload is an open-source, cross-platform local video player that supports playback on Windows, macOS, Linux, Android, and iOS. It offers advanced features such as live chat support, multi-format subtitle support, multiple audio track switching, and new episode notifications. Developed using Flutter, NipaPlay-Reload provides a unified user experience and seamless integration with Emby/Jellyfin media libraries.
