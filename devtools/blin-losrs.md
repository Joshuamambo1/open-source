## 📝 Summary

Losrs is an open-source project that brings spaced repetition learning to the terminal, utilizing markdown cards for efficient retention of knowledge. This tool allows users to create digital flashcards in markdown format, which are then reviewed at optimized intervals to aid in long-term memorization. By leveraging the terminal, Losrs offers a distraction-free learning experience for users.
