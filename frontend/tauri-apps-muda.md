## 📝 Summary

### English

Muda is a lightweight Rust library that provides a set of cross‑platform menu utilities for desktop applications built with Tauri (or any native GUI). It simplifies creating, updating, and handling native menus—such as context menus, system trays, and keyboard shortcuts—while abstracting away platform‑specific quirks. Ideal for developers who need a clean, type‑safe way to manage application menus in Rust‑based front‑end projects.
