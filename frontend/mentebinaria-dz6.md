## 📝 Summary

Dz6 is a fast, open-source text-based hex editor inspired by Vim, designed to provide an efficient and customizable user interface. This Terminal User Interface (TUI) hex editor aims to offer a seamless editing experience for developers and enthusiasts alike. With its Vim-inspired design, dz6 caters to users familiar with the popular Vim editor.
