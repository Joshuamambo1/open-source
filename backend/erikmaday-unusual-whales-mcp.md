## 📝 Summary

Here's a 2-3 sentence summary for the open-source project:

The `unusual-whales-mcp` project provides an open-source MCP (Market Client Protocol) server for interacting with the Unusual Whales API. It enables access to various features, including options flow, dark pool trades, and congress trades, facilitating the development of applications that leverage these data points. This project offers a flexible and customizable backend solution for integrating with the Unusual Whales API.
