## 📝 Summary

Here's a brief summary of the open-source project:

Nudgy is a native macOS menu bar application that provides real-time notifications about AI coding agents, specifically Claude Code, when they finish tasks, request permissions, or ask questions. This tool prioritizes user privacy, operates without external dependencies, and is limited to local network activity. By providing these notifications, Nudgy streamlines the development process and helps users stay informed about AI coding agent interactions.
