## 📝 Summary

Allsky is an open‑source all‑sky camera system that runs on a Raspberry Pi, enabling users to capture and monitor the entire night sky for astronomical, weather, or environmental observations. The project provides scripts, hardware guides, and software tools to automate image acquisition, processing, and remote viewing, making it easy to set up a low‑cost, continuously operating sky‑watch station. It is packaged as a versatile, community‑driven solution for hobbyists and researchers alike.
