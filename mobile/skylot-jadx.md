## 📝 Summary

skylot/jadx is an open‑source decompiler that translates Android Dalvik Executable (DEX) files back into readable Java source code. It supports a wide range of Android APKs, provides a graphical user interface as well as command‑line tools, and helps developers analyze, debug, and understand compiled Android applications.
