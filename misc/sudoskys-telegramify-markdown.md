## 📝 Summary

### English

sudoskys/telegramify-markdown is a lightweight Python library that converts standard Markdown syntax into Telegram‑compatible MarkdownV2, handling the platform’s special escaping rules and formatting quirks automatically. By processing plain text or Markdown files, it ensures messages are rendered correctly in Telegram bots or chats without manual tweaking. The project is open‑source, easy to integrate, and suitable for any application that needs reliable Telegram markdown output.
