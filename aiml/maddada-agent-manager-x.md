## 📝 Summary

Agent Manager X is an open-source, macOS desktop application designed to monitor and manage Claude Code, Codex, and OpenCode AI coding agents in real-time. It provides features such as voice or bell notifications, RAM and CPU usage tracking, and the ability to quickly jump to a conversation in any editor or terminal. Additionally, it allows users to easily code out Stale or Idle chats that are wasting memory with a single click.
