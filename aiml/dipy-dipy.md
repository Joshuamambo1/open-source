## 📝 Summary

### English

DIPY (Diffusion Imaging in Python) is an open‑source Python library for advanced medical image analysis, offering a comprehensive suite of tools for spatial normalization, signal processing, machine learning, statistical analysis, and interactive visualization of 3‑D/4‑D+ datasets. In addition to generic image‑processing functions, it provides specialized algorithms for computational anatomy, including diffusion, perfusion, and structural imaging techniques. The project serves researchers and clinicians seeking a flexible, extensible platform for cutting‑edge neuro‑imaging research.
