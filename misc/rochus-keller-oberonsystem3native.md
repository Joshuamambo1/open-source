## 📝 Summary

This open-source project showcases the Oberon System 3 running on a Raspberry Pi Zero 2. A pre-configured and ready-to-use SD card is provided, making it easy for users to experience the Oberon System 3 on the low-cost Raspberry Pi Zero 2 hardware. This project aims to bring the Oberon System, a retro-style operating system, to a modern, portable device.
