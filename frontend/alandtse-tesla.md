## 📝 Summary

Tesla custom integration for Home Assistant – a frontend component that lets users monitor and control their Tesla vehicles directly from Home Assistant. It works by generating a refresh token via third‑party apps, which the integration uses to authenticate with the Tesla API. This open‑source project simplifies Tesla‑Home Assistant connectivity while keeping credential handling secure.
