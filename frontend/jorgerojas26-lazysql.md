## 📝 Summary

Lazysql is an open-source, cross-platform TUI (Text User Interface) database management tool built with Go. It provides an intuitive and interactive way to manage databases, offering a user-friendly interface for tasks such as creating, modifying, and querying databases. This versatile tool allows users to efficiently manage their databases from the command line.
