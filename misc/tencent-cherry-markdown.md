## 📝 Summary

Tencent/cherry-markdown is an open‑source, web‑based Markdown editor that provides a clean, WYSIWYG interface for writing and previewing Markdown content in real time. Built with modern JavaScript, it supports extensible plugins, customizable themes, and a full suite of editing features such as syntax highlighting, image handling, and table editing. The project aims to offer a lightweight yet powerful solution for developers and content creators who need an intuitive Markdown authoring tool.
