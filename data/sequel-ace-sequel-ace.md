## 📝 Summary

### English

Sequel Ace is a lightweight, open‑source MySQL and MariaDB client designed specifically for macOS. It provides a clean, native interface for managing databases, executing queries, and browsing tables, complete with features like syntax highlighting, query history, and SSH tunneling. Ideal for developers and DBAs, Sequel Ace streamlines everyday database tasks while remaining fully extensible and community‑driven.
