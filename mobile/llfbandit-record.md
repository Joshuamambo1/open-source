## 📝 Summary

### English

The **llfbandit/record** plugin provides a simple yet powerful audio recording solution for Flutter apps, supporting customizable settings such as sample rate, bit rate, file format, and source selection. It offers seamless integration across iOS and Android, delivering easy‑to‑use APIs for starting, pausing, stopping, and retrieving recorded audio files. This open‑source library is ideal for developers needing flexible, cross‑platform audio capture in mobile applications.
