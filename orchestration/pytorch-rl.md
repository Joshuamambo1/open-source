## 📝 Summary

The pytorch/rl project is an open-source library that provides a modular and primitive-first approach to Reinforcement Learning using PyTorch. This Python-first library allows users to construct Reinforcement Learning algorithms and models from fundamental building blocks, facilitating flexibility and customization. By leveraging PyTorch's strengths in deep learning, pytorch/rl aims to accelerate RL research and development.
