## 📝 Summary

Here's a brief summary of the open-source project:

OneImg is an open-source image hosting system that converts images to WebP format for efficient storage. Built with Go, this system provides a simple and scalable solution for storing and serving images. It enables users to upload images and automatically converts them to WebP format for optimized storage and display.
