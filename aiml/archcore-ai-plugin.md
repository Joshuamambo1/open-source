## 📝 Summary

### English

archcore‑ai/plugin is an open‑source library that provides a Git‑native repository context layer for AI coding assistants such as Claude Code, Cursor, and similar agents. By exposing the full history, file structure, and diffs of a project directly to the AI, it enables more accurate code generation, refactoring, and debugging without manual prompting. The plugin is lightweight, language‑agnostic, and can be integrated into any development workflow that leverages AI‑driven code assistance.
