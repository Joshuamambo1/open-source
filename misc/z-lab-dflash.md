## 📝 Summary

DFlash is an open-source project that utilizes Block Diffusion for Flash Speculative Decoding, aiming to improve the efficiency of flash memory access. This project employs a novel approach to decoding, leveraging block diffusion to enhance data retrieval from flash storage. By implementing this technique, DFlash strives to reduce power consumption and increase overall system performance.
