## 📝 Summary

The Stunner tool, developed as part of the firefart project, is an open-source solution designed to test and exploit STUN (Session Traversal Utilities for NAT), TURN (Traversal Using Relays around NAT), and TURN over TCP servers. This tool helps in identifying vulnerabilities in these servers, which are crucial for real-time communication over the internet, such as in WebRTC-based applications. By utilizing Stunner, developers and security researchers can assess the security posture of their STUN and TURN servers.
