## 📝 Summary

Embody is an open network format for TouchDesigner that combines the power of Master Control Protocol (MCP) and TouchDesigner's Network Data (TDN). This open-source project enables users to create and work with TouchDesigner projects at speeds that rival the velocity of thought. With Embody, developers can streamline their workflow and collaborate more efficiently.
