## 📝 Summary

Pass With High Score/Block Ads Android is an open-source project that provides a system-wide ad-blocking solution for Android devices without the need for root access. This project uses local VPN-based DNS filtering to effectively block ads, ensuring users' online browsing experience remains uninterrupted and private. By utilizing this solution, users can enjoy a distraction-free internet experience while keeping their personal data secure.
