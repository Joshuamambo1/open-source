## 📝 Summary

Here's a brief summary for the project:

Yet Another Pixiv Illust&Novel Viewer is an open-source online platform for browsing Pixiv's illustrations, anime, comics, and novels on multiple devices. The platform offers customizable layouts, supports Progressive Web App (PWA) installation, and allows users to select from various browsing styles. It also features login options via RefreshToken, OAuth, and Cookie methods, as well as customizable API and image hosting settings.
