## 📝 Summary

### English

Accessibility Insights for Web is an open‑source browser extension and web tool that helps developers find and fix accessibility issues early in the development process. It provides automated scanning, guided manual testing, and detailed remediation guidance, leveraging the WAI‑ARIA and WCAG standards. The project is built with modern frontend technologies and is maintained by Microsoft to promote inclusive, accessible web experiences.
