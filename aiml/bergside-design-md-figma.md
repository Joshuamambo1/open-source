## 📝 Summary

This open-source project is a Figma plugin that automates the process of extracting local styles from designs and generates a DESIGN.md file. This file provides a clear and concise overview of the design elements, allowing for easier collaboration and documentation. By streamlining this process, the plugin helps designers and developers work more efficiently together.
