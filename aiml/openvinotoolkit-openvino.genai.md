## 📝 Summary

The openvino.genai project is an open-source library that enables seamless integration with Generative AI models using a simple and intuitive C++/Python API. Built on top of the OpenVINO Runtime, it provides a streamlined way to run these complex models with ease. This project simplifies the process of leveraging Generative AI capabilities in various applications.
