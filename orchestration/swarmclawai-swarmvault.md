SwarmVault is an open‑source, local‑first knowledge‑graph platform that lets users build a personal LLM‑powered wiki, RAG‑enhanced knowledge base, and persistent agent memory store. Inspired by Andrej Karpathy’s “local‑first” pattern, it functions as an Obsidian‑style PKM system and “second brain” for AI, providing durable storage for Claude, Code, Codex, and OpenClaw contexts. It combines orchestration, AI/ML, and a modern frontend to enable seamless retrieval‑augmented generation and agent‑driven workflows.
