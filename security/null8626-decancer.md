## 📝 Summary

### English

**decancer** is an open‑source library that sanitizes text by stripping or normalizing Unicode confusables and homoglyphs—characters that look alike but have different code points. By converting these deceptive symbols to their plain equivalents, the tool helps prevent phishing, spoofing, and other security‑related attacks that rely on visually similar characters. It can be easily integrated into applications that need robust input validation and safe string handling.
