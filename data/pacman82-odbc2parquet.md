## 📝 Summary

The odbc2parquet project is an open-source command line tool that enables users to retrieve data from an ODBC data source and export it to a compressed Parquet file. This tool streamlines the process of transforming relational data into a columnar storage format, making it ideal for big data analytics. By leveraging ODBC connections, users can access a wide range of data sources and efficiently store them in Parquet files.
