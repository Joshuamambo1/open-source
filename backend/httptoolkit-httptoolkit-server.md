## 📝 Summary

### English

The **HTTP Toolkit server** is the open‑source backend that powers the HTTP Toolkit suite of developer tools. It handles request interception, recording, and replay for HTTP, HTTPS, and WebSocket traffic, providing a programmable API and extensible plugin system for custom analysis and automation. Designed for reliability and performance, it serves as the core service that enables developers to debug, test, and monitor network interactions directly from their local environment.

### Русский

**HTTP Toolkit Server** — открытый сервер‑бэкенд для инструмента HTTP Toolkit, предоставляющий мощные возможности перехвата, анализа и манипуляции HTTP/HTTPS‑трафиком в режиме реального времени. Он реализован на Node.js, поддерживает расширяемую архитектуру плагинов и интегрируется с различными клиентскими и серверными приложениями, облегчая отладку и разработку сетевых сервисов. Проект ориентирован на разработчиков и DevOps‑инженеров, которым нужны гибкие и надёжные инструменты для работы с сетевыми запросами.
