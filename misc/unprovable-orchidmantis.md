## 📝 Summary

Orchid Mantis is an open-source proof-of-concept (PoC) project in Rust that enables Zero Knowledge Proofs (ZKP) of exploit, allowing users to prove the existence of a vulnerability without revealing sensitive information. This innovative approach aims to revolutionize the way security vulnerabilities are reported and validated. By leveraging ZKP, Orchid Mantis facilitates a more secure and efficient process for detecting and addressing potential security issues.
