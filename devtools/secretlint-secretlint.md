## 📝 Summary

### English

secretlint is a pluggable linting framework designed to detect and block the accidental inclusion of secrets—such as API keys, passwords, and other credentials—in source code before they are committed. By supporting a flexible plugin ecosystem, it can be tailored to scan a wide range of file types and secret patterns, helping development and DevOps teams enforce security best‑practices early in the workflow. It integrates smoothly with common development tools and CI pipelines, making it a practical addition to any project’s code‑quality and security toolchain.
