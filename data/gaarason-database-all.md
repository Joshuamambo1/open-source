## 📝 Summary

gaarason/database-all is an open‑source, lightweight ORM that brings Laravel‑style Eloquent capabilities to Java (8, 11, 17, 21, 23) and integrates seamlessly with Spring Boot 2.x and 3.x. It provides a fluent, type‑safe API for building, querying, and persisting relational data while supporting modern Java features and a full set of transaction, relationship, and pagination utilities. The project is packaged as a single dependency, making it easy to add powerful database handling to any Java or Spring Boot application.
