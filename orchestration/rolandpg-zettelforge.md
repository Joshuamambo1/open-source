## 📝 Summary

The Zettelforge project is an open-source, Python-based platform designed for Cyber Threat Intelligence (CTI) that utilizes agentic memory. It incorporates STIX knowledge graphs, threat-actor alias resolution, and an offline-first Reasoning and Action Graph (RAG) for efficient analysis. The platform also features an MCP server for integrating Claude Code and LangChain agents for enhanced AI capabilities.
