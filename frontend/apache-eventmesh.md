## 📝 Summary

### English

Apache EventMesh is a server‑less event middleware designed to simplify the development of distributed, event‑driven applications. It provides a unified, high‑performance gateway for publishing, routing, and consuming events across heterogeneous back‑end services and front‑end clients. By abstracting transport details and offering built‑in scalability, EventMesh enables developers to focus on business logic rather than infrastructure.

### Русский

Apache EventMesh — это современный сервер‑лесный middleware, позволяющий быстро создавать распределённые event‑driven приложения.
