## 📝 Summary

The lusingander/ddv project is an open-source terminal-based viewer for DynamoDB databases. It allows users to interactively explore and query DynamoDB data in a simple and intuitive command-line interface. With ddv, developers can easily inspect and test DynamoDB operations, streamlining their development workflow.
