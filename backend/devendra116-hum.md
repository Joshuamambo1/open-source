## 📝 Summary

Hum is an open‑source, ad‑free terminal music player written in Rust that streams audio directly from public sources without requiring any API keys or external services. Designed for developers and power users, it offers a lightweight, command‑line interface with simple playback controls, playlist management, and support for common audio formats. The project focuses on privacy, speed, and ease of integration into backend workflows.
