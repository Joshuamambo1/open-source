## 📝 Summary

### English

RhinoMCP is an open‑source server that enables AI agents to programmatically create, modify, and query Rhino 3D models via a lightweight, network‑accessible API. Built as a backend service, it bridges Rhino’s modeling capabilities with machine‑learning workflows, allowing developers to integrate intelligent agents into CAD pipelines for automated design, optimization, and content generation.
