## 📝 Summary

The OctopusTakopi/binance_l3_est open-source project enables the estimation of a Level 3 (L3) order book microstructure on Binance using Level 2 (L2) data's change in time. This innovative approach leverages L2 data to approximate the dynamics of the underlying L3 order book, providing valuable insights for traders and researchers. By utilizing L2 data, this project facilitates the analysis and simulation of high-frequency trading strategies on Binance.
