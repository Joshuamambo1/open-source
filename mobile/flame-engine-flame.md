## 📝 Summary

### English

Flame is a lightweight, open‑source game engine built on top of Flutter, enabling developers to create high‑performance 2D games for mobile platforms with minimal boilerplate. It provides a rich set of components—including a component system, sprite rendering, collision detection, input handling, and a flexible game loop—while leveraging Flutter’s widget tree and rendering pipeline. The project is actively maintained, well‑documented, and integrates seamlessly with the broader Flutter ecosystem.

### Русский

Flame — это легковесный, полностью открытый игровой движок, построенный на Flutter и оптимизированный для разработки 2‑D‑игр под мобильные устройства. Он предоставляет простой API, набор готовых компонентов (спрайты, анимации, физика, звук и т.д.) и интеграцию с экосистемой Flutter, позволяя быстро реализовать интерактивные проекты без необходимости изучать низкоуровневые инструменты. Благодаря кроссплатформенной природе Flutter, игры, созданные на Flame, легко распространяются на iOS и Android.
