## 📝 Summary

### English

The **aws/aws-lambda-rust-runtime** project provides a lightweight, idiomatic Rust library that enables developers to write AWS Lambda functions in Rust. It abstracts away the Lambda invocation protocol, offers seamless integration with the AWS SDK, and supports both synchronous and asynchronous handlers, making it easy to build high‑performance, low‑latency serverless applications in Rust.
