## 📝 Summary

### English

Google‑Parfait’s **TensorFlow Federated** is an open‑source framework that enables developers to build and experiment with machine learning models and other computations across decentralized data sources. It extends TensorFlow to support federated learning, allowing algorithms to be trained locally on edge devices while aggregating updates centrally, which helps preserve privacy and reduce data movement. The library provides high‑level APIs, simulation tools, and reference implementations for a wide range of AI/ML and data‑centric use cases.
