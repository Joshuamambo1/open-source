## 📝 Summary

WinDirStat is an open-source disk usage statistics viewer and cleanup tool designed for Microsoft Windows. It provides detailed information on disk space usage, allowing users to identify and manage files and folders that are consuming the most space. With WinDirStat, users can easily scan their drives, visualize usage statistics, and clean up unnecessary files to optimize their system's storage capacity.
