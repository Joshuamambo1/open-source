## 📝 Summary

The **wch/r-source** project provides a read‑only, hourly‑updated mirror of the official R programming language source code hosted on the R project’s Subversion repository (https://svn.r-project.org/R/). It offers developers easy access to the latest R codebase for inspection, debugging, or building, with detailed build instructions available on the project's wiki. This repository serves as a convenient frontend for tracking R’s development without affecting the upstream source.
