## 📝 Summary

### English

Gravitl’s **Netmaker** is an open‑source platform that automates the creation and management of fast, secure, and distributed virtual networks using WireGuard. It provides both a web‑based frontend and a robust backend API, enabling developers and DevOps teams to provision, monitor, and scale peer‑to‑peer mesh networks across cloud, on‑premises, and edge environments with minimal configuration.

### Русский

Gravitl / Netmaker — это open‑source решение для быстрой и безопасной автоматизации создания распределённых виртуальных сетей на базе WireGuard. Проект предоставляет как серверные (Backend, DevOps/Infra) компоненты для управления топологиями, так и удобный веб‑интерфейс (Frontend) для визуального контроля и настройки сетей. Благодаря простому развертыванию и масштабируемой архитектуре, Netmaker упрощает построение приватных сетей в облаках, дата‑центрах и на edge‑устройствах.
