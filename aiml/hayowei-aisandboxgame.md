## 📝 Summary

The AI Sandbox Game is an open-source text RPG that allows users to create highly customizable worlds and configure multi-step AI agents, all within a local-first framework that doesn't require a backend. This project leverages Large Language Models (LLMs) to drive gameplay, enabling users to experiment and innovate in a self-contained environment. With its modular design, AI Sandbox Game empowers developers and enthusiasts to build and play unique text-based adventures.
