## 📝 Summary

lesovsky/pgcenter is a lightweight, terminal‑based administration utility for PostgreSQL that provides real‑time monitoring of server activity, performance metrics, and query statistics. It offers an interactive, ncurses‑styled interface with features such as live charts, per‑database and per‑user views, and the ability to cancel or terminate queries on the fly, making it a handy tool for quick diagnostics and troubleshooting.
