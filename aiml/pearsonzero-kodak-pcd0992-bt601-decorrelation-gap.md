## 📝 Summary

This open-source project measures the decorrelation gap in images when applying BT.601 color space transformation using the Karhunen-Loève Transform (KLT) method, specifically across the Kodak image suite. The project aims to analyze and quantify the effects of decorrelation on image quality, which is crucial in color image processing and compression. By providing a quantitative measure, this project enables developers to evaluate and improve image processing algorithms.
