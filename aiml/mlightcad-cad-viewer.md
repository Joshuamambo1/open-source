## 📝 Summary

Here's a brief summary of the mlightcad/cad-viewer project:

The mlightcad/cad-viewer is a groundbreaking open-source project that allows users to view and edit DXF/DWG files directly in their web browser, eliminating the need for a backend server. This fully web-based application harnesses the power of web technologies to provide a seamless and efficient experience for users. By offering a serverless solution, mlightcad/cad-viewer empowers developers and engineers to work with CAD files more conveniently than ever before.
