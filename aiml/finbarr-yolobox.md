## 📝 Summary

YoloBox is an open-source project that enables AI to run on external storage devices, keeping your home directory organized and protected from AI-related data. This innovative solution allows users to run their AI models and applications on external storage, freeing up space on their primary device. By separating AI data from user files, YoloBox helps maintain a safe and clutter-free computing environment.
