## 📝 Summary

Here's a brief summary of the project:

The `hass-mcp-server` is an open-source Home Assistant custom component that provides an MCP server using HTTP transport, enabling seamless integration with AI assistants like Claude. This allows users to interact with their Home Assistant instance remotely over HTTP, enhancing the possibilities for voice control and automation. By leveraging this MCP server, users can unlock new functionalities in their smart home setup.
