## 📝 Summary

The potato-mesh project is an open-source, federated dashboard for managing local Meshcore and Meshtastic nodes. This community-focused tool streamlines node management by utilizing local LoRa connections, eliminating the need for MQTT clutter. It provides a simple and efficient way to monitor and control nodes within your local network.
