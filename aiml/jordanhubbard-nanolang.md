## 📝 Summary

Jordanhubbard’s **nanolang** is a minimal, experimental programming language created specifically as a target for coding‑oriented large language models. It provides a tiny, well‑defined syntax and semantics that make it easy for LLMs to generate correct, compilable code and for developers to evaluate model performance on real programming tasks. The project serves both as a research tool for AI/ML model testing and a design sandbox for exploring language design concepts.
