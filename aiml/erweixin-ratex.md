## 📝 Summary

RaTeX is an open-source project that provides a pure Rust LaTeX math renderer, offering near 100% coverage of KaTeX syntax. This high-performance library allows for rendering LaTeX equations across multiple platforms, including iOS, Android, Flutter, web, and image formats like PNG, without the need for JavaScript, a WebView, or DOM manipulation. By providing a C ABI, WASM, and server-side PNG/SVG capabilities, RaTeX enables efficient, cross-platform development in various AI/ML, backend, and mobile applications.
