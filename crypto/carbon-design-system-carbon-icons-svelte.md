### 中文

**项目简介（2‑3 句）**  
Carbon‑Design‑System 的 SVG 图标已封装为 Svelte 组件，提供统一、可定制的视觉资源，帮助开发者在 Svelte 项目中快速使用 Carbon 设计体系的图标。

**价值**  
- **统一视觉**：一次性引入即拥有 Carbon 全套图标，保持 UI 风格一致。  
- **提升效率**：图标即为 Svelte 组件，可直接在模板中使用，无需手动处理 SVG，适合快速原型和 Web3 前端开发（如钱包、DeFi 仪表盘）。  
- **开源透明**：实现细节全部公开，便于审计和二次定制，特别适合需要展示或检查区块链工作流的场景。

**典型接入方式**  
1. **安装**：`npm i -D carbon-icons-svelte`（或 `pnpm add carbon-icons-svelte`）。  
2. **在 Svelte 文件中使用**：  
   ```svelte
   <script>
     import { Add24, Search24 } from 'carbon-icons-svelte';
   </script>

   <Add24 class="icon" />
   <Search24 class="icon" />
   ```  
3. **按需引入**：仅导入需要的图标可保持打包体积最小。  
4. **自定义样式**：通过 CSS 或 Svelte 的 `class`、`style` 属性调整颜色、大小等。

- **活跃度**：截至 2026‑07‑02 最近一次提交，GitHub ★458、Fork 26，社区活跃。  
- **技术成熟度**：使用 TypeScript 编写，提供完整的类型声明，兼容 SvelteKit 与普通 Svelte 项目。
