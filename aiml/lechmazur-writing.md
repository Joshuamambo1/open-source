## 📝 Summary

This open-source project, "writing," is a benchmark designed to assess the creative writing capabilities of Large Language Models (LLMs). It evaluates how effectively these models incorporate a set of ten essential story elements into a short narrative. By providing a standardized test, this project aims to help developers and researchers evaluate the storytelling abilities of various LLMs.
