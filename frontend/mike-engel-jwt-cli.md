## 📝 Summary

Here's a brief summary for the open-source project:

The jwt-cli is a lightweight, open-source CLI tool built with Rust that enables fast and efficient decoding and encoding of JSON Web Tokens (JWTs). This project provides a simple and intuitive interface for developers to work with JWTs, making it an ideal addition to any development toolkit. By utilizing Rust for its high-performance capabilities, jwt-cli streamlines the process of handling JWTs.
