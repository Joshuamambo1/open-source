## 📝 Summary

### English

Mearman’s **mcp-wayback-machine** is a lightweight backend service and command‑line interface that lets developers query and retrieve archived web pages from the Internet Archive’s Wayback Machine. It provides a simple API for programmatic access as well as convenient CLI commands for searching, fetching snapshots, and handling redirects, making it easy to integrate historical web data into tools and workflows.
