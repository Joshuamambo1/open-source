## 📝 Summary

Claude-IDE-Bridge is an open-source project that provides full IDE integration for Claude Code through a MCP bridge. This bridge offers over 25 tools for features such as Language Server Protocol (LSP) support, debugging, terminals, Git and GitHub integration, diagnostics, and more. It is compatible with popular IDEs like VS Code, Windsurf, Cursor, and Antigravity.
