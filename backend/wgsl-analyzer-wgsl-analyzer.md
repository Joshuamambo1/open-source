## 📝 Summary

WGSL-analyzer is an open-source project that implements a language server for WGSL (WebGPU Shader Language) and WESL (Web Embedded Standard Language), providing essential tools for developers working with these languages. This project enables features like syntax highlighting, code completion, and diagnostics, enhancing the overall development experience. By leveraging the language server protocol, WGSL-analyzer streamlines the workflow for creating efficient and error-free shaders for WebGPU applications.
