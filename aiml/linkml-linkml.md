## 📝 Summary

### English

LinkML (Linked Open Data Modeling Language) is an open‑source framework for defining, validating, and serializing data models that are interoperable with Linked Data standards such as RDF and JSON‑LD. It provides a concise, schema‑first language for describing entities, attributes, and relationships, along with tooling for automatic code generation, documentation, and data validation across AI/ML and data‑centric applications.

### Русский

LinkML (Linked Open Data Modeling Language) — это открытый язык и набор инструментов для описания, валидации и генерации схем данных в формате RDF/JSON‑LD. Он позволяет разработчикам быстро создавать машиночитаемые модели, автоматически генерировать API, сериализаторы и документацию, а также интегрировать данные из разных источников в проекты в области ИИ/ML и аналитики. Благодаря поддержке множества форматов и активному сообществу, LinkML упрощает построение interoperable data pipelines и способствует более эффективному использованию Linked Open Data.
