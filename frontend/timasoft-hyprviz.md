## 📝 Summary

timasoft/hyprviz is a lightweight, Rust‑based graphical interface for configuring the Hyprland Wayland compositor. It provides a fast, user‑friendly UI that lets users edit Hyprland settings, preview changes, and manage profiles without leaving the desktop. Ideal for anyone looking for a sleek, front‑end solution to streamline Hyprland customization.
