## 📝 Summary

### English

Apache InLong is an open‑source integration framework that provides a unified, end‑to‑end solution for ingesting, transporting, and processing massive volumes of data across a wide range of scenarios—including real‑time streaming, batch, and offline analytics. It offers extensible connectors, unified data governance, and high‑throughput, low‑latency pipelines, enabling enterprises to build scalable data pipelines with minimal operational overhead.

### Русский

Apache InLong — это открытая интеграционная платформа от Apache, предназначенная для единой подачи, обработки и передачи больших объёмов данных в любых сценариях (от IoT‑устройств до бизнес‑систем). Проект обеспечивает конвейер «от источника к потребителю» с поддержкой множества протоколов, форматов и систем хранения, позволяя быстро построить надёжные конвейеры данных без написания собственного кода. Он ориентирован на масштабируемость, отказоустойчивость и лёгкую интеграцию в существующую инфраструктуру.
