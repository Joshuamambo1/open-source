## 📝 Summary

Here's a brief summary of the project:

The cad2data project provides a workflow for AI agents to automate the conversion of CAD files (.rvt, .ifc, .dwg, .dgn) into machine-readable formats on a local Windows machine. This process leverages command-line converters, simplifying the data extraction and processing pipeline. By streamlining the conversion process, the project enables efficient data-driven construction workflows.
