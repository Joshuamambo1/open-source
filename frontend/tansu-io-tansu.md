## 📝 Summary

### English

tansu‑io/tansu is an open‑source, Kafka‑compatible message broker that stores streams directly in cloud‑native data lakes and databases such as Amazon S3, PostgreSQL, SQLite, Apache Iceberg, and Delta Lake. By exposing the standard Kafka API, it lets existing producers and consumers work unchanged while providing durable, query‑able storage and seamless integration with modern analytics pipelines. The project is positioned for front‑end and data teams needing a lightweight, scalable alternative to traditional Kafka deployments.
