## 📝 Summary

### English

Infisical / agent‑vault is an open‑source HTTP credential proxy that securely stores and serves API keys, tokens, and other secrets to AI agents at runtime. By acting as a lightweight vault, it centralises credential management, enforces access controls, and eliminates hard‑coded secrets in agent code, making deployments safer and more maintainable. It’s designed specifically for AI/ML workflows that require dynamic, programmatic access to sensitive credentials.
