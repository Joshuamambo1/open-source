## 📝 Summary

The CodexBar project is an open-source tool that provides anonymous usage statistics for OpenAI's Codex and Claude Code models. This allows users to evaluate the performance and usage of these AI models without the need for login credentials. By offering a transparent view into the operation of these models, CodexBar aims to promote accountability and understanding in the field of artificial intelligence.
