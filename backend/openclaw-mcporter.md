## 📝 Summary

### English

openclaw/mcporter is a lightweight TypeScript library that lets developers invoke Minecraft Classic Protocol (MCP) commands through a clean, idiomatic TypeScript API. It abstracts the low‑level networking details, enabling both programmatic use within backend services and packaging as a command‑line tool for quick scripting. Ideal for developers building bots, automation, or server utilities that need simple, type‑safe access to MCP functionality.
