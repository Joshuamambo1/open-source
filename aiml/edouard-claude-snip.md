## 📝 Summary

Here's a brief summary of the project:

Snip is an open-source CLI proxy that significantly reduces Large Language Model (LLM) token usage by 60-90%. This tool, written in Go, provides declarative YAML filters for popular AI development tools like Claude Code, Cursor, Copilot, and Gemini, offering an alternative to rtk. By streamlining interactions with these tools, Snip helps developers optimize their workflows and reduce costs associated with LLM usage.
