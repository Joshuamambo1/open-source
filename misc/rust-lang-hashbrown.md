## 📝 Summary

### English

rust-lang/hashbrown is a high‑performance hash map implementation for Rust, based on Google's SwissTable algorithm. It provides a fast, memory‑efficient associative container with robust collision handling and support for custom hashers, making it a solid foundation for Rust’s standard library collections and other projects. The library is open‑source, well‑documented, and actively maintained by the Rust community.
