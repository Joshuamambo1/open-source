## 📝 Summary

This open-source project, "Usage Monitor for Claude", is a lightweight Windows tray application that provides real-time monitoring of Claude rate limits. The tool is designed to be portable, with a single EXE file that requires no configuration, making it easy to use. It also offers full auditable code for transparency and reliability.
