## 📝 Summary

OptiScaler is an open-source project that bridges upscaling and frame generation capabilities across various GPUs. It supports multiple upscaling technologies like DLSS2+, XeSS, and FSR2+, enabling users to replace native upscalers and extend frame generation capabilities to non-supporting titles. This project also supports the Nukem mod for seamless DLSSG-to-FSR3 frame generation.
