## 📝 Summary

The Instagram CLI and TUI client, developed as an open-source project, provides a command-line interface (CLI) and text-based user interface (TUI) for interacting with Instagram, aiming to be a convenient and efficient alternative to the traditional mobile or web applications. This project is designed to combat "brainrot" by enabling users to manage their Instagram accounts through a more streamlined and automated process. By leveraging AI/ML and frontend technologies, it offers a unique solution for Instagram enthusiasts.
