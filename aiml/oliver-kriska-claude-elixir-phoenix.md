## 📝 Summary

The **Claude‑Code plugin for Elixir/Phoenix/LiveView** provides a suite of 20 specialist Claude agents that can plan, execute, and audit code changes directly within a Phoenix project. It enforces “Iron Laws” for safety, integrates with Tidewave’s MCP for knowledge capture, and supports parallel research, automatic verification, and four‑agent audit cycles to turn insights into reusable code knowledge. This open‑source tool streamlines AI‑assisted development while maintaining rigorous quality controls.
