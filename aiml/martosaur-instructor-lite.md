**Instructor Lite** is an open‑source Elixir library that enables developers to define and enforce structured output schemas for large language model (LLM) responses. By providing type‑safe, declarative specifications and automatic parsing of LLM completions, it simplifies the integration of reliable, schema‑validated AI/ML functionality into Elixir applications.
