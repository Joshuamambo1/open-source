## 📝 Summary

Tinyblok is an open‑source firmware for ESP32 devices that reads sensor data, applies user‑defined conditioning (e.g., scaling, filtering, unit conversion), and streams the processed measurements to a NATS messaging server. It provides a lightweight, configurable pipeline for real‑time IoT telemetry, making it easy to integrate ESP32 sensors into NATS‑based data architectures.
