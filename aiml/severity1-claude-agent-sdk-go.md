### English

The **severity1/claude-agent-sdk-go** library is an unofficial Go SDK that streamlines integration with the Claude Code CLI, providing developers with a convenient wrapper around the Claude Agent SDK endpoints. It offers typed request/response structures, authentication helpers, and utility functions to enable seamless interaction with Claude’s code‑generation capabilities from Go applications. The project, formerly known as `claude-code-sdk-go`, is open‑source and targets AI/ML and DevTools use cases.
