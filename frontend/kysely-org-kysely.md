## 📝 Summary

### English

kysely‑org/kysely is a lightweight, type‑safe SQL query builder written in TypeScript that lets developers construct database queries with full compile‑time validation and autocomplete support. By generating precise TypeScript types for tables, columns, and query results, it eliminates runtime SQL errors and works seamlessly with any SQL dialect on both frontend (e.g., server‑side rendering) and backend environments.
