> Auto search for movie/series on torrent, usenet, ddl, subtitles, streaming, predb and other sites. Adds links to IMDb pages from hundreds various sites. Adds movies/series to Radarr/Sonarr. Adds external ratings from Metacritic, Rotten Tomatoes, Letterboxd, Douban, Allocine. Media Server indicators for Plex, Jellyfin, Emby. Dark theme/style for Reference View. Adds/Removes to/from Trakt's watchlist. Removes ads

## 📝 Summary

Here's a 2-3 sentence summary for the Purfview/IMDb-Scout-Mod open-source project:

Purfview/IMDb-Scout-Mod is an open-source project that enhances the functionality of Purfview, a media center application. It introduces auto search features for various media sources, including torrent, usenet, and streaming sites, while also providing links to IMDb pages from hundreds of websites. Additionally, the mod adds integrations with popular media management tools, such as Radarr, Sonarr, and Trakt, for a seamless media experience.
