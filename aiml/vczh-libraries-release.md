## 📝 Summary

### English

The **vczh‑libraries/Release** repository contains the compiled release packages for the suite of projects centered around **GacUI**, a native C++ user‑interface framework. GacUI offers a cross‑platform, MVVM‑style UI library with data binding, XML‑based UI description, multi‑language support, and a clean separation between core logic and rendering processes, making it suitable for high‑performance desktop applications. This release folder provides ready‑to‑use binaries and resources for developers looking to integrate GacUI into AI/ML, frontend, or data‑driven software.
