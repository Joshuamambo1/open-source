## 📝 Summary

Subarr is an open-source project that provides a queue and coverage graphical user interface (GUI) for the Subgen/Bazarr subtitle stack. This tool aims to offer a user-friendly interface for managing and tracking subtitles, enhancing the overall experience for users working with the Subgen/Bazarr system. By utilizing a GUI, Subarr simplifies the subtitle management process.
