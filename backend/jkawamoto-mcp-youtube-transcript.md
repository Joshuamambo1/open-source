## 📝 Summary

MCP‑YouTube‑Transcript is a lightweight backend service that fetches and serves subtitles for any public YouTube video. It integrates with an MCP (Modular Compute Platform) server, exposing simple API endpoints that return transcripts in plain text or JSON formats, making it easy to incorporate video captions into other applications.
