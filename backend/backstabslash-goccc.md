## 📝 Summary

The goccc project is an open-source, zero-dependency tool that provides fast and customizable cost calculations for Claude Code. It offers detailed breakdowns by model, day, project, and branch, making it an efficient solution for managing costs. This lightweight binary can be easily integrated into development workflows without the need for runtime dependencies.
