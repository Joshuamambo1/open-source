## 📝 Summary

The PleasePrompto/ductor project is an open-source tool that enables seamless integration and control of various AI and development tools from within the Telegram messaging platform. It supports live streaming, persistent memory, cron jobs, webhooks, and Docker sandboxing for tools like Claude Code, Codex CLI, and Gemini CLI. This project aims to simplify the workflow of developers and AI practitioners by providing a convenient and centralized interface.
