## 📝 Summary

Portal Explorer is an open‑source web application that renders stunning, interactive visualizations of “portals” using real‑time ray‑tracing techniques. Built with modern JavaScript/WebGL, it lets users explore complex, mind‑bending scenes directly in the browser. The project is a playful showcase of graphics programming and is licensed for free use and contribution.
