## 📝 Summary

BetterThanTomorrow’s **calva‑backseat‑driver** is an open‑source VS Code extension that turns AI assistants (such as CoPilot, Claude, or other LLM‑backed agents) into interactive programming partners. It provides a set of tools and APIs for “back‑seat driving” code—letting the AI suggest, edit, and execute code in real time—and can also run as a standalone MCP (Message Control Protocol) server for external integration. Ideal for developers who want tighter, programmable control over AI‑assisted coding workflows.
