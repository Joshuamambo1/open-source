## 📝 Summary

This open-source project is an MCP (Machine-Readable CV) server that utilizes five real-world Automated Tracking System (ATS) parsers to provide in-depth feedback on a CV's formatting and compatibility. By linting a CV against these parsers, users can identify potential issues and optimize their resume for increased job application success. This project combines AI/ML, frontend, and backend expertise to facilitate a more efficient and effective job search process.
