## 📝 Summary

### English

Python Social Auth is a flexible, open‑source authentication library that streamlines the integration of third‑party login providers (e.g., Google, Facebook, GitHub) into Django applications. The `social-app-django` package supplies Django‑specific models, middleware, and view helpers, enabling developers to add secure, configurable social authentication with minimal setup. It supports a wide range of OAuth, OAuth2, and OpenID Connect providers while handling token storage, user association, and common security concerns out of the box.
