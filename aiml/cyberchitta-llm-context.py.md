## 📝 Summary

Here's a brief summary of the open-source project:

LLM-Context.py is an open-source tool that enables seamless collaboration between developers and large language models (LLMs) by sharing code and context through the Model Context Protocol or clipboard. This tool features a rule-based customization system, allowing users to easily switch between different tasks, such as code review and documentation. Additionally, it includes smart code outlining capabilities to enhance productivity.
