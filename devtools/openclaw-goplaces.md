### English

openclaw/goplaces is a modern, command‑line interface written in Go for interacting with the Google Places API. It provides developers with fast, typed access to place search, details, photos, and autocomplete features directly from the terminal, making it easy to integrate location data into scripts and tooling. The project is lightweight, open‑source, and designed for developers who need a quick, programmatic way to query Google Places without writing boiler‑plate HTTP code.
