## 📝 Summary

Commitizen (cz-cli) is an open-source tool that helps developers write high-quality commit messages by guiding them through a standardized commit message format. This utility streamlines the commit process, promoting consistency and clarity in code changes. With Commitizen, developers can ensure their commit messages are clear, concise, and follow best practices.
