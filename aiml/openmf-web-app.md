## 📝 Summary

The openMF/web-app project is an open-source web application built on top of the Mifos X platform, utilizing Apache Fineract as its backend and Angular as its frontend framework. This web app enables users to interact with Mifos X services in a user-friendly and accessible way. The main branch represents the latest stable release, while the dev branch is used for ongoing development and testing.
