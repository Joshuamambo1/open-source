## 📝 Summary

### English

The **async‑stripe** crate provides idiomatic Rust bindings for the Stripe API, supporting both asynchronous and synchronous (blocking) usage patterns. It offers type‑safe request/response structs, comprehensive coverage of Stripe’s payment, customer, and subscription endpoints, and integrates seamlessly with popular async runtimes. Ideal for backend services that need reliable, high‑performance Stripe integration in Rust.

### Русский

Проект **arlyon/async-stripe** предоставляет асинхронные (а также синхронные) привязки к API Stripe на языке Rust, позволяя легко интегрировать платёжные функции в backend‑приложения. Библиотека реализует полный набор методов Stripe, поддерживает работу с токенами, подписками, вебхуками и другими объектами, при этом полностью типизирована и использует современные возможности async/await. Благодаря открытой лицензии и активному сообществу, проект удобен для быстрого и безопасного внедрения платёжных решений в Rust‑проекты.
