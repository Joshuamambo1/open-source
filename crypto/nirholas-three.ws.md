> 3D AI agent platform for the browser. Load any GLB/glTF avatar, give it an LLM brain with memory, emotions, and tool use, then embed it anywhere as a web component. Includes a character studio, on-chain identity via ERC-8004, MCP server, OAuth 2.1, animation gallery, and five embeddable widget types. No plugins. No installs.

## 📝 Summary

Here's a brief summary of the project:

Three.ws is an open-source, browser-based 3D AI agent platform that enables developers to create interactive, customizable digital avatars with advanced capabilities, including tool use and emotional intelligence. These avatars can be easily embedded as web components, allowing for seamless integration into various projects and applications. The platform includes a range of features, from character studios and on-chain identity to animation galleries and OAuth 2.1 integration.
