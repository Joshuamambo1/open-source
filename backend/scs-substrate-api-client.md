## 📝 Summary

This open-source project, scs/substrate-api-client, provides a library for establishing a connection to Substrate APIs over WebSockets, enabling seamless communication and interaction with blockchain networks. The library simplifies the process of building applications that integrate with Substrate-based blockchains, making it easier to develop and deploy blockchain-related projects. Its open-source nature fosters community collaboration and contributions.
