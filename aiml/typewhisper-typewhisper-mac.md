## 📝 Summary

TypeWhisper is an open-source project that brings local speech-to-text capabilities to macOS, utilizing on-device AI for secure and private functionality. With the option to connect to the cloud, users can choose to keep their speech data fully private on their device or opt for cloud-based functionality. This project combines AI/ML, frontend, and design elements to create a seamless and secure speech-to-text experience.
