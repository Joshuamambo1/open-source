## 📝 Summary

Gptman is an open-source, cross-platform tool for managing GUID Partition Table (GPT) disks. It provides a simple and efficient way to copy partitions from one disk to another, allowing for easy migration of data between storage devices. This tool is ideal for system administrators and users who need to manage and maintain their disk partitions.
