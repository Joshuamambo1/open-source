## 📝 Summary

Monocurl is an open‑source desktop application designed to help users create math‑focused videos and slide‑show presentations. It provides tools for drawing equations, annotating graphics, and exporting polished multimedia content, making it ideal for educators, students, and anyone needing to illustrate mathematical concepts. The project is lightweight, cross‑platform, and freely available under an open‑source license.
