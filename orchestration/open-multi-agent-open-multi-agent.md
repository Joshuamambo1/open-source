## 📝 Summary

Open‑Multi‑Agent is a TypeScript‑native framework that turns high‑level goals into executable task DAGs, orchestrating multiple AI agents with built‑in multi‑call‑prompt (MCP) support and live tracing. With only three runtime dependencies, it simplifies complex multi‑agent workflows for orchestration and AI/ML applications.
