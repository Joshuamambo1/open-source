## 📝 Summary

Here is a brief summary for the pompelmi/pompelmi project:

Pompelmi is a lightweight, open-source library for Node.js that integrates ClamAV antivirus scanning, allowing developers to easily scan file uploads with a single function call. This zero-dependency library provides fast and reliable virus detection, returning typed Symbol verdicts for quick integration into applications. It supports both local and Docker/clamd configurations for flexibility and scalability.
