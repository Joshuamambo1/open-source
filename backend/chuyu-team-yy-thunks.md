## 📝 Summary

The YY-Thunks project is an open-source initiative by the Chuyu-Team aimed at addressing the absence of certain essential APIs in Windows XP RTM, including DecodePointer, EncodePointer, and RegDeleteKeyEx. This project aims to provide a solution to these missing APIs, enabling developers to create applications that can run on Windows XP systems while utilizing these critical functions. By filling the gap of these APIs, the project enhances the overall compatibility and functionality of applications on the Windows XP platform.
