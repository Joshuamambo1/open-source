## 📝 Summary

khuynh22/mcp-wireshark is an open‑source MCP (Message Control Protocol) server that bridges Wireshark/tshark functionality with AI assistants and development environments. It lets you capture live traffic, parse .pcap files, run display filters, follow streams, and export results as JSON—all accessible from Claude Desktop, VS Code, or the command line. The project is cross‑platform, fully typed, unit‑tested, and installable via pip.
