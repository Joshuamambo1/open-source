## 📝 Summary

The ani-mime project is an open-source macOS utility that provides a floating status pill to track activity in both the terminal and Claude Code. This pill offers real-time updates, allowing users to stay informed about their productivity and workflow across multiple applications. It aims to simplify task management and increase user efficiency on macOS devices.
