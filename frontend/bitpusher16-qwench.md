## 📝 Summary

Qwench is a cross‑platform terminal typing game that runs on Linux, Windows, and macOS, leveraging the Rust Crossterm library for seamless console handling. It offers fast‑paced, customizable typing challenges directly in the terminal, making it a lightweight yet engaging way to improve typing speed without leaving the command line. Ideal for developers and hobbyists, Qwench combines simple graphics, real‑time scoring, and open‑source extensibility.
