## 📝 Summary

### English

The **cosmic‑greeter** project provides a libcosmic‑based graphical greeter for the **greetd** login daemon, designed to run seamlessly within the Cosmic desktop environment (cosmic‑comp). It offers a lightweight, customizable login screen that integrates with Cosmic’s theming and component system, making it easy to replace the default greeter on Pop!_OS and other Linux distributions.
