## 📝 Summary

Here's a brief summary of the project:

Keydd is an open-source tool designed to extract sensitive information from network traffic packets, making it suitable as a backend proxy or browser downstream proxy. The tool operates stealthily, with zero latency and supports HTTPS connections, ensuring seamless performance. Its functionality makes it a valuable asset for users requiring sensitive data detection in a secure and efficient manner.
