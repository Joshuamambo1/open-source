## 📝 Summary

The Terraform Scaleway provider is an open-source project that allows users to manage Scaleway infrastructure resources through the HashiCorp Terraform configuration language. This provider enables users to create, update, and delete Scaleway resources such as servers, storage, and networks in a declarative and version-controlled manner. By using Terraform with the Scaleway provider, developers can easily manage and provision their Scaleway infrastructure.
