## 📝 Summary

Supabase/Swift is an open-source SDK for integrating Supabase databases into Swift-based applications. It enables developers to query their databases, subscribe to real-time events, upload and download files, and invoke edge functions or PostgreSQL functions via RPC. This SDK provides a comprehensive set of tools for building robust and efficient mobile applications.
