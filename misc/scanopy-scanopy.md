## 📝 Summary

Scanopy is an open-source project that generates network diagrams that automatically update themselves in real-time. This allows users to visualize complex network relationships and observe dynamic changes without manual intervention. By providing an interactive and self-updating visualization, Scanopy facilitates easier understanding and exploration of network structures.
