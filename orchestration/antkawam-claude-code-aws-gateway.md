## 📝 Summary

Here is a brief summary for the open-source project:

Claude Code AWS Gateway is a self-hosted API gateway for Claude Code on Amazon Bedrock. This project enables teams to unlock advanced features such as web search, tool use, and budget management, while also integrating with OIDC SSO and providing an admin portal for management. By self-hosting the API gateway, teams can maintain control over their data and workflow integration.
