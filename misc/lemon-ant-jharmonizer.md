## 📝 Summary

JHarmonizer is an open‑source Java tool that automatically reorders source files and applies consistent formatting, making codebases ready for continuous‑integration pipelines. It parses Java source trees, enforces a deterministic ordering of imports, classes, and members, and rewrites the files according to configurable style rules, helping teams maintain clean, reproducible code without manual refactoring.
