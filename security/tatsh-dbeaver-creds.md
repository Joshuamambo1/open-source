## 📝 Summary

The Tatsh/dbeaver-creds project is an open-source tool designed to decrypt and display the contents of DBeaver's credentials-config.json file. This JSON string contains sensitive information such as database credentials and connections, which can be decrypted using this project. By providing a clear view of these credentials, the tool aims to enhance security and usability for users managing multiple database connections in DBeaver.
