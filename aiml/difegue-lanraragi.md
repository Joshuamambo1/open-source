## 📝 Summary

Difegue/LANraragi is a lightweight, Docker‑ready web application that lets users archive, organize, and read manga or doujinshi collections from any device on their local network. Designed for NAS or server environments, it offers a simple backend for managing large libraries and supports seamless integration with AI/ML tools for metadata extraction and automated tagging.
