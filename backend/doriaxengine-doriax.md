## 📝 Summary

The doriax engine is an open-source game engine designed for both 2D and 3D projects, utilizing a data-oriented approach and the entity component system (ECS) architecture. This robust engine allows developers to create scalable and efficient applications by separating game logic into manageable components. With its open-source nature, the doriax engine invites collaboration and customization to suit diverse project needs.
