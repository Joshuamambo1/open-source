## 📝 Summary

### English

absolute-version/commit-and-tag-version is an open‑source fork of the popular **standard-version** tool that streamlines automated version bumps and CHANGELOG creation. It follows the SemVer and Conventional Commits specifications to generate consistent releases, and adds additional commit‑and‑tag functionality for smoother CI/CD workflows. Suitable for developers who need reliable, convention‑driven release automation.
