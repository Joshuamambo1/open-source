## 📝 Summary

The AgentRules-Architect project is an open-source tool designed to integrate and orchestrate various coding agents, such as Codex and Claude Code, into a unified workflow. This platform allows users to generate AGENTS.md and CLAUDE.md files, and harness the capabilities of these agents through the ExecPlan interface, streamlining coding processes and enhancing productivity. The project supports authentication via API keys or direct authentication with the agents, enabling seamless interaction with a range of coding tools.
