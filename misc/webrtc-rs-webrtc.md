## 📝 Summary

### English

webrtc‑rs/webrtc is a pure‑Rust implementation of the WebRTC stack designed for asynchronous, non‑blocking environments. It provides full support for real‑time audio, video, and data channels, along with ICE, DTLS, and SRTP, while exposing a safe, ergonomic API that integrates seamlessly with async runtimes such as Tokio and async‑std. The project aims to deliver a high‑performance, cross‑platform WebRTC solution for developers building real‑time communication applications in Rust.
