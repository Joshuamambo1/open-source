## 📝 Summary

### English

The **1amageek/Toolbar** library provides a lightweight, auto‑layout‑based solution for creating customizable input toolbars on iOS, making it especially suited for chat interfaces. It abstracts the complexities of constraint management, allowing developers to quickly add and configure an input bar with built‑in support for text entry, send actions, and accessory views. The framework is open‑source and integrates seamlessly into any Swift or Objective‑C mobile project.
