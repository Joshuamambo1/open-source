## 📝 Summary

Here's a 2-3 sentence summary of the gookit/gcli project:

gokit/gcli is an open-source Go library for building command-line applications, providing a comprehensive set of tools for running CLI commands, interacting with users, and displaying data in a visually appealing manner. It supports console color, progress display, and data formatting, making it an ideal choice for developers who want to create robust and user-friendly CLI tools. Additionally, it can generate bash and zsh completion scripts to enhance user experience.
