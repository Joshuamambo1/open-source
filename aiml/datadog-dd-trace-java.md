## 📝 Summary

### English

The **dd-trace-java** library is Datadog’s official APM (Application Performance Monitoring) client for Java applications. It provides automatic and manual instrumentation for a wide range of Java frameworks and libraries, enabling developers to capture traces, metrics, and logs that are sent to Datadog for real‑time observability, performance analysis, and troubleshooting. The project is open source, actively maintained, and integrates with CI/CD pipelines and other DevTools to help teams monitor and optimize their Java services.

### Русский

DataDog/dd-trace-java — это открытый клиент APM от Datadog, написанный на Java, который позволяет автоматически собирать, трассировать и отправлять метрики, логи и распределённые трассы из ваших микросервисов и приложений. Библиотека поддерживает интеграцию с популярными фреймворками (Spring, Play, gRPC и др.), обеспечивает низкую нагрузку на производительность и предоставляет гибкие настройки для кастомных метрик и аналитики. Проект ориентирован на разработчиков, стремящихся улучшить наблюдаемость и диагностику распределённых систем.
