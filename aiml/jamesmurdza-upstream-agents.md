## 📝 Summary

The upstream-agents project is an open-source initiative that enables running AI coding agents in isolated environments, known as sandboxes, directly connected to GitHub repositories. This allows developers to leverage AI-assisted coding capabilities while maintaining the security and isolation of their codebases. By integrating AI agents with GitHub repositories, the project streamlines development workflows and enhances productivity.
