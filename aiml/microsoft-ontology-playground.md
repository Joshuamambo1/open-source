## 📝 Summary

The Microsoft Ontology Playground is an open-source web application designed to facilitate learning and exploration of ontologies and Microsoft Fabric IQ. This zero-backend, fully static platform allows users to browse a catalogue of pre-built ontologies, create their own visually, and export them as RDF/XML files. Additionally, users can share interactive diagrams for collaborative knowledge management and discovery.
