## 📝 Summary

Here's a brief summary of the open-source project in 2-3 sentences:

SubConverter-Extended is an enhanced version of the SubConverter backend, optimized for handling subscription links and node links with improved processing logic. It features a built-in mihomo kernel parser module for perfect node parameter handling, automated scanning and support for new protocols, and compatibility with the original SubConverter template. This project aims to address compatibility issues and provide a more efficient subscription conversion experience.
