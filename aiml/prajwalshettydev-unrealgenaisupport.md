## 📝 Summary

Here's a brief summary for the UnrealGenAISupport project:

UnrealGenAISupport is an open-source plugin for Unreal Engine that enables seamless integration of various Large Language Models (LLMs) and Generative AI (GenAI) models, including popular ones like OpenAI's GPT-5 and Claude Opus. This plugin allows developers to leverage advanced AI capabilities such as NPC AI, 3D generation, and text-to-speech synthesis in Unreal Engine projects. It also supports the UnrealMCP/UnrealClaude server for distributed AI processing.
