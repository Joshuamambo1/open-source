## 📝 Summary

### English

monteslu/agentgate is an open‑source API gateway that lets AI agents retrieve and manipulate a user’s personal data while requiring explicit human approval for any write operations. By acting as a secure, configurable middleware, it enables seamless integration of AI services with backend data stores without compromising privacy or control. The project targets developers building AI/ML applications that need a safe, human‑in‑the‑loop data access layer.
