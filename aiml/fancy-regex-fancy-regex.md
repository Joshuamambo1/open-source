## 📝 Summary

fancy‑regex is a Rust library that extends traditional regular‑expression capabilities with advanced features such as look‑around assertions, backreferences, and conditional patterns. It includes a compatibility mode that mirrors Oniguruma’s syntax, making it easier to port patterns from languages like Ruby and Perl. The crate is open‑source and aims to provide a powerful, ergonomic regex engine for Rust developers.
