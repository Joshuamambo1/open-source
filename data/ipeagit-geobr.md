## 📝 Summary

i​peaGIT /geobr is an open‑source library that provides simple, programmatic access to Brazil’s official geographic datasets (e.g., states, municipalities, biomes, and hydrographic units) for both R and Python users. By wrapping the official IBGE, INPE, and other government sources, it lets researchers download, manipulate, and visualize up‑to‑date spatial data with just a few function calls. The package is especially useful for analysts, planners, and developers needing reliable Brazilian geospatial information in their data‑science workflows.
