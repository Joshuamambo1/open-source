## 📝 Summary

Here's a 2-3 sentence summary for the DOMPurify project:

DOMPurify is an open-source, high-performance XSS sanitizer for HTML, MathML, and SVG. This DOM-only library provides a secure default while offering extensive configurability and hooks for custom sanitization needs. With its robust and flexible approach, DOMPurify helps protect web applications from cross-site scripting (XSS) attacks.
