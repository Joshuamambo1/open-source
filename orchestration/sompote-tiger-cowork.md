## 📝 Summary

Here's a brief summary of the tiger_cowork project:

tiger_cowork is an open-source, self-hosted AI workspace that integrates various tools and services to streamline project development and management. This platform unifies chat, code execution, parallel multi-agent orchestration, and project management, supporting multiple AI providers such as OpenAI-compatible APIs, Claude Code, and Codex CLI. With features like context compression and checkpoint recovery, it enables seamless collaboration and efficient workflow management for large projects.
