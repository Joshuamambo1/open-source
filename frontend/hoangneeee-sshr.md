## 📝 Summary

The sshr project is an open-source Terminal User Interface (TUI) designed for managing and connecting to SSH hosts. It provides a user-friendly, interactive interface for browsing and connecting to SSH servers, streamlining the process of remote access and server management. With sshr, users can easily organize and connect to their SSH hosts in a convenient and efficient manner.
