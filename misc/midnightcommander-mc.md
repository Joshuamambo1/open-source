### English

Midnight Commander ( `mc` ) is a full‑screen, text‑mode file manager for Unix‑like systems, offering a Norton‑Commander‑style dual‑pane interface, powerful file operations, and extensive configurability. The open‑source repository hosts the core C code, build scripts, documentation, and plugins, enabling developers to contribute enhancements, bug fixes, and platform support. It serves as a versatile, cross‑platform tool for navigating and managing files directly from the terminal.
