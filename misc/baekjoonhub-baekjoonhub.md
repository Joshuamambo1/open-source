## 📝 Summary

This open-source project, BaekjoonHub, is an extension designed to simplify the coding experience for Baekjoon Online Judge (BOJ) users. It automates the Git push process for BOJ submissions, streamlining the development workflow and reducing manual effort. This extension is specifically developed to cater to the needs of BOJ users.
