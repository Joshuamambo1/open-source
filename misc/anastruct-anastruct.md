## 📝 Summary

AnaStruct is an open-source Python library for 2D structural analysis, designed to facilitate the calculation and visualization of structural systems. It provides a comprehensive set of tools for engineers and researchers to model, simulate, and analyze various types of 2D structures, including beams, frames, and trusses. With AnaStruct, users can easily perform linear and nonlinear static and dynamic analyses, making it a valuable resource for structural engineering applications.
