## 📝 Summary

### English

PyMySQL is a pure‑Python MySQL client library that enables Python applications to connect to and interact with MySQL databases without requiring any compiled extensions. It implements the DB‑API 2.0 specification, offering familiar cursor and connection objects, support for prepared statements, transactions, and Unicode handling. Ideal for developers who need a lightweight, cross‑platform solution for database access in Python projects.

### Русский

PyMySQL — это полностью реализованная на Python библиотека‑клиент для работы с MySQL и MariaDB, не требующая установки бинарных драйверов. Она поддерживает все основные возможности MySQL протокола, обеспечивает совместимость с API `MySQLdb` и позволяет выполнять запросы, управлять транзакциями и работать с результатными наборами в привычном для Python стиле. Проект активно поддерживается сообществом и подходит для разработки как небольших скриптов, так и крупных веб‑приложений.
