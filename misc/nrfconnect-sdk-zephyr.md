## 📝 Summary

nrfconnect/sdk-zephyr is the Nordic Semiconductor‑specific downstream of the Zephyr Project RTOS, providing a ready‑to‑use SDK that integrates Nordic’s hardware libraries, board support packages, and sample applications with the upstream Zephyr codebase. It streamlines development for Nordic nRF series devices by delivering pre‑configured build scripts, toolchain support, and documentation tailored to the Nordic Connect SDK ecosystem.
