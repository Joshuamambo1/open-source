### English

A lightweight, self‑hosted web portal that securely connects ChatGPT‑Pro models to a user’s personal accounts (e.g., email, calendars, cloud storage) while keeping all data local and private. It provides a simple frontend interface for authentication, request routing, and safe browsing of account information, eliminating the need to expose credentials to external services. The project is open‑source and aimed at developers who want to leverage ChatGPT’s capabilities without compromising security.
