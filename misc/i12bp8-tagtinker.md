## 📝 Summary

Here's a brief summary for the TagTinker project:

TagTinker is an open-source project that utilizes the Flipper Zero app for research purposes, specifically in the field of ESL (Electronic Security Label) studies using Infrared (IR) technology. Based on the work of Furrtek, this project aims to advance knowledge and understanding in the area of ESL security. It allows researchers to explore and analyze ESL security features using a custom-built IR-based solution.
