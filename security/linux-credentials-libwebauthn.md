## 📝 Summary

### English

linux-credentials/libwebauthn is an open‑source Rust library that implements the FIDO2 (WebAuthn) and FIDO U2F specifications for Linux platforms. It provides a secure, low‑level API for handling credential creation, authentication, and device management, enabling developers to integrate password‑less authentication into their applications. The project is focused on security and aims to offer a robust, cross‑compatible foundation for modern, hardware‑backed authentication flows.
