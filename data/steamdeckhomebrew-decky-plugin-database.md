## 📝 Summary

Decky Plugin Database is an open‑source repository that aggregates community‑made plugins for the Decky homebrew platform on the Steam Deck. Developers can submit pull requests with their plugins, which are then listed in the official Decky store for easy discovery and installation. The project serves as a centralized, searchable catalog of Decky extensions.
