## 📝 Summary

Augustus is an open-source LLM (Large Language Model) security testing framework designed to detect potential threats such as prompt injection, jailbreaks, and adversarial attacks. This comprehensive framework offers over 190 probes and 28 providers, all packed into a single Go binary, making it a powerful tool for AI/ML, DevTools, and Security professionals. Augustus enables robust testing and validation of LLMs to ensure their security and resilience in real-world scenarios.
