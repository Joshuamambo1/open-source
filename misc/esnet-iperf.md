## 📝 Summary

### English

esnet/iperf is an open‑source implementation of **iperf3**, a versatile network performance testing tool that measures bandwidth, jitter, and loss for TCP, UDP, and SCTP streams. It provides a client‑server architecture with a simple command‑line interface, supporting detailed reporting, JSON output, and options for parallel streams, reverse testing, and custom socket parameters. The project is maintained as a community‑driven utility for diagnosing and benchmarking network throughput across a wide range of environments.
