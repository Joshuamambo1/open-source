## 📝 Summary

Doors is an open-source, server-driven framework for building stateful and reactive web applications entirely in Go. This framework provides a robust solution for developers to create robust, scalable, and performant web applications. It simplifies the development process by allowing developers to focus on the application logic with a strong emphasis on reactivity and state management.
