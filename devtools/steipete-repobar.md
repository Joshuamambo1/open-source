## 📝 Summary

RepoBar is an open-source project that displays vital GitHub repository information directly in your system's menu bar and terminal. It provides real-time updates on the repository's status, including continuous integration (CI), issues, pull requests, and the latest release. This tool helps developers stay informed and up-to-date with their projects without having to navigate to the GitHub website.
