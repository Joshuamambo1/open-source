## 📝 Summary

PAVO-Bench is an open-source project that showcases a large-scale benchmark for Automatic Speech Recognition (ASR), Language Model (LLM), and Text-to-Speech (TTS) tasks. It features 50,000 voice turns and an 85,000-parameter router, providing a comprehensive testing ground for AI/ML researchers and developers. This benchmark aims to push the boundaries of speech-related AI capabilities.
