## 📝 Summary

The Vera project is a local code search engine that leverages BM25, vector similarity, and cross-encoder reranking techniques to efficiently locate relevant code snippets across multiple languages. Built on top of the Rust programming language, Vera supports over 60 languages and provides structured results with detailed metadata, including file paths, line ranges, and symbol information. This offline code search engine is designed to provide a seamless and efficient development experience.
