## 📝 Summary

mitulgarg/env‑doctor is a lightweight, open‑source tool that automatically detects and resolves CUDA and GPU compatibility problems across local machines, Docker containers, and CI/CD pipelines. It offers both a command‑line interface and a micro‑service (MCP) server, making it easy to integrate GPU health checks into orchestration workflows for AI/ML and backend applications.
