## 📝 Summary

Cannon07/code-preview.nvim is a Neovim plugin that provides an interactive, side‑by‑side diff view of changes generated by AI coding assistants. Before committing an AI‑suggested edit, the plugin renders a live preview of the file differences, allowing you to accept, reject, or refine the modifications directly within the editor. It streamlines AI‑assisted development by giving developers full visibility and control over automated code changes.
