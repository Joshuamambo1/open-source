## 📝 Summary

LeSuisse/vue-dompurify-html is an open-source project that provides a safe alternative to the v-html directive in Vue.js applications. It utilizes the DOMPurify library to sanitize user input, protecting against DOM-based cross-site scripting (XSS) attacks and ensuring a secure rendering of untrusted HTML content. This project helps developers maintain the security and integrity of their web applications.
