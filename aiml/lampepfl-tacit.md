## 📝 Summary

TACIT (Tracked Agent Capabilities In Types) is an open-source project that provides a framework for modeling and tracking agent capabilities in various types, enabling more effective collaboration and decision-making in AI and ML applications. Developed by the lampepfl team, TACIT uses a structured approach to represent agent capabilities, facilitating better communication and integration between different agents and systems. This project aims to improve the reliability and scalability of complex AI systems.
