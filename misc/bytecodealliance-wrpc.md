## 📝 Summary

### English

bytecodealliance/wrpc is an open‑source framework that enables remote procedure calls (RPC) between WebAssembly components and native code. By leveraging the Wasm component model, it provides a type‑safe, language‑agnostic interface for invoking functions across the Wasm‑native boundary, simplifying the development of distributed and polyglot applications. The project is maintained by the Bytecode Alliance and is categorized under miscellaneous utilities.

### Русский

`bytecodealliance/wrpc` — это открытый RPC‑фреймворк, позволяющий вызывать функции и обмениваться данными между WebAssembly‑компонентами и нативным кодом. Он использует возможности Wasm‑компонентной модели для безопасного, эффективного и типобезопасного взаимодействия, поддерживая как синхронные, так и асинхронные вызовы. Проект подходит для построения распределённых систем, микросервисов и интеграции Wasm‑модулей в существующие приложения.
