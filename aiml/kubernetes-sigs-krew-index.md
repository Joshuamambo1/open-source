## 📝 Summary

### English

The **kubernetes-sigs/krew-index** repository hosts the official plugin index for Krew, the package manager for Kubernetes command‑line extensions. It provides a curated, version‑controlled list of plugin manifests and tooling that plugin maintainers use to publish, update, and distribute their plugins to Kubernetes users. By centralizing plugin metadata, the index simplifies discovery, installation, and lifecycle management of Krew plugins across DevOps and infrastructure environments.
