## 📝 Summary

The idp-scim-sync project is an open-source tool that synchronizes AWS Single Sign-On (SSO) groups and users with a Google Workspace directory. This helps maintain consistency between identity management systems, streamlining user access and permissions across both platforms. By automating the syncing process, organizations can reduce administrative burdens and improve their overall identity management experience.
