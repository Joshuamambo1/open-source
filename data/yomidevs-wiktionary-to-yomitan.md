## 📝 Summary

### English

Yomidevs’ **wiktionary-to-yomitan** project converts raw Wiktionary dumps into dictionary files that can be directly imported into the Yomitan browser extension. By parsing and restructuring multilingual lexical data, it generates Yomitan‑compatible JSON/TSV formats, enabling users to build custom, offline dictionaries for language learning and reference.

### Русский

Open‑source проект **yomidevs/wiktionary-to-yomitan** преобразует данные из Википедии‑словаря (Wiktionary) в форматы, совместимые с Yomitan, позволяя быстро создавать локальные словари для изучения японского языка. Скрипты автоматически извлекают лексические единицы, их чтения, значения и примеры, формируя готовые файлы CSV/TSV и JSON, которые можно импортировать в Yomitan. Проект ориентирован на разработчиков и лингвистов, желающих расширять свои словарные базы без ручного ввода.
