## 📝 Summary

InstaPurge is an open-source project that allows users to rapidly delete large volumes of Instagram direct messages (DMs) with a simple script. This tool is designed to streamline the process of cleaning up cluttered inboxes, making it ideal for individuals or businesses with multiple Instagram accounts. By automating the DM deletion process, InstaPurge aims to save users time and reduce the hassle associated with manual message management.
