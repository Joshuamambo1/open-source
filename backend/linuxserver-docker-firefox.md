## 📝 Summary

### English

linuxserver/docker-firefox provides a lightweight, containerized Firefox browser that runs headlessly or with a VNC interface, making it easy to launch a fully functional web browser in any Docker‑compatible environment. It’s pre‑configured with the latest Firefox release, necessary dependencies, and optional extensions, allowing developers, testers, and CI pipelines to automate web tasks without installing a full desktop stack. The image is maintained by LinuxServer.io and follows best practices for security, updates, and resource isolation.
