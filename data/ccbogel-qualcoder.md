## 📝 Summary

### English

QualCoder is an open‑source, cross‑platform tool for qualitative data analysis that supports coding and annotating text, images, audio, and video files. Built with Python 3.10+ and PyQt6, it provides a flexible, GUI‑driven environment for researchers to organize, search, and visualize their coded data. The project is freely available under an open‑source license, making it a cost‑effective solution for mixed‑media qualitative research.
