## 📝 Summary

Mexit/MultiOS-USB is an open-source project that enables users to boot various operating systems directly from ISO or WIM image files. This mobile-focused project streamlines the boot process, allowing users to easily access and test multiple operating systems on a single device. By supporting ISO and WIM formats, users can explore and experience different operating systems without the need for physical installation.
