## 📝 Summary

Aves is an open‑source Android app built with Flutter that lets users browse image and video galleries while providing powerful metadata exploration tools. It offers a sleek, responsive frontend for browsing, sorting, and searching media based on EXIF, GPS, timestamps, and other embedded information. Ideal for developers looking for a mobile‑first, data‑rich gallery solution.
