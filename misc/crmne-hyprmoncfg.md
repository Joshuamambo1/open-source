## 📝 Summary

Hyprmoncfg is an open-source terminal-first monitor configurator and daemon for Hyprland. It allows users to easily manage and configure their monitor settings, making it a convenient tool for those using the Hyprland display manager. This project streamlines the process of setting up and customizing monitor configurations within the Hyprland environment.
