## 📝 Summary

yoshiko-pg/difit is a lightweight command‑line utility that launches a local web server to render Git commit diffs in a GitHub‑style “Files changed” view. By parsing repository history and serving an interactive HTML page, it lets developers quickly inspect changes without leaving the terminal. The tool is focused on backend workflows and requires no external services.
