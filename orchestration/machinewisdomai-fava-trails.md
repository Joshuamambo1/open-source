## 📝 Summary

FAVA Trails is an open-source project that provides a Git-native, curated memory system for AI agents. It utilizes the MCP (Memory Curation Protocol) to manage an AI's memory, offering features such as draft isolation, promotion gates, and thought lifecycle hooks. This enables developers to create more effective and efficient AI agents with a robust memory management system.
