## 📝 Summary

Stride (formerly Xenko) is a free, open‑source, cross‑platform game engine written in C#. It provides a modern rendering pipeline, real‑time editing tools, and a flexible component‑based architecture for building 2D and 3D games on Windows, Linux, macOS, Android, iOS and consoles. The project is community‑driven and licensed under the MIT license.
