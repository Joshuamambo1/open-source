## 📝 Summary

### English

g-plane/markup_fmt is an open‑source, highly configurable formatter for a wide range of markup languages—including HTML, Vue, Svelte, Astro, Angular, Jinja, Twig, Nunjucks, Vento, Mustache, Handlebars, and XML—designed to work seamlessly with the dprint ecosystem. It lets developers enforce consistent code style across diverse templating syntaxes while offering flexible rules and easy integration into modern frontend workflows.
