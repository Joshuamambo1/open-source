## 📝 Summary

Soularr is an open-source Python script that integrates Lidarr with Soulseek, a peer-to-peer file sharing network. This script enables seamless communication between Lidarr, a music library management system, and Soulseek, allowing users to efficiently manage and download music. It simplifies the process of searching, downloading, and organizing music across both platforms.
