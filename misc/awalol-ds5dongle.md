## 📝 Summary

The **awalol/DS5Dongle** project turns a Raspberry Pi Pico 2 W into a functional DualSense 5 (PlayStation 5) wireless dongle, allowing the console to communicate with the Pico as a controller adapter. It provides the necessary firmware and software tools to emulate a DualSense dongle over Bluetooth, enabling custom input handling and integration with the Pico’s hardware features. This open‑source solution is categorized under Miscellaneous utilities.
