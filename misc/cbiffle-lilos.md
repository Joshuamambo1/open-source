## 📝 Summary

### English

cBiffle / lilos is a lightweight, asynchronous real‑time operating system designed specifically for ARM Cortex‑M microcontrollers. It provides a minimal‑footprint, event‑driven kernel with non‑blocking primitives, making it easy to write concurrent firmware without the overhead of a full‑blown RTOS. The project is open‑source and targets resource‑constrained embedded applications.
