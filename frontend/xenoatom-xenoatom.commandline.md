## 📝 Summary

XenoAtom.CommandLine is an open-source, lightweight command-line parsing library for .NET that offers powerful features and NativeAOT compatibility. This library allows developers to create robust and efficient command-line interfaces for their applications, making it easier to parse and validate user input. With its NativeAOT-friendly design, XenoAtom.CommandLine is optimized for performance and can be used in a wide range of .NET projects.
