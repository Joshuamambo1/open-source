### English

The **JetBrains Claude Code and Codex GUI Plugin** is an open‑source extension that adds a graphical interface for interacting with Anthropic’s Claude and OpenAI’s Codex models directly within JetBrains IDEs. It streamlines AI‑assisted development by letting users send prompts, receive code completions, and view responses without leaving the editor. The project targets AI/ML and frontend developers looking to integrate powerful language models into their coding workflow.
