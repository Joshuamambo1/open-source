## 📝 Summary

Peekaboo is an open-source project that allows AI agents to capture screenshots of macOS applications or the entire system, providing an interface for AI-powered visual analysis. This tool offers both command-line interface (CLI) and optional Multi-Client/Server Protocol (MCP) server functionality, enabling users to leverage local or remote AI models for visual question answering. By integrating Peekaboo, developers can accelerate the development and testing of AI-driven applications.
