## 📝 Summary

The ivelin-web/tempo-mcp-server project is an open-source backend solution that integrates with Tempo worklogs in Jira. It functions as an MCP (Machine Cloud Provider) server, allowing for centralized management of worklogs and related data. This project aims to streamline Tempo worklog administration, enhancing productivity and efficiency for Jira users.
