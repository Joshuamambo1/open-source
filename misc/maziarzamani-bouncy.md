## 📝 Summary

Bouncy is a lightweight, open-source Rust web scraper that provides a convenient interface for extracting data from websites. It also includes built-in support for Master Control Program (MCP), a protocol used for controlling and interacting with websites in a programmatic way. With Bouncy, developers can easily scrape and process web data with a simple and efficient Rust API.
