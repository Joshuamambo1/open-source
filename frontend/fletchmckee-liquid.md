## 📝 Summary

### English

Liquid is an open‑source library that brings real‑time shader‑based visual effects to Jetpack Compose Multiplatform applications. By providing a lightweight `RuntimeShader` wrapper and Compose UI components, it enables developers to apply dynamic GPU‑accelerated graphics—such as blurs, color shifts, and custom filters—directly within their multiplatform UI code. The project is geared toward frontend developers looking to enhance UI aesthetics with performant, shader‑driven animations and effects.
