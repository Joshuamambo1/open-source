## 📝 Summary

lessweb/deepcode-cli is an open‑source command‑line interface that integrates DeepCode’s AI‑powered static analysis into developers’ workflows. It scans source code for bugs, security vulnerabilities, and code‑quality issues, providing actionable suggestions directly in the terminal. The tool is designed for quick, automated code reviews and can be incorporated into CI pipelines and local development environments.
