## 📝 Summary

### English

Apache Sedona (formerly GeoSpark) is an open‑source cluster computing framework that extends Apache Spark, Flink and other big‑data engines with native support for large‑scale geospatial analytics. It provides distributed spatial data structures, indexing, and a rich library of spatial SQL functions, enabling fast processing of massive vector and raster datasets for tasks such as spatial joins, range queries, and proximity analysis.

### Русский

Apache Sedona (ранее GeoSpark) — это открытая распределённая платформа для обработки геопространственных данных в масштабах кластера, построенная поверх Apache Spark и Apache Flink. Она предоставляет набор высокопроизводительных операций (операторы, функции и индексы) для работы с векторными и растровыми данными, поддерживая форматы OGC и интеграцию с популярными GIS‑инструментами. Sedona подходит для аналитики, машинного обучения и визуализации больших геоданных в реальном времени.
