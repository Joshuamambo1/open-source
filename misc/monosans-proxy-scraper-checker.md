## 📝 Summary

### English

Monosans / proxy‑scraper‑checker is an open‑source utility that gathers public HTTP, SOCKS4 and SOCKS5 proxy lists from a variety of online sources and validates them in bulk. It provides extensive features such as multi‑threaded scanning, customizable timeout and anonymity checks, output filtering, and support for exporting results in several formats, making it a versatile tool for building reliable proxy pools.
