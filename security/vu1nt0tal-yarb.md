## 📝 Summary

Vu1nT0tal/yarb is an open‑source crawler and notification tool designed to automatically fetch daily security news and alerts. It scrapes multiple reputable sources, aggregates the latest vulnerability information, and pushes concise updates to users via configurable channels (e.g., email, messaging apps). The project aims to simplify staying informed about current security threats for analysts and developers.
