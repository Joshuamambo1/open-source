## 📝 Summary

VecturaKit is an open-source project that provides a Swift-based vector database designed for on-device use with RAG (Relation Augmented Graph) applications. It utilizes MLTensor and MLX Embedders to enable efficient machine learning computations. VecturaKit allows developers to leverage vector databases for their AI and ML projects on local devices.
