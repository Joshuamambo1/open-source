## 📝 Summary

### English

Parallel Browser MCP is an open‑source micro‑control‑plane (MCP) server that enables AI agents to orchestrate browser automation tasks across multiple cloud providers in parallel. It provides a unified backend API for launching, managing, and scaling headless browsers, handling provider‑specific authentication, resource allocation, and result collection, making large‑scale, distributed web interactions accessible to AI‑driven workflows.
