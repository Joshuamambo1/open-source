### English

Meltdown is an open‑source Python client that lets users interact with large language models (LLMs) through a lightweight, native desktop interface built with Tkinter. It provides a simple, cross‑platform GUI for sending prompts, viewing responses, and managing API keys, making LLM experimentation and integration easier for developers and hobbyists. The project is positioned as a dev‑tool in the AI/ML ecosystem, offering quick setup and extensibility without the overhead of web‑based front‑ends.
