## 📝 Summary

### English

MarginallyClever’s Makelangelo software is an open‑source control suite for DIY plotters, with a focus on the wall‑hanging polargraph (also known as Makelangelo). It provides a user‑friendly frontend for designing, converting, and sending vector graphics to the machine, handling calibration, motor control, and real‑time drawing feedback. The project is geared toward hobbyists and makers who want to build or upgrade their own large‑format plotters.
