## 📝 Summary

Here is a 2-3 sentence summary of the Zalafina/QKeyMapper project:

QKeyMapper is an open-source key mapping tool designed for Windows 10 and 11, developed using Qt. It allows users to remap game controller inputs to keyboard and mouse, including joystick control over mouse movement and vice versa, without modifying the system registry or requiring a system restart. This tool provides immediate and reversible key mapping functionality for gamers and enthusiasts.
