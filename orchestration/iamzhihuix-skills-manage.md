## 📝 Summary

The skills-manage project is an open-source desktop application designed to streamline the management of AI coding agent skills across multiple platforms. It integrates with popular tools such as Claude Code, Cursor, Gemini CLI, and Codex, as well as over 20 other platforms, providing a centralized hub for skill management. This allows developers to efficiently orchestrate and optimize their AI coding workflows.
