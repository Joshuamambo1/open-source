## 📝 Summary

Here's a brief summary of the open-source project:

Next-yak is an open-source, zero-runtime CSS-in-JS solution powered by Rust. It enables users to write styled-components syntax and leverage build-time CSS extraction, ensuring seamless compatibility with React Server Components (RSC). This efficient solution allows developers to create performant and scalable web applications.
