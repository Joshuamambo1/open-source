## 📝 Summary

### English

tinygrad is a minimalist deep‑learning framework that implements the core concepts of autograd and tensor operations in just a few hundred lines of Python code. It provides a lightweight, easy‑to‑read alternative to larger libraries like PyTorch while still supporting basic neural‑network building blocks, gradient computation, and GPU acceleration via CUDA. Ideal for learning, experimentation, and contributing to a truly tiny, open‑source AI stack.

### Русский

tinygrad — это минималистичный open‑source‑фреймворк для автодифференцирования и обучения нейронных сетей, написанный на Python и объединяющий простоту micrograd с функциональностью, напоминающей PyTorch. Он реализует базовые операции над тензорами, обратное распространение и оптимизаторы, при этом оставляя код небольшим и легко читаемым, что делает его отличным инструментом для обучения, экспериментов и исследования новых идей в машинном обучении.
