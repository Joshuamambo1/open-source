## 📝 Summary

Here's a brief summary for the AtomUI project:

AtomUI is an open-source project that brings the refined design language and efficient user experience of Ant Design to cross-platform desktop applications developed with .NET. Built on top of Avalonia, it utilizes its robust cross-platform capabilities to deliver a seamless user experience across various platforms. This implementation enables developers to leverage the Ant Design system for .NET in their desktop applications.
