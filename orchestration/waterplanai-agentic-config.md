### 中文

**项目简介**  
WaterplanAI/agentic-config 是一个与项目无关、可组合的 AI 工作流自动化框架，基于 Python 包和 Claude Code 插件，将零散的 Prompt 与工具包装成可复用的 Agent 流程。

**价值**  
- 将孤立的 Prompt、工具和记忆模块统一管理，快速构建多 Agent 协同的业务流程。  
- 支持灵活的工具调用管线和记忆标准化，帮助团队在原型阶段即实现可重复、可追踪的 AI 自动化。  

**典型接入方式**  
1. **SDK / API**：通过 `pip install agentic-config` 引入库，使用提供的 `Agent`, `Workflow`、`Tool` 等类直接在 Python 代码中组装工作流。  
2. **CLI**：安装后可使用 `agentic-config run <config.yaml>` 运行基于 YAML/JSON 的配置文件，适合 CI/CD 或脚本化调用。  
3. **Claude Code 插件**：在 Claude 开发环境中安装插件，可在 IDE 中可视化编辑、调试 Agent 配置，实现低代码集成。
