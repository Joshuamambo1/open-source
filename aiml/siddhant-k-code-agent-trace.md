## 📝 Summary

The agent-trace project is an open-source tool that provides a strace-like functionality for AI agents, enabling the capture and replay of every interaction, including tool calls, prompts, and responses. This includes support for popular AI clients like Claude Code, Cursor, and MCP. By leveraging agent-trace, developers can gain valuable insights into AI agent behavior and optimize their models for improved performance.
