## 📝 Summary

### English

Apache Iceberg is an open‑source table format for large analytic datasets that brings reliability, performance, and evolvability to data lakes. It provides features such as schema evolution, hidden partitioning, time‑travel queries, and ACID‑compliant writes while being compatible with engines like Spark, Flink, Hive, and Trino. The project enables scalable, query‑optimizable data management across cloud and on‑premise storage systems.

### Русский

Apache Iceberg — это открытая библиотека для работы с аналитическими таблицами в формате колонкового хранилища, предоставляющая атомарные операции DDL/DML, схематическую эволюцию и масштабируемый контроль версий данных. Проект интегрируется с популярными вычислительными движками (Spark, Flink, Hive, Presto и др.), обеспечивая надёжную обработку больших объёмов данных в распределённых системах. Iceberg упрощает построение Data Lake, повышая согласованность и производительность запросов без необходимости изменения существующей инфраструктуры.
