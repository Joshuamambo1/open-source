## 📝 Summary

Here's a brief summary of the WhatsApp/eqwalizer project:

Eqwalizer is an open-source type-checker specifically designed for the Erlang programming language. This tool aims to provide accurate and efficient type checking, helping developers ensure their Erlang code is robust and maintainable. By using eqwalizer, developers can catch type-related errors early in the development process, reducing the likelihood of bugs and improving overall code quality.
