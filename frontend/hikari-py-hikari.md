## 📝 Summary

Hikari is an open-source, Python-based Discord API wrapper built around asyncio. This project aims to provide a user-friendly, intuitive interface for interacting with the Discord API, making it an ideal choice for developers looking to create Discord bots or integrations. With a focus on good intentions, Hikari strives to simplify the development process while maintaining a robust and feature-rich Discord API integration.
