## 📝 Summary

### English

myPKA (TomSolid/myPKA) is an open‑source Personal Knowledge Architecture that stores all data as plain Markdown files in a single folder, eliminating vendor lock‑in. It integrates a four‑person AI team using the ICOR® methodology and works seamlessly with Claude Code, Codex, Gemini, Cursor, and Obsidian, positioning it as a lightweight orchestration and dev‑tool platform for AI‑enhanced knowledge management.
