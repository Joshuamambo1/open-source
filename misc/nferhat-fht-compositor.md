## 📝 Summary

This project, fht-compositor, is an open-source dynamic tiling Wayland compositor that allows users to customize their desktop experience with flexibility and efficiency. Built on top of the Wayland protocol, it provides a robust and highly customizable tiling system for managing windows and workspaces. This compositor is ideal for those seeking a tailored and productive desktop environment.
