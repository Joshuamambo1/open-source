## 📝 Summary

Patina is an open‑source tool that identifies and rewrites AI‑generated text across Korean, English, Chinese, and Japanese. It can be used as a plug‑in for Claude Code, Codex CLI, Cursor, and OpenCode, or run independently via a Node.js command‑line interface, making it a versatile solution for developers who need to detect and transform AI writing patterns.
