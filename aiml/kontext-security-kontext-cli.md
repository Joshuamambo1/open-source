## 📝 Summary

Here's a brief summary for the kontext-cli project:

Kontext-cli is an open-source command-line interface (CLI) that enables secure access to services for AI coding agents, shielding sensitive keys from exposure. This innovative tool empowers developers to give their coding agents the necessary permissions without compromising security. By utilizing kontext-cli, developers can streamline the integration of AI agents with various services while maintaining robust security standards.
