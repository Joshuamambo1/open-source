### 中文

**项目简介**  
DocSentinel（arthurpanhku/DocSentinel）是面向网络安全领域的 MCP（Model Context Protocol）服务器，能够让 AI 代理自动化评估文档、问卷和报告。它支持多种文件格式解析、RAG（检索增强生成）知识库构建，自动识别风险、合规缺口并提供修复建议。

**价值主张**  
- **统一协议**：通过标准的 MCP 接口，把 AI 助手与真实的安全工具、数据源无缝对接。  
- **全链路自动化**：从文档解析、知识抽取到风险评估和整改建议，一站式完成，显著降低人工审计成本。  
- **可扩展的知识库**：基于 RAG 的向量检索让模型能够实时查询最新的合规政策和安全基线，提高判断的准确性。

**典型接入方式**  
1. **API/SDK**：直接调用 DocSentinel 提供的 HTTP API 或 Python SDK，发送文档内容或文件路径，获取结构化的风险报告。  
2. **CLI**：在 CI/CD 流程或安全运营脚本中使用 `docsentinel-cli`，实现批量文档评估。  
3. **MCP Server**：将 DocSentinel 部署为 MCP 服务器，供其他 AI 代理（如 OpenAI、Claude 等）通过标准协议调用，实现“模型即服务”。
