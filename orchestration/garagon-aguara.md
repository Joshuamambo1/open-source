## 📝 Summary

The aguara project is an open-source security scanner designed to identify vulnerabilities in AI agent skills and MCP servers. Utilizing static analysis and incident response techniques, aguara provides a robust detection engine that does not rely on Large Language Models (LLM). This single-binary solution offers a comprehensive security solution for AI-powered systems.
