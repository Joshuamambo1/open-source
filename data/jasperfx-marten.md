### English

JasperFx Marten is an open‑source .NET library that provides a transactional document database and event‑sourcing store built on PostgreSQL. It enables developers to persist POCO objects, query them with LINQ, and manage reliable event streams—all with ACID guarantees and seamless integration into the .NET ecosystem.
