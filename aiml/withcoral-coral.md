## 📝 Summary

Coral is an open-source project that provides a unified SQL interface for various data sources, including APIs, files, and live sources. This interface is designed specifically for agents, allowing them to interact with disparate data sources through a single, SQL-based query language. By streamlining data access and integration, Coral aims to simplify the development process for agents working with complex data ecosystems.
