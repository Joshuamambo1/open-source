## 📝 Summary

This open-source project enables a closed-loop WhatsApp advisor for Indian farmers, catering to a vast audience of over 100 million individuals. Developed with serverless architecture on Amazon Web Services (AWS), the project combines frontend and backend capabilities to provide an efficient solution. By leveraging a fully serverless setup, it ensures scalability and cost-effectiveness for a large-scale user base.
