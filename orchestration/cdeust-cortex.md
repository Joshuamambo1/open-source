## 📝 Summary

Cortex is an open‑source, PostgreSQL‑backed persistent memory system for Claude Code that stores and retrieves detailed neuroscience knowledge (41 papers, 26 mechanisms) with strong per‑mechanism ablation evidence. It achieves state‑of‑the‑art retrieval performance—LongMemEval R@10 98.4% (MRR 0.9124) and LoCoMo R@10 94.2% (MRR 0.8278)—and boosts BEAM‑10M generation quality by 33.4% versus flat retrieval, as confirmed by a 31‑row, two‑benchmark ablation study. The project combines orchestration, AI/ML, and backend components, leveraging pgvector for efficient vector search.
