## 📝 Summary

This open-source project provides access to uncapped UK sold house prices data from the Land Registry, allowing users to view prices by postcode or county. The dataset is sourced from public records and made available for anyone to use, offering valuable insights into the UK's property market. This project is an example of the transparency and accessibility of government data through the Land Registry.
