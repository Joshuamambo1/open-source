## 📝 Summary

This open-source project aims to port the classic first-person shooter game "Doom" to the GL-iNet GL-BE10000 Slate 7 Pro, a compact travel router. By utilizing the device's capabilities, gamers can now experience the iconic game on a portable and versatile platform. This project demonstrates the potential for repurposing and optimizing hardware for unique applications.
