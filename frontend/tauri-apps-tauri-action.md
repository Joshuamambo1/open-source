## 📝 Summary

### English

tauri-action is an open‑source GitHub Action that automates the process of compiling a web application into a native Tauri binary for macOS, Linux, and Windows. By integrating seamlessly into CI/CD pipelines, it packages your frontend code with Tauri’s lightweight Rust backend, producing cross‑platform desktop executables without manual setup. This tool streamlines the delivery of performant, secure desktop apps directly from your existing web project.
