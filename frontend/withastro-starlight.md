### 中文

**项目简介（2‑3 句）**  
`withastro/starlight` 是基于 Astro 的文档站点框架，专注于构建外观精美、可访问且高性能的技术文档网站。它提供了一套开箱即用的 UI 组件、主题和最佳实践，让开发者无需从头编写样式和交互，即可快速产出专业的文档站点。

**价值**  
- **降低 UI 开发成本**：内置的可访问组件和主题让团队不必自行设计和实现文档页面的 UI，节省大量前端工时。  
- **提升交付速度**：通过约定式的目录结构和自动化构建流程，文档的编写、预览和部署几乎是即时完成。  
- **优化前端性能**：依托 Astro 的零 JS 运行时和静态输出特性，生成的文档站点加载极快，SEO 与可访问性表现优秀。

**典型接入方式**  
1. **创建小型 PoC**：在已有的 Astro 项目中 `npm i @astrojs/starlight`，按照官方 README 添加 `starlight` 配置，即可在 `src/pages` 下生成默认文档页面。  
2. **迁移现有文档**：将 Markdown 文档搬入 `src/content/docs`，利用 Starlight 的侧边栏、搜索和版本切换等功能，几行配置即可完成迁移。  
3. **CI/CD 集成**：在 GitHub Actions、GitLab CI 等流水线中执行 `astro build && astro export`，将生成的静态文件部署到 Vercel、Netlify 或自建 CDN。
