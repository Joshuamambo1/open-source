## 📝 Summary

The rooster project is an open-source, cross-platform password manager designed to securely store and manage your login credentials. Compatible with Windows, macOS, and Linux, rooster offers a simple and intuitive interface for users to efficiently manage their passwords. This lightweight application prioritizes user privacy and security.
