## 📝 Summary

### English

NetworkManager‑rs (NMRS) provides safe, idiomatic Rust bindings for interacting with NetworkManager via D‑Bus, enabling Rust applications to manage network connections, devices, and settings programmatically. It abstracts the low‑level D‑Bus protocol into ergonomic Rust types and async APIs, making it easy to integrate NetworkManager functionality into modern Rust projects.

### Русский

Проект **networkmanager‑rs/nmrs** предоставляет безопасные и удобные Rust‑обёртки для взаимодействия с NetworkManager через D‑Bus, позволяя писать сетевые приложения и сервисы на Rust без необходимости работать с низкоуровневым C‑API. Библиотека покрывает основные функции менеджера (управление соединениями, мониторинг состояния, настройка устройств) и полностью открыта как open‑source. Подходит для любых проектов, где требуется надёжное и типобезопасное управление сетью в экосистеме Linux.
