## 📝 Summary

### English

The **hassio-addons/repository** project hosts a collection of community‑maintained add‑ons for Home Assistant, providing ready‑to‑install extensions that enhance the Home Assistant platform with additional services, integrations, and utilities. It serves as an open‑source add‑on store, offering curated Docker‑based components that can be easily added via the Home Assistant UI or HACS, and is continuously updated by contributors.
