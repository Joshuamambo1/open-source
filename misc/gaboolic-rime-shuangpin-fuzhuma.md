## 📝 Summary

Here's a brief summary of the project in English:

The rime-shuangpin-fuzhuma project is an open-source, highly customizable input scheme for Chinese typing using the Rime input method engine. It supports various pinyin input methods, including small hanyi, natural pinyin, Sogou pinyin, and Microsoft pinyin, among others, and allows for combination with various auxiliary codes, such as the innovative "mōjī" code. This project aims to empower pinyin users with a powerful and versatile input solution.
