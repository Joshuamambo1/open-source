## 📝 Summary

### English

VyOS is an open‑source network operating system that provides a fully functional, Linux‑based routing platform. The **vyos/vyos-1x** repository contains the core command definitions, scripts, and utility tools that implement VyOS’s CLI, configuration parsing, and system automation, enabling users to build and manage virtual routers, firewalls, and VPN gateways. It is maintained as a community‑driven project and includes cryptographic utilities for secure networking functions.
