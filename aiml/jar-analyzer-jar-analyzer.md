## 📝 Summary

### English

Jar Analyzer is an open‑source GUI tool for in‑depth inspection of Java JAR files. It provides features such as method‑call relationship search, depth‑first search of call chains, simulated JVM taint analysis, string and custom‑expression searches, Java Web component entry detection, CFG and JVM stack‑frame analysis, and integrates AI‑driven capabilities like MCP calls and n8n workflow support. The project targets developers and security analysts who need comprehensive static analysis and visualization of Java bytecode.

### Русский
