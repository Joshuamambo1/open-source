## 📝 Summary

### English

DirectXMesh is an open‑source C++ library that provides a collection of high‑performance algorithms for processing and optimizing 3D mesh data, including vertex cache optimization, mesh simplification, and attribute conversion. Designed for use with DirectX and other graphics APIs, it offers a lightweight, header‑only interface that simplifies tasks such as mesh generation, reordering, and validation while remaining portable across Windows platforms.

### Русский

Open‑source библиотека **DirectXMesh** (microsoft/DirectXMesh) предоставляет набор высокопроизводительных алгоритмов для обработки и оптимизации геометрии 3D‑моделей: генерацию индексов, упрощение мешей, вычисление касательных, конвертацию форматов и т.д.
