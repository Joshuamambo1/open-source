## 📝 Summary

Here's a brief summary for the open-source project:

Cortex is an open-source, persistent memory system designed for Claude Code, a large language model. Built on top of PostgreSQL and pgvector, Cortex leverages stage-aware context assembly to significantly improve memory recall and retrieval performance, achieving 97.8% R@10 on LongMemEval and 92.6% R@10 on LoCoMo. This solution outperforms flat retrieval models by +33.4% on the BEAM-10M dataset.
