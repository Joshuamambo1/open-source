delexw/claude-code-trace is an open-source project that provides a real-time log viewer and monitor for Claude Code sessions. It allows users to visualize, search, and tail JSONL session files through a native desktop GUI, a web interface, and a terminal user interface (tui), making it easier to track and analyze AI/ML model performance. Built with modern technologies like Tauri 2, React, and TypeScript, the project offers a seamless and efficient experience for developers.
