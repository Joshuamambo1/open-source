## 📝 Summary

The aclysma/profiling project offers a lightweight, high-level interface for instrumented profiling libraries, including puffin, optick, tracy, and superluminal-perf. This abstraction streamlines the process of integrating these profiling tools into applications, providing a consistent and simplified experience. By using aclysma/profiling, developers can easily switch between different profiling frameworks without significant code changes.
