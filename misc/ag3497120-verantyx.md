## 📝 Summary

Verantyx is an open-source, native Integrated Development Environment (IDE) designed to protect code confidentiality when interacting with Cloud-based Large Language Models (LLMs). This innovative tool obfuscates code before sending it to Cloud LLMs, enhancing the security and privacy of sensitive code. By using Verantyx, developers can leverage the power of LLMs while safeguarding their intellectual property.
