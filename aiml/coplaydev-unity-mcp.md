## 📝 Summary

Here's a brief summary for the Unity MCP open-source project:

Unity MCP is an open-source bridge that enables seamless integration between AI assistants and the Unity Editor. This tool allows developers to utilize Large Language Model (LLM) tools, such as Claude and Cursor, to manage assets, control scenes, and edit scripts within Unity, streamlining the development process. By automating tasks and enhancing collaboration, Unity MCP bridges the gap between human developers and AI assistants.
