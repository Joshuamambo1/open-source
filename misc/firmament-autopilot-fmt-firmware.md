## 📝 Summary

### English

Firmament‑Autopilot / FMT‑Firmware is an open‑source embedded software suite that implements the Firmament autopilot stack for unmanned aerial vehicles and other robotic platforms. It provides a lightweight, modular flight control architecture with real‑time sensor integration, state estimation, and control loops, enabling developers to customize and extend the firmware for a wide range of hardware configurations. The project is licensed under an open‑source license, encouraging community contributions and reuse across diverse aerospace and robotics applications.
