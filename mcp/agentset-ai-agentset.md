### 中文

**项目简介（2‑3 句话）**  
AgentSet（`agentset-ai/agentset`）是一款开源 RAG（检索增强生成）平台，内置引用管理、深度研究能力，支持 22+ 文件格式、分区存储以及 MCP（Model Context Protocol）服务器等特性。它通过统一的协议让 AI 助手能够安全、可靠地调用真实工具和数据，实现「AI + 工具」的无缝连接。

**价值**  
- **标准化接入**：提供统一的 Model Context Protocol（MCP），让不同的 AI 代理、工具和数据源能够以同一套接口进行交互，降低集成成本。  
- **强大的 RAG 能力**：内置文档分区、跨格式解析（PDF、Word、HTML、CSV 等）以及自动引用生成，显著提升生成内容的可信度和可追溯性。  
- **可扩展的生态**：提供 API、SDK 与 CLI 三种接入方式，支持 TypeScript/JavaScript 生态，同时可通过 Docker 部署 MCP 服务器，方便在本地、私有云或公有云环境中运行。

**典型接入方式**  
1. **API / HTTP 接口**：直接调用 MCP 服务器的 RESTful API，适用于后端服务或微服务架构。  
2. **SDK（TypeScript/JavaScript）**：在 Node.js、前端或 Electron 应用中引入 `@agentset/sdk`，使用类库封装的高层函数完成文档上传、检索、引用生成等操作。  
3. **CLI 工具**：通过 `agentset-cli` 在 CI/CD 流程或脚本中执行文档导入、索引构建、查询等任务，适合 DevOps 与自动化场景。
