## 📝 Summary

Here's a brief summary for the open-source project:

The AMS project is an open-source tool that enables rapid detection of potentially unsafe Large Language Models (LLMs) within 30 seconds. This is achieved through activation analysis, a method that helps identify problematic patterns in model behavior. By leveraging this technology, developers and researchers can quickly assess the safety and reliability of LLMs.
