## 📝 Summary

### English

OpenTelemetry JS is the official JavaScript client library for the OpenTelemetry observability framework, enabling developers to generate, collect, and export traces, metrics, and logs from Node.js and browser applications. It provides a standardized API and SDKs, along with integrations for popular frameworks and libraries, making it easy to instrument code and send telemetry data to a wide range of back‑ends for monitoring and debugging.

### Русский

OpenTelemetry JS — это клиентская библиотека JavaScript для сбора, обработки и экспорта телеметрических данных (traces, metrics, logs) из веб‑ и серверных приложений. Проект предоставляет единый API и SDK, позволяющие легко интегрировать инструментирование в Node.js и браузерные среды, а также поддерживает множество экспортёров и процессоров. Он ориентирован на разработчиков backend‑сервисов и DevTools, стремящихся к наблюдаемости и стандартизации мониторинга.
