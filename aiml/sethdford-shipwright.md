## 📝 Summary

Shipwright is an open-source project that enables the orchestration of fully autonomous Claude Code agent teams. It streamlines delivery pipelines, fleet operations, and provides DORA (Deployment Frequency, Lead Time for Changes, Mean Time to Recovery, and Mean Time to Restore) metrics, along with auto-scaling workers. This automation eliminates the need for human intervention, from GitHub issue to deployed PR.
