## 📝 Summary

### English

**MSD** is an open‑source Android application that enables a phone to emulate various mass‑storage devices (e.g., USB flash drives, CD/DVDs) when connected to a host PC via USB. By leveraging Android’s USB Host and Accessory modes, the app presents selected files or disk images as a virtual storage device, allowing seamless data transfer without needing additional hardware. It’s useful for developers, testers, and power users who need quick, software‑only USB mass‑storage functionality on mobile devices.
