## 📝 Summary

The cargo-shear tool is an open-source project designed to remove unused dependencies from Rust projects. It helps maintain clean and efficient project dependencies by identifying and eliminating unused crates. This can improve build time, reduce binary size, and make projects easier to manage and understand.
