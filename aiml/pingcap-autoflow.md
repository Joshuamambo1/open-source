## 📝 Summary

Here's a brief summary of the pingcap/autoflow project:

Autoflow is an open-source conversational knowledge base tool that utilizes Graph RAG and is powered by TiDB Serverless Vector Storage. It allows users to build and interact with graph-based knowledge bases in a conversational manner, enabling efficient knowledge management and exploration. By leveraging the capabilities of TiDB Serverless Vector Storage, Autoflow provides a scalable and performant solution for knowledge base applications.
