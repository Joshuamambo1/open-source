## 📝 Summary

The n8n-mcp project is an open-source MCP (Machine Control Panel) designed to streamline workflow creation for n8n users. It integrates seamlessly with Claude Desktop, Claude Code, Windsurf, and Cursor, allowing users to build n8n workflows more efficiently and effectively. This project aims to enhance the user experience for n8n workflow automation.
