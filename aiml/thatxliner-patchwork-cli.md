## 📝 Summary

### English

Patchwork is an open‑source framework that lets large language models (LLMs) edit code directly at the abstract syntax tree (AST) level, rather than manipulating raw text. By exposing the program’s structural representation to the model, Patchwork enables more precise, syntax‑aware modifications, reduces errors, and simplifies tasks such as refactoring, bug fixing, and feature addition. The project is positioned at the intersection of AI/ML and software engineering, offering developers a new way to harness LLMs for reliable code transformation.
