## 📝 Summary

depado/goploader is a lightweight, self‑hosted file‑sharing service that encrypts files on the server before storage, ensuring confidentiality without requiring client‑side tools. It offers a simple API compatible with common command‑line utilities such as curl, httpie, and wget, making uploads and downloads straightforward for scripts and users alike. The project focuses on security and ease of integration, targeting developers who need a secure, encrypted file transfer solution.
