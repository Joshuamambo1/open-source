## 📝 Summary

The **Mck‑ppt‑design‑skill** project provides a consulting‑firm style PowerPoint design system that AI agents can use to generate professional slide decks. It offers a library of 70 flat‑design layout patterns implemented with python‑pptx, enabling rapid creation of clean, corporate‑look presentations (also known as “麦麸风格”). The system is open‑source and targets AI/ML workflows that need automated, high‑quality slide design.
