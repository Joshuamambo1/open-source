## 📝 Summary

The luxonis/depthai-core project is an open-source C++ library that provides a low-level interface for the DepthAI hardware, enabling developers to harness the power of computer vision and AI on edge devices. This library offers a robust and efficient way to access DepthAI's capabilities, including depth sensing, object detection, and machine learning. By leveraging the depthai-core library, developers can build innovative applications across various industries, from robotics and healthcare to security and surveillance.
