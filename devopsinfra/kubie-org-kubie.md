## 📝 Summary

### English

Kubie is an open‑source CLI tool that extends the functionality of kubectx and kubens, providing a more powerful and flexible way to switch between Kubernetes contexts and namespaces. It lets users create isolated “workspaces” that remember the selected context and namespace per directory, supports temporary context switches, and integrates with shells for seamless autocompletion and prompt awareness. Ideal for developers and operators who need efficient, context‑aware navigation across multiple Kubernetes clusters.
