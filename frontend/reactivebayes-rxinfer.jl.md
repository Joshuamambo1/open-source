## 📝 Summary

ReactiveBayes/RxInfer.jl is an open-source Julia package that enables automated Bayesian inference on factor graphs using reactive message passing. This approach allows for efficient and scalable inference in complex probabilistic models, making it a valuable tool for various fields such as machine learning and artificial intelligence. By providing a reactive framework for message passing, RxInfer.jl facilitates real-time inference and updates, enabling applications to adapt quickly to changing conditions.
