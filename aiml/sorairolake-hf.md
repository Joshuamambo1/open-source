## 📝 Summary

A lightweight, cross‑platform library for creating, detecting, and managing hidden files and directories, accompanied by a command‑line utility for quick operations. It abstracts OS‑specific hidden‑file handling (Windows attributes, Unix dot‑files, macOS Finder flags) into a simple API, making it easy for developers to integrate hidden‑file support into AI/ML pipelines, dev tools, or any application. The project is open‑source, well‑documented, and aims to provide consistent behavior across Windows, macOS, and Linux.
