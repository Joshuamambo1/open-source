## 📝 Summary

### English

SkyEye is an open‑source intelligent office OA platform built on Spring Boot 2, offering a rapid‑development framework for hospitals, schools, and small‑to‑medium enterprises. It leverages Activiti 5.22 with dynamic forms to enable complex workflow automation without writing Java code, and includes a full suite of modules such as file management, logging, attendance, CRM, ERP inventory, project management, drag‑and‑drop surveys, scheduling, notes, planning, and administrative tools. The system also supports fine‑grained authorization and easy customization for secondary development.
