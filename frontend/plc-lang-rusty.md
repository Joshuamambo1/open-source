## 📝 Summary

PLC-lang/rusty is an open-source project that provides a Structured Text Parser and LLVM Frontend. This project enables the compilation and analysis of Structured Text, a standardized programming language used in various Programmable Logic Controllers (PLCs). By utilizing LLVM, the project facilitates the creation of efficient and high-performance code from Structured Text programs.
