## 📝 Summary

### English

DaemonEngine/Daemon is an open‑source game engine that builds upon two decades of development rooted in the id Tech 3 lineage, tracing its heritage through ioquake3, Wolfenstein: Enemy Territory, and XreaL. It offers a modern, extensible frontend framework for creating first‑person shooters and other real‑time 3D games while preserving compatibility with classic id Tech 3 assets and features.
