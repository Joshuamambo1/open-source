## 📝 Summary

### English

LitMotion is a high‑performance tweening library for Unity that delivers smooth animations without generating garbage‑collected allocations. By leveraging Unity’s Burst compiler and job system, it achieves lightning‑fast execution while keeping memory usage to zero, making it ideal for performance‑critical games and real‑time applications. The library is open‑source, easy to integrate, and works across all Unity platforms.
