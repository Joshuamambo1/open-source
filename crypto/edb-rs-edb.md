## 📝 Summary

### English

EDB (Ethereum Project Debugger) is an open‑source, Rust‑based tool that enables developers and security researchers to step through Ethereum smart‑contract execution, inspect state changes, and analyze transaction traces. It provides a user‑friendly interface for debugging EVM bytecode, helping identify bugs and vulnerabilities in decentralized applications.
