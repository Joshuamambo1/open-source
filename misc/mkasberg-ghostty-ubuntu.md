## 📝 Summary

Here's a brief summary for the mkasberg/ghostty-ubuntu project:

Ghostty-ubuntu is a repository providing pre-built Ubuntu and Debian packages for Ghostty, a command-line text-based terminal multiplexer. This project aims to simplify the installation process of Ghostty on Ubuntu and Debian-based systems. By offering pre-packaged .deb files, users can easily install and utilize Ghostty on their Linux platforms.
