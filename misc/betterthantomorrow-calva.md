## 📝 Summary

Calva is an open-source project that provides interactive Clojure and ClojureScript programming capabilities within the Visual Studio Code (VS Code) environment. This extension allows developers to write, debug, and execute code seamlessly within the VS Code interface. With Calva, users can leverage the power of Clojure and ClojureScript languages in a more efficient and intuitive manner.
