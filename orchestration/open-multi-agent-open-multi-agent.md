## 📝 Summary

### English

**Project Summary**

Open-multi-agent is an open-source, TypeScript-based multi-agent orchestration framework that enables users to decompose complex tasks into task DAGs (Directed Acyclic Graphs) and execute them on various large language models (LLMs) or local models. This framework helps standardize agent memory, add tool-use pipelines, and turn isolated prompts and tools into repeatable agent workflows. By coordinating multi-agent workflows, users can streamline their processes and increase efficiency.

**Value Proposition**

The primary value of open-multi-agent lies in its ability to orchestrate complex workflows involving multiple agents and tools, allowing users to:

* Standardize agent memory and ensure consistent results
* Integrate various tools and pipelines into repeatable workflows
* Increase efficiency and reduce manual effort in coordinating multi-agent tasks

**Practical Adoption Path**

To adopt open-multi-agent, users can follow these steps:

1. **Evaluate the framework**: Assess the framework's capabilities, documentation, and community support.
2. **Choose a suitable LLM**: Select from the supported LLMs (Claude, ChatGPT, Gemini, DeepSeek, or local models) that best fit your use case.
3. **Define task DAGs**: Decompose your complex tasks into task DAGs using the framework's API or SDK
