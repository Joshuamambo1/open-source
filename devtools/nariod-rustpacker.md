## 📝 Summary

Nariod/RustPacker is an open-source template-based shellcode packer written in Rust, designed to help pentesters and security researchers. This tool provides indirect syscall support, making it a valuable resource for those working with low-level system exploitation. With its template-based approach, users can easily create and customize their own shellcode packers.
