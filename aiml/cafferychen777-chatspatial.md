## 📝 Summary

The ChatSpatial project is an open-source initiative that enables users to interact with spatial transcriptomics data through natural language conversations. Built on top of the MCP server, this project integrates with Claude Code and Codex for seamless analysis and exploration of complex biological data. By shifting from code-based interactions to conversational dialogue, researchers and developers can more efficiently discover insights from their spatial transcriptomics data.
