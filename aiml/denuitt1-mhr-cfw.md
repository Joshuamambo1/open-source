## 📝 Summary

The mhr-cfw project is an open-source Domain-Fronting Relay that leverages Google Apps Script (GAS) to route traffic and Cloudflare Workers to bypass Deep Packet Inspection (DPI) restrictions. This innovative approach enables users to circumvent internet censorship and access blocked content. By utilizing a combination of GAS and Cloudflare Workers, the project provides a secure and reliable solution for domain-fronting traffic.
