## 📝 Summary

### English

Dstack-TEE/dstack is an open‑source framework that enables developers to build and deploy AI models within secure, hardware‑based Trusted Execution Environments (TEEs). By abstracting away the complexities of confidential computing, it lets data scientists run training and inference workloads while keeping data and model parameters encrypted and isolated from the host system. The project targets AI/ML applications that require strong privacy guarantees without sacrificing performance.
