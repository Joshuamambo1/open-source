## 📝 Summary

The Markdown Preview Language Server (mpls) is an open-source project that provides a language server for Markdown previewing, enabling features like live preview and syntax highlighting in integrated development environments (IDEs). This project aims to improve the Markdown editing experience by providing a seamless previewing and editing workflow. It is designed to be extensible and customizable for developers and users alike.
