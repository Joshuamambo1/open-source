## 📝 Summary

The DotGit extension is an open-source project that helps users identify exposed .git directories in visited websites, which can potentially lead to security vulnerabilities and unauthorized access. This extension provides a simple and effective way to scan for exposed .git files, promoting website security and best practices for web developers. By using DotGit, users can protect themselves from potential security risks and ensure a safer browsing experience.
