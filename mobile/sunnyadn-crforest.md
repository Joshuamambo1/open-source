## 📝 Summary

Crforest is an open-source project that provides a Python implementation of competing-risks random survival forest (RSF) models, a statistical technique used to analyze time-to-event data. This project boasts performance improvements of up to 6 times compared to the existing R implementation, rfSRC, making it a valuable resource for researchers and analysts. By leveraging Python's speed and efficiency, Crforest enables faster and more efficient analysis of complex survival data.
