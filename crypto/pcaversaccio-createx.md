## 📝 Summary

The createx project is an open-source smart contract designed to simplify and streamline the use of CREATE, CREATE2, and CREATE3 opcodes in EVM (Ethereum Virtual Machine) environments. By providing a factory contract, createx aims to make it easier and safer for developers to create new contracts, reducing the complexity and potential risks associated with manual implementations. This project is particularly beneficial for developers working with contracts in the Ethereum ecosystem.
