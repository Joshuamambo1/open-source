## 📝 Summary

### English

googleapis/js-genai is an open‑source TypeScript/JavaScript SDK that provides developers with a simple, idiomatic interface for interacting with Google’s Gemini generative AI models and Vertex AI services. It abstracts authentication, request handling, and response parsing, enabling seamless integration of generative AI capabilities into web, Node.js, and server‑side applications. The library is designed for AI/ML and backend use cases, offering type‑safe APIs, extensive documentation, and community‑driven support.
