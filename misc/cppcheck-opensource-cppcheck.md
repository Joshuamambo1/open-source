## 📝 Summary

### English

Cppcheck is an open‑source static analysis tool that inspects C and C++ source files to detect bugs, undefined behavior, memory leaks, and style issues without requiring compilation. It provides a lightweight, fast, and highly configurable command‑line interface that can be integrated into build systems, CI pipelines, and IDEs, helping developers improve code quality early in the development cycle. The project is actively maintained and licensed under GPL, making it freely available for both personal and commercial use.
