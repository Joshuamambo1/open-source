## 📝 Summary

Kopuz is a feature‑rich, cross‑platform music player built in Rust with the Dioxus UI framework. It offers a modern, responsive interface and supports common audio formats, playlists, and playback controls while leveraging Rust’s safety and performance. The project is open source, inviting contributions and extensions from the community.
