## 📝 Summary

Aegis-icons is an open-source project providing unofficial two-factor authentication (2FA) entry icons for the Aegis Android authenticator. This collection of icons enhances the user experience by making it easier for individuals to identify and manage their 2FA accounts. By offering customizable design options, Aegis-icons aims to promote secure authentication practices.
