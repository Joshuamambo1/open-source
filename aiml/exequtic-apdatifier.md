## 📝 Summary

exequtic/apdatifier is a KDE Plasma widget that monitors the Arch Linux system for new package updates, tracks the latest releases of other Plasma widgets, and checks Flatpak applications for available upgrades. It consolidates this information into unobtrusive desktop notifications, helping users stay up‑to‑date across multiple software sources without leaving their Plasma environment.
