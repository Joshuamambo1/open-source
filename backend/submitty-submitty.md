## 📝 Summary

Submitty is an open-source project that offers a comprehensive homework submission, automated grading, and TA grading system for educational institutions. It allows instructors to create assignments, track student submissions, and grade assignments with ease, streamlining the grading process. This platform aims to provide a user-friendly and efficient solution for educators and students alike.
