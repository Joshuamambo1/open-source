## 📝 Summary

### English

UNLINEARITY/CLI‑WeChat‑Bridge is an open‑source tool that embeds AI command‑line interfaces directly into the WeChat ClawBot, enabling seamless integration with models such as Codex, Claude Code (non‑channel, approval‑enabled), and OpenCode. It synchronizes sessions between WeChat and a local terminal, supports bidirectional conversations, and allows users to transfer local files to WeChat for collaborative development.
