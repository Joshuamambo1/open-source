## 📝 Summary

The anima project is an open-source companion application for Claude Code, a native macOS tool that enhances the user experience through several features. It includes per-project ASCII familiars, a nim token economy, and a cross-session watcher, all designed to work seamlessly with Claude Code. Developed using Tauri v2 and Rust, anima is a lightweight option that comes in at just 4MB.
