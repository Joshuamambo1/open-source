**项目简介**  
withceleste/celeste-python 是一套开源、类型安全的多模态 AI 基元库，统一封装了所有模型形态（文本、图像、音频等）和主流供应商的接口，提供“一站式”调用体验 🌟。

**价值**  
- **快速落地 AI 能力**：无需自行搭建模型栈，只需引入库即可调用各种大模型，实现原型开发或功能验证。  
- **统一抽象、类型安全**：通过统一的 Python 接口和静态类型检查，降低跨模型、跨供应商的代码碎片化和错误风险。  
- **灵活组合**：支持 RAG（检索增强生成）和智能体工作流等复杂场景的快速拼装，帮助团队在实验阶段快速迭代。

1. **Python SDK**：`pip install celeste-python` 后，直接在代码中导入 `celeste` 包，使用统一的 `Model`、`Pipeline` 等类创建并调用模型。  
2. **CLI**：提供 `celeste` 命令行工具，可在脚本或 CI 中通过命令行调用模型，实现快速测试或批处理。  
3. **API/SDK 扩展**：库内部暴露底层 API 客户端，可按需对接自定义的 HTTP/SDK 调用，实现与现有系统的深度集成。
