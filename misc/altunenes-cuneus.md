## 📝 Summary

The Cuneus project is an open-source engine for developing compute shader applications. It offers a range of features, including easy multi-pass pipelines, hot reload capabilities, and support for audio and video input. This framework allows developers to create and export frames, making it a versatile tool for compute shader-based projects.
