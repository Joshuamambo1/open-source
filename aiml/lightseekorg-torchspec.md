## 📝 Summary

LightSeek’s **TorchSpec** is an open‑source, PyTorch‑native library that enables efficient training of speculative decoding models—techniques that accelerate text generation by predicting multiple tokens in parallel. By integrating tightly with PyTorch’s autograd and CUDA ecosystem, TorchSpec provides easy‑to‑use APIs, customizable loss functions, and performance‑optimized kernels, helping researchers and developers build faster, high‑quality language models.
