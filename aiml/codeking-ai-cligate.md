## 📝 Summary

codeking-ai/cligate is an open‑source, multi‑protocol proxy server that lets developers interact with Claude Code, Codex CLI, Gemini CLI and OpenClaw through a unified API. It handles account pooling, API‑key management, free‑model routing, and provides a visual dashboard for monitoring and controlling requests, making AI model integration easier for backend and dev‑tool workflows.
