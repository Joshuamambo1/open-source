> A bundler for javascript and friends. Packs many modules into a few bundled assets. Code Splitting allows for loading parts of the application on demand. Through "loaders", modules can be CommonJs, AMD, ES6 modules, CSS, Images, JSON, Coffeescript, LESS, ... and your custom stuff.

## 📝 Summary

### English

Webpack is a powerful open‑source JavaScript bundler that compiles and packages a wide variety of resources—such as ES6, CommonJS, AMD modules, CSS, images, JSON, and custom loaders—into optimized bundles. Its code‑splitting capabilities enable on‑demand loading of application chunks, improving performance for modern web front‑end projects.

### Русский

Webpack — это мощный open‑source сборщик JavaScript‑приложений, который объединяет модули разных форматов (CommonJS, AMD, ES6, CSS, изображения, JSON, CoffeeScript, LESS и др.) в небольшие, оптимизированные бандлы. Благодаря функции Code Splitting он позволяет загружать части приложения по требованию, а система загрузчиков (loaders) и плагинов делает процесс сборки гибким и расширяемым.
