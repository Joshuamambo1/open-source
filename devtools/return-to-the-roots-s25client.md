### English

Return‑To‑The‑Roots (s25client) is an open‑source clone of the classic real‑time strategy game Settlers II, recreating the original gameplay, graphics, and mechanics while offering modern, extensible code. The project provides a full client implementation with networking support, customizable AI, and developer tools that make it easy to modify, extend, or integrate new features. It serves both as a faithful recreation for fans of the classic title and as a flexible platform for developers interested in building upon a classic RTS engine.
