## 📝 Summary

auto‑novel is an open‑source web platform that automatically translates light novels using machine translation. It supports fetching and processing novels from online sources, document repositories, or local files, providing a simple frontend for users to read the translated content. The project combines AI/ML translation models with a user‑friendly interface for quick access to translated literature.
