## 📝 Summary

CamillaDSP is an open-source, cross-platform digital signal processing engine designed for crossovers and room correction. This project offers advanced IIR (Infinite Impulse Response) and FIR (Finite Impulse Response) capabilities, allowing users to create high-quality audio processing systems. By providing a flexible and customizable framework, CamillaDSP enables developers to create innovative audio solutions for a variety of applications.
