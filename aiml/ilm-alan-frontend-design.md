## 📝 Summary

Ilm-Alan/frontend-design is an open-source project that provides a set of frontend design templates for AI-powered tools such as Claude Code, Codex, and Gemini CLI. The project features eight aesthetic anchors, each defining a specific CSS token palette, typography, and texture combination. This project enables developers to select a pre-designed style and easily integrate it into their applications.
