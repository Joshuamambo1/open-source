## 📝 Summary

The rsc/tmp project provides a simple implementation of the /tmp directory, a temporary file system commonly found in Unix-like operating systems. This project aims to replicate the functionality of /tmp for development and testing purposes. It allows users to create and manage temporary files and directories in a controlled environment.
