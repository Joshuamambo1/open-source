**项目简介**

**价值**  
- **提升前端交付效率**：提供可复用的 UI 组件库和统一的交互模式，让产品 UI 能在更短时间内上线。  
- **降低开发成本**：通过统一的 API/SDK/CLI，团队可以直接调用已有实现，避免重复编写相似的界面代码。  

**典型接入方式**  
1. **CLI**：使用 `classroom50-cli` 在项目中快速生成页面模板和配置文件。  
2. **SDK**：在 TypeScript 项目中引入 `@foundation50/classroom50-sdk`，通过提供的组件和服务调用接口即可渲染教学相关的 UI。  
3. **API**：后端通过 REST/GraphQL 暴露课程、作业等元数据，前端 SDK 自动同步并渲染对应视图。
