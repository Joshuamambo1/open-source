**项目简介**  
vibeislandapp/vibe-island 为 AI 代理提供「Dynamic Island」式的交互界面，帮助开发者在现有模型之上快速添加可视化、可交互的 AI 能力，而无需从零搭建完整的模型栈。

**价值**  
- **快速原型**：只需几行配置即可让 AI 代理具备动态 UI，适合验证概念、演示产品。  
- **RAG 与工作流**：内置对检索增强生成（RAG）和多步骤代理流程的支持，降低集成复杂度。  
- **模型工具评估**：提供统一的 UI 层，便于对比不同模型或工具链的表现。

**典型接入方式**  
1. **阅读 README**，确认运行环境（Python 3.9+、依赖库）。  
2. **克隆仓库**并在虚拟环境中执行 `pip install -r requirements.txt`。  
3. **创建最小化的 PoC**：在 `examples/` 目录挑选一个示例脚本，替换为自己的模型 API（如 OpenAI、Claude、Gemini），运行验证 UI 是否正常。  
4. **在项目中封装**：将示例代码抽象为库函数或微服务，供内部工具或前端调用。
