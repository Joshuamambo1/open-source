## 📝 Summary

Dembrandt is an open-source project that enables users to extract a website's design system into tokens in seconds, providing a comprehensive set of design elements in one command. This includes logos, colors, typography, borders, and more, making it a valuable tool for frontend development and design. By automating this process, Dembrandt streamlines the design-to-development workflow and facilitates consistent branding across digital platforms.
