## 📝 Summary

### English

AtlassianPS/JiraPS is an open‑source PowerShell module that provides a comprehensive set of cmdlets for communicating with Atlassian JIRA’s REST APIs. It enables administrators and developers to automate common JIRA tasks—such as creating, updating, and querying issues, managing projects, and handling users—from the command line or scripts. The module is built for backend automation and integrates smoothly with existing PowerShell workflows.

### Русский
