## 📝 Summary

The ontosphere is an open-source project that offers a browser-based RDF/ontology knowledge graph editor. This tool allows users to load and visualize datasets from various sources, create and modify nodes and edges on a canvas, and perform OWL-RL reasoning for knowledge graph analysis. It also features AI-native capabilities, including multi-algorithm layout and clustering, via its integration with the MCP tool surface and relay bridge.
