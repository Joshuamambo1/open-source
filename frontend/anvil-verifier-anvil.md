## 📝 Summary

Anvil is an open-source framework that enables the creation of cluster management controllers with guaranteed correctness, leveraging formal verification techniques to ensure their reliability and security. This experimental tool aims to bridge the gap between theory and practice in distributed systems management, promoting the development of robust and trustworthy controllers. By providing a foundation for formally verified cluster management, Anvil seeks to address the increasing need for secure and efficient distributed systems.
