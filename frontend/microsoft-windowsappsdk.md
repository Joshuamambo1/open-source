## 📝 Summary

### English

The Windows App SDK (formerly Project Reunion) is an open‑source framework that brings modern Windows UI, APIs, and platform capabilities to any Windows desktop application while preserving backward compatibility. Delivered as NuGet packages, it lets developers integrate the latest Windows features—such as WinUI 3, XAML Islands, and native system services—without being tied to a specific OS version, simplifying both frontend and backend development for Windows apps.

### Русский

Windows App SDK — это открытый набор инструментов от Microsoft, который позволяет современным Windows‑приложениям использовать новейшие UI‑компоненты, API и платформенные возможности, при этом сохраняя обратную совместимость со старыми версиями ОС. Пакет распространяется через NuGet, что упрощает интеграцию как в фронтенд‑, так и в бэкенд‑части десктопных приложений. Проект активно поддерживается сообществом и подходит для разработки гибких, масштабируемых Windows‑приложений.
