## 📝 Summary

Here's a brief summary for the PcapXray project:

PcapXray is an open-source network forensics tool that enables users to visualize packet captures offline as detailed network diagrams. This tool not only identifies devices within the network but also highlights important communication flows and facilitates the extraction of files from captured data. With PcapXray, network analysts can gain valuable insights into network activity and conduct thorough investigations.
