## 📝 Summary

Claude Remote Runner is an open-source project that enables secure remote control of Claude Code through end-to-end encryption. This allows users to monitor, approve, and respond to agent prompts from their mobile device via a companion app and push notifications. The project aims to bridge the gap between Claude Code and remote work, providing a secure and convenient experience for developers and teams.
