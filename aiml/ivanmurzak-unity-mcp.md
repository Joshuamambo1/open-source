## 📝 Summary

Here's a brief summary for the Unity-MCP project:

Unity-MCP is an open-source project that provides a comprehensive AI development and testing loop for the Unity Engine. It offers a CLI for quick setup, efficient token usage, and advanced tools for AI skills and MCP (Machine Learning Platform) integration. With Unity-MCP, developers can easily turn any C# method into a tool with a single line, making it compatible with various AI coding platforms like Claude Code, Gemini, and Copilot.
