**项目简介**  
bvisible/mcp-ssh-manager 是一套基于 Model Context Protocol（MCP）的 SSH 服务器实现，提供 37 种远程运维工具（DevOps 自动化、备份、数据库操作、健康监控等），并通过统一的协议让 AI 助手（如 Claude、OpenAI Codex）直接调用真实的系统工具和数据。

**价值**  
- **AI‑工具桥梁**：为大型语言模型提供标准化、可编程的 SSH 接口，使它们能够安全、可靠地执行实际的运维任务。  
- **统一协议**：采用 MCP 统一协议，简化 AI Agent 与多种后端系统的集成，降低开发和维护成本。  
- **即插即用**：提供 API、SDK 与 CLI 三种接入方式，适配 JavaScript/Node 环境，快速嵌入现有 CI/CD、监控或备份流水线。

**典型接入方式**  
1. **API/SDK**：在 Node.js 项目中通过 `npm install @bvisible/mcp-ssh-manager` 引入库，使用 `McpSshClient` 调用远程命令或文件操作。  
2. **CLI**：安装全局命令 `mcp-ssh`，在脚本或 CI 中直接执行 `mcp-ssh run <tool> --args …`。  
3. **MCP Server**：部署 `mcp-ssh-manager` 作为独立服务，AI Agent 通过 HTTP/WebSocket 与其通信，完成跨语言、跨平台的工具调用。
