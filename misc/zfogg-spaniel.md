## 📝 Summary

Spaniel is an open-source project that provides a local OpenTelemetry viewer for developers, offering a Postman-like interface to visualize and interact with distributed traces. This tool allows users to inspect and explore their application's telemetry data in real-time, making it easier to debug and optimize performance issues. By providing an intuitive interface for local development, Spaniel streamlines the process of tracing and debugging distributed systems.
