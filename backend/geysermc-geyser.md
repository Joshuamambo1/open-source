## 📝 Summary

Geyser is an open-source project that serves as a bridge or proxy, enabling seamless communication between Minecraft: Java Edition servers and Minecraft: Bedrock Edition clients. This allows players to connect to and play on Java Edition servers using the more user-friendly and accessible Bedrock Edition. Geyser facilitates cross-platform compatibility, expanding the reach and accessibility of Java Edition servers.
