## 📝 Summary

### English

Versatica Mediasoup is a high‑performance, open‑source WebRTC SFU (Selective Forwarding Unit) that enables scalable, low‑latency video conferencing for backend services. Built in Node.js and C++, it provides a flexible API for managing media streams, handling thousands of participants with efficient bandwidth usage and full support for modern WebRTC features.
