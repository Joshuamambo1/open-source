## 📝 Summary

Here's a brief summary of the mxyhi/token_proxy project:

mxyhi/token_proxy is an open-source, local API gateway for popular AI services such as OpenAI, Gemini, and Anthropic. This tool enables users to manage AI tokens locally, utilizing SQLite for token counting, and offers priority-based load balancing, along with optional format conversion for OpenAI Chat and Responses formats. It also provides a seamless one-click setup for Claude Code and Codex.
