## 📝 Summary

The sktime-mcp project provides an MCP layer that enables seamless interaction between sktime's native time series data registry and large language models (LLMs). This integration allows for the transfer of complex time series knowledge and semantics to LLMs, facilitating more accurate and informed decision-making. By exposing sktime's registry to LLMs, sktime-mcp empowers the development of more sophisticated AI applications.
