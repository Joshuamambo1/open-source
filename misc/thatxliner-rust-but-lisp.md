## 📝 Summary

**Rust but Lisp** is an experimental project that brings Lisp‑style macros and homoiconic code manipulation to the Rust programming language, allowing developers to write and transform Rust code using Lisp‑like s‑expressions. By embedding a small Lisp interpreter/compiler within Rust’s toolchain, it enables powerful metaprogramming techniques while still producing native Rust binaries. The project serves as a proof‑of‑concept and playground for exploring how Rust’s safety and performance can be combined with Lisp’s expressive macro system.
