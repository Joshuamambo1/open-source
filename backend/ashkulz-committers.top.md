## 📝 Summary

The committers.top project is an open-source command-line tool that leverages the GitHub GraphQL API to rank users based on their contribution history. It utilizes this ranking data to generate a static website showcasing the top contributors. This project serves as a valuable resource for developers and communities looking to visualize and recognize their contributors.
