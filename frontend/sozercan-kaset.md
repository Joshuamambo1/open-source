## 📝 Summary

Sozercan’s **kaset** is an open‑source macOS client that fills the gap for YouTube Music users, offering a native desktop experience with a clean, responsive UI. It combines a frontend built with modern web technologies and a backend that handles authentication, streaming, and playlist management, making YouTube Music feel like a first‑class macOS app.
