### 中文

**项目简介（2‑3 句）**  
AetherLink 是一款基于 React、TypeScript 与 Capacitor 的跨平台 AI 助手应用，内置对 OpenAI、Google Gemini、Anthropic Claude、Grok 等主流大模型的统一调用。它支持自定义模型配置、多话题会话管理以及 AI 推理可视化，提供流畅的对话体验。

**价值**  
- 通过 **Model Context Protocol**（MCP）实现 AI 助手与真实工具、数据的标准化对接，降低了业务方自行封装各模型 API 的成本。  
- 统一的模型抽象层让同一套前端代码即可在不同大模型之间切换，帮助企业快速实验、评估并部署最适合的模型。  
- 开源且跨平台（Web、iOS、Android），可直接嵌入内部系统或面向终端用户的产品，提升 AI 功能的可达性和一致性。

**典型接入方式**  
1. **SDK / API**：在项目中通过 npm 安装 `@aetherlink/sdk`，使用提供的 `createClient({model, apiKey, endpoint})` 方法即可调用任意支持的模型。  
2. **CLI**：通过 `npx aetherlink-cli` 运行本地或远程的 MCP 服务器，快速搭建模型转发层。  
3. **Capacitor 插件**：在移动端项目中直接引用 `@capacitor/aetherlink` 插件，利用原生桥接实现离线缓存与本地推理。  
4. **自定义配置**：在 `aetherlink.config.ts` 中声明模型列表、路由规则和工具插件（如数据库、文件系统），即可实现“一键”连接业务工具。
