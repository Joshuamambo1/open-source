### 中文

**项目简介（2‑3 句）**  
Dart‑Code 是官方维护的 VS Code 扩展，为 Dart 与 Flutter 开发提供语法高亮、智能补全、调试、热重载等完整 IDE 体验。它基于 TypeScript 实现，持续更新，已被数千名开发者在日常移动端项目中采用。

**价值**  
- **一站式开发体验**：在 VS Code 中即可完成代码编辑、运行、调试和 UI 预览，无需切换工具链。  
- **官方质量保障**：由 Dart 与 Flutter 团队维护，兼容最新 SDK，及时跟进语言特性和平台改动。  
- **生态兼容**：支持常用的 Flutter 插件（如 DevTools、pub 包管理）以及 VS Code 的主题、键位映射和工作区配置，易于融入现有开发流程。

**典型接入方式**  
1. **安装扩展**：在 VS Code Marketplace 搜索 “Dart” 或直接运行 `code --install-extension Dart-Code.dart-code`。  
2. **配置 SDK 路径**（可选）：在工作区或用户设置 `dart.sdkPath` 指向本地 Dart/Flutter 安装目录。  
3. **创建/打开 Flutter 项目**：打开 `pubspec.yaml`，扩展会自动激活并提供代码补全、分析、调试等功能。  
4. **CI/CD 集成**（如需）  
   - 在流水线中使用 `flutter analyze`、`flutter test`、`flutter build` 等命令，VS Code 本身不参与构建，只负责本地开发体验。  
   - 如需统一代码检查，可在项目根目录加入 `.vscode/settings.json`，统一开启 `dart.analysisExcludedFolders`、`dart.lineLength` 等规则。
