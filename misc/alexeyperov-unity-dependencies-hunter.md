## 📝 Summary

The Unity-Dependencies-Hunter is an open-source tool designed to identify unreferenced assets within Unity projects. It scans all project files to provide a comprehensive list of unused assets, helping developers optimize project sizes and maintain cleaner, more efficient projects. By automating this process, developers can streamline their workflow and reduce waste.
