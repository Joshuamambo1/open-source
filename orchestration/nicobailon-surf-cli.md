## 📝 Summary

The surf-cli project is an open-source CLI tool that enables AI agents to control Chrome in a seamless and agent-agnostic manner, requiring no configuration. This battle-tested solution provides a powerful orchestration tool for AI/ML applications, making it easier to interact with Chrome through a command-line interface. By leveraging surf-cli, developers can simplify their workflows and unlock new possibilities for AI-driven browser automation.
