## 📝 Summary

Here's a brief summary of the project:

ChenYilong/CYLTabBarController is an open-source iOS UI module library that simplifies the process of adding animations to tab bar items and icons using Lottie, Liquid Glass, and other features. It supports a unique "middle plus" tab bar style, customizable badges, and dynamic refresh capabilities. This library is tested on iOS 26, Dark Mode, and iPhone 17.
