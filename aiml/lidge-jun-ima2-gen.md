## 📝 Summary

The ima2-gen project is an open-source tool that enables users to generate images using OpenAI's GPT-Image capabilities. It provides a dual-authentication system, allowing users to access the feature either through a paid API key or a free OAuth connection via ChatGPT. With ima2-gen, users can perform text-to-image, image-to-image, and parallel generation tasks, as well as customize image sizes.
