## 📝 Summary

This open-source project, vscode-amiga-debug, is a Visual Studio Code extension designed to streamline the development process for Amiga C/C++ programs. It provides a one-stop solution for compiling, debugging, and profiling these programs using the bundled gcc 14.2, in conjunction with the bundled emulation tools WinUAE/FS-UAE. This extension aims to simplify the development experience for Amiga enthusiasts and developers.
