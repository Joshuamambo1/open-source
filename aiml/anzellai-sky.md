## 📝 Summary

Sky is an open-source programming language inspired by Elm, designed to compile to Go and provide features such as Hindley-Milner types and server-driven UI (Sky.Live). This language enables developers to build efficient and scalable applications with a single binary output. By combining Elm's functional programming style with Go's performance, Sky aims to provide a unique solution for modern frontend and backend development.
