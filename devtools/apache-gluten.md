## 📝 Summary

### English

Gluten is an open‑source middleware that bridges JVM‑based SQL engines (such as Spark or Flink) with high‑performance native execution back‑ends. By translating the logical and physical plans generated by the SQL engine into native code, it offloads compute‑intensive workloads to optimized native runtimes, delivering faster query execution while preserving the familiar JVM ecosystem. The project is categorized under DevTools.

### Русский

Gluten — это открытая библиотека‑прослойка, которая позволяет JVM‑ориентированным SQL‑движкам (Spark, Flink и т.п.) делегировать выполнение запросов нативным движкам, тем самым повышая производительность и снижая нагрузку на JVM. Проект обеспечивает прозрачную интеграцию, поддерживая стандарты Apache Arrow и выполняя тяжелые вычисления в C++ без изменения пользовательского кода. Он относится к категории DevTools и распространяется под лицензией Apache 2.0.
