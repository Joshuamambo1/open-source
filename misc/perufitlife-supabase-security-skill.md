## 📝 Summary

The project is a personal security audit that examines backend implementations across five different Backend‑as‑a‑Service (BaaS) platforms. By testing each service with the author’s own code, the author uncovered a data‑leak vulnerability in every one, documenting the flaws and providing proof‑of‑concept examples. The repository shares the audit methodology, findings, and remediation suggestions to help developers and BaaS providers improve their security posture.
