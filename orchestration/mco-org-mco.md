## 📝 Summary

mco (mco-org/mco) is a neutral orchestration layer that lets developers invoke any AI coding agent—Claude Code, Codex CLI, Gemini CLI, OpenCode, Qwen Code, etc.—from any environment, whether it’s Cursor, Trae, Copilot, Windsurf, or a plain terminal. By abstracting the interaction with prompts, agents, and IDEs, it enables seamless, plug‑and‑play AI‑assisted coding across diverse workflows.
