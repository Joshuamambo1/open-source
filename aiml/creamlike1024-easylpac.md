## 📝 Summary

EasyLPAC is an open-source, user-friendly graphical user interface (GUI) for the lpac algorithm, designed to simplify the use of this AI/ML technology. This project aims to bridge the gap between the complexity of the lpac algorithm and its accessibility to a wider audience, making it easier to integrate into applications. The GUI frontend provides an intuitive interface for users to interact with lpac, streamlining the process and reducing the barrier to entry.
