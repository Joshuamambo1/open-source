> AppCUI is a fast, cross-platform console and text-based user interface (CUI/TUI) framework for Rust. It combines a low-level console engine for input (mouse, keyboard, clipboard, etc.), colors, and rendering with a high-level, rich toolkit of widgets such as windows, menus, buttons, checkboxes, and many more, available for Windows, Linux and Mac.

## 📝 Summary

AppCUI-rs is an open-source, cross-platform console and text-based user interface (CUI/TUI) framework for Rust. It provides a fast and versatile toolkit that supports low-level console operations and high-level widgets, including windows, menus, buttons, and more, making it suitable for Windows, Linux, and Mac applications. This framework is ideal for developers creating console-based interfaces for AI/ML, frontend, and devtool applications.
