## 📝 Summary

Here's a 2-3 sentence summary for the open-source project "uhop/stream-json":

uhop/stream-json is a lightweight Node.js library that enables efficient JSON processing pipelines with minimal memory usage. It achieves this by streaming individual JSON primitives using a SAX-inspired API, making it suitable for parsing large JSON files that exceed available memory. This micro-library is ideal for applications that require high-performance JSON processing, such as AI/ML and backend systems.
