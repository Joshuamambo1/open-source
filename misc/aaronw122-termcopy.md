## 📝 Summary

A small utility that cleans up unwanted line breaks introduced when copying code snippets from Claude (or similar AI chat interfaces). It automatically removes spurious newlines and re‑flows the text into proper, continuous code blocks, making pasted code ready to compile or edit without manual fixing. The project is open‑source, lightweight, and categorized under miscellaneous tools.
