## 📝 Summary

Here's a 2-3 sentence summary for the project:

This open-source project provides a self-hosted MCP (Machine Learning Pipeline) server for Odoo, a popular enterprise management system. The project integrates Claude, an AI-powered chatbot, and Claude Code, enabling users to leverage 197+ tools and features, including multi-tenancy, Bulgaria localization, and support for Odoo versions 15-19. The solution is built using Docker and K3s, making it easy to deploy and manage on various infrastructure setups.
