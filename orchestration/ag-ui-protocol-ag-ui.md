## 📝 Summary

### English

AG-UI is an open‑source protocol that standardizes communication between AI agents and frontend applications, enabling seamless agent‑driven user interactions. It provides a lightweight, extensible framework for orchestrating AI/ML services directly within web interfaces, simplifying integration and enhancing UI responsiveness.

### Русский

AG‑UI (ag‑ui‑protocol/ag‑ui) — открытый протокол взаимодействия агента и пользователя, позволяющий интегрировать интеллектуальные агенты непосредственно в веб‑приложения. Проект упрощает оркестрацию AI/ML‑моделей на клиентской стороне, предоставляя стандартизированный набор API и UI‑компонентов для создания интерактивных и адаптивных интерфейсов.
