## 📝 Summary

doceazedo/godot_wry is an open‑source extension that adds cross‑platform webview capabilities to Godot 4, allowing developers to embed and interact with web content directly within their games or applications. It wraps the WRY library to provide a unified API for Windows, macOS, Linux, and Web, handling rendering, input forwarding, and JavaScript integration. The project is licensed under an open‑source license and can be easily integrated into any Godot 4 project.
