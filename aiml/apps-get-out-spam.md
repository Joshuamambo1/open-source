## 📝 Summary

A lightweight bot monitors new comments on GitHub issues, automatically flagging potential spam and routing suspicious messages to a review queue. Built after the author nearly migrated an issue discussion to Telegram, the tool leverages simple AI/ML heuristics to keep conversations clean without leaving the GitHub interface. It’s open‑source, easy to integrate into any repository, and includes a minimal frontend for reviewing and approving flagged comments.
