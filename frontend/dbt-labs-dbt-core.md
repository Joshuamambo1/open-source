## 📝 Summary

### English

dbt‑core, the open‑source engine behind dbt (data build tool), lets data analysts and engineers define, test, and version‑control data transformations using software‑engineer best practices such as modular SQL, dependency graphs, and CI/CD pipelines. By compiling SQL into ordered, reproducible runs, it streamlines the creation of reliable data models and enables robust data testing, documentation, and deployment across modern data warehouses.
