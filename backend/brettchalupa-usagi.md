## 📝 Summary

BrettChalupa’s **Usagi** is a lightweight 2D game engine designed for rapid prototyping using Lua. It offers live‑code reloading, a straightforward API, and cross‑platform export capabilities, making it ideal for developers who want to iterate quickly and target multiple platforms without heavy boilerplate. The project is open‑source and positioned as a backend solution for fast, script‑driven game development.
