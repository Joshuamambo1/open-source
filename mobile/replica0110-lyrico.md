## 📝 Summary

Here's a brief summary of the project:

Lyrico is an open-source Android music tag editor that allows users to efficiently search and edit song metadata. It also offers word-by-word lyrics support for a seamless music experience. Inspired by LDDC and music tag editors, Lyrico aims to provide a user-friendly interface for music enthusiasts.
