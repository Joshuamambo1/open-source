## 📝 Summary

### English

mexirica/aptui is a lightweight terminal user interface (TUI) for managing packages on APT‑based Linux distributions. It provides an intuitive, ncurses‑styled front‑end that lets users search, install, upgrade, and remove packages without leaving the command line. Designed for speed and simplicity, aptui streamlines common APT tasks while maintaining full compatibility with the underlying package manager.

### Русский

**mexirica/aptui** – это удобный TUI‑интерфейс для управления пакетами в дистрибутивах Linux, основанных на APT. Приложение предоставляет полностью текстовый, но интуитивно понятный пользовательский опыт для поиска, установки, обновления и удаления пакетов, позволяя работать с системой без графических оболочек. Подходит как для серверов, так и для настольных сред, где нужен быстрый и лёгкий фронтенд к APT.
