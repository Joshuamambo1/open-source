## 📝 Summary

### English

Wideawake is an open‑source macOS utility that automatically detects active agents (such as builds, downloads, or remote sessions) and temporarily disables the system’s sleep mode to keep those tasks running uninterrupted. By monitoring system activity and using lightweight heuristics, it ensures your Mac stays awake only when needed, then restores normal power‑saving behavior once the work is finished. The project is hosted on GitHub and is licensed for community contributions.
