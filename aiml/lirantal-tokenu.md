## 📝 Summary

The tokenu project is an open-source, Unix-like command line tool that provides a simple and efficient way to count token usage per files and directories, offering insights into file composition and structure. Built as a du command alternative, tokenu is designed to facilitate data analysis and understanding in various applications, including AI and machine learning development. This tool caters to developers and data scientists who require detailed information about file token distribution.
