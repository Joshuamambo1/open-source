## 📝 Summary

The cry-inc/dmarc-report-viewer is an open-source project that provides a lightweight, standalone application for viewing DMARC and SMTP TLS reports. It features an IMAP client capability, allowing users to easily access and analyze their email authentication reports in a user-friendly interface. This project is ideal for organizations looking to monitor and improve their email security and authentication protocols.
