## 📝 Summary

noborus/trdsql is a command‑line utility that lets you run standard SQL queries directly against data stored in CSV, LTSV, JSON, YAML, or TBLN files, treating them as relational tables. It supports a wide range of output formats—including CSV, JSON, TSV, and pretty‑printed tables—making it easy to transform, filter, and join heterogeneous data sources without writing custom scripts. The tool is useful for developers and data analysts who need quick, SQL‑based data manipulation in a lightweight, open‑source package.
