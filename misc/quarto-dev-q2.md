The q2 project is an experimental implementation of Quarto 2, a next-generation Quarto version, in Rust. This open-source project aims to push the boundaries of Quarto's capabilities while providing a cutting-edge, modern, and high-performance framework for document authoring and publishing. By leveraging Rust's strengths, q2 seeks to deliver improved speed, reliability, and flexibility.
