## 📝 Summary

Here's a brief summary of the project in English:

The Fitzgerald-Porthmouth-Koenigsegg/Plangothic Project is an open-source initiative that supplements the Unified Ideographs Extension (CJK Unified Ideographs Extension) with additional Chinese, Japanese, Korean, and Vietnamese character shapes, standardizing on Simplified Chinese glyph forms. This project builds upon the Source Han Sans font family. Its primary goal is to provide a comprehensive and consistent character set for these languages.
