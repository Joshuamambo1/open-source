## 📝 Summary

The opencode-bar is an open-source project that provides an effortless token usage tracker for OpenCode users, offering them a convenient and streamlined way to monitor their token activity. This project is designed to make managing OpenCode tokens easier and more accessible, saving users time and effort in tracking their usage. By utilizing the opencode-bar, users can gain a clearer understanding of their token expenditure.
