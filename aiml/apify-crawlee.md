> Crawlee—A web scraping and browser automation library for Node.js to build reliable crawlers. In JavaScript and TypeScript. Extract data for AI, LLMs, RAG, or GPTs. Download HTML, PDF, JPG, PNG, and other files from websites. Works with Puppeteer, Playwright, Cheerio, JSDOM, and raw HTTP. Both headful and headless mode. With proxy rotation.

## 📝 Summary

Crawlee is an open-source, Node.js library for web scraping and browser automation, enabling the development of reliable crawlers for extracting data from websites. With support for Puppeteer, Playwright, and other popular libraries, Crawlee allows users to download various file types, including HTML, PDF, and images, while also providing proxy rotation capabilities. This versatile library is suitable for applications involving AI, LLMs, RAGs, and GPTs, making data collection a seamless process.
