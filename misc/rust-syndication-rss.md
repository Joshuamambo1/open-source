## 📝 Summary

### English

rust‑syndication/rss is a lightweight, open‑source Rust library that provides full support for the RSS 2.0 (and related) web content syndication format. It enables developers to effortlessly parse, generate, and serialize RSS feeds with a type‑safe API, handling all required XML structures and namespaces. Ideal for building feed readers, publishers, or any application that needs to work with RSS data in a safe and idiomatic Rust manner.
