## 📝 Summary

Here's a brief summary for the VirtusLab/sandcat project:

sandcat is an open-source project that provides a secure Docker and dev container setup for running AI agents in a controlled environment. By routing all container traffic through a transparent proxy, it enforces network access rules and injects secrets to safeguard agent interactions. This setup is ideal for developers working with AI/ML projects that require a high level of security and access control.
