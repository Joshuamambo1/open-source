## 📝 Summary

Here's a brief summary of Mano-P:

Mano-P is an open-source, locally-run GUI-VLA (Visual Learning Agent) project that enables pure visual-driven GUI automation on various platforms. It runs independently on Apple M4 Mac mini/MacBook devices, processing data entirely locally without any data leaving the device. This allows for complex multi-step task planning and execution in a fully secure and private environment.
