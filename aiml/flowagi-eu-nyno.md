Here's a brief summary for the open-source project:

Nyno is an open-source, commercial-friendly AI workflow builder and engine that enables users to create and run workflows without limits. With its asynchronous and multi-process capabilities, Nyno facilitates efficient automation, utilizing a Postgres database for seamless data management. This project caters to developers and organizations seeking a scalable, no-code solution for complex AI and data workflows.
