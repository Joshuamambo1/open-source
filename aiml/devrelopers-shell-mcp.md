## 📝 Summary

Shell-MCP is an open-source project that enables per-directory shell allowlisting for the Claude Desktop. This allows users to specify which shells are allowed to run in each directory, providing an additional layer of security and customization. By enabling fine-grained control over shell execution, users can better manage their system's security and productivity.
