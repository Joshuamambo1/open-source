### 中文

**项目简介**  
`material-extensions/vscode-material-icon-theme` 为 Visual Studio Code 提供基于 Material Design 风格的文件图标主题，界面美观、配色统一，能够显著提升开发者的视觉体验和文件辨识效率。

**价值**  
- **统一视觉语言**：使用 Google Material Design 规范的图标，使工作区看起来更专业、整洁。  
- **提升效率**：直观的图标帮助快速定位文件类型，减少在文件列表中搜索的时间。  
- **社区认可**：拥有 2.8k+ ⭐、近 800 个 Fork，且仍在活跃维护，适合作为团队统一的 UI 规范。

**典型接入方式**  
1. 在 VS Code 扩展市场搜索并安装 “Material Icon Theme”。  
2. 安装后打开 VS Code 设置（`Ctrl+,`），搜索 `material-icon-theme`，将 `File Icon Theme` 设为 **Material Icon Theme**。  
3. 如需在项目中统一配置，可在工作区 `.vscode/settings.json` 中加入：  
   ```json
   {
     "workbench.iconTheme": "material-icon-theme"
   }
   ```  
4. 对特殊文件或文件夹的图标进行自定义时，可参考仓库的 `icon-manifest.json` 进行覆盖。

- **活跃度**：最近一次提交为 2026‑06‑23，维护者仍在持续更新。  
- **生态兼容**：基于 TypeScript 编写，兼容所有主流 VS Code 版本（≥1.70）。  
- **安全/许可证**：采用 MIT 许可证，符合大多数企业合规要求；代码体积小，安全风险低。
