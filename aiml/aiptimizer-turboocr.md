## 📝 Summary

TurboOCR is an open-source, high-performance GPU-based OCR (Optical Character Recognition) server. Built with TensorRT FP16 and utilizing the PP-OCRv5 model, it is capable of processing up to 270 images per second on the FUNSD dataset. TurboOCR offers flexible APIs via HTTP and gRPC for seamless integration with various applications.
