## 📝 Summary

Here's a brief summary for the project:

"juanluispaz/ts-sql-query is an open-source, type-safe SQL query builder for TypeScript, offering support for various database systems including MariaDB, MySQL, Oracle, PostgreSQL, SQLite, and SQL Server. This library provides a powerful and intuitive API, similar to QueryDSL or JOOQ in Java, or Linq in .NET, to simplify database interactions and reduce errors. It enables developers to construct and execute SQL queries in a safe and efficient manner, making it an ideal tool for both frontend and backend projects."
