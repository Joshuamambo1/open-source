## 📝 Summary

### English

MewUI is a lightweight, code‑first GUI framework for .NET that targets cross‑platform native applications, with special emphasis on NativeAOT compilation for fast, self‑contained binaries. It lets developers build modern user interfaces entirely in C# without relying on XAML or external UI designers, making it ideal for performance‑critical desktop and embedded scenarios. The project also includes AI/ML‑friendly components, enabling seamless integration of intelligent features into the UI.
