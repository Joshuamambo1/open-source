## 📝 Summary

### English

A lightweight Rust library that brings the Handlebars templating language to the Rust ecosystem, allowing developers to render dynamic content using familiar Handlebars syntax. It provides a straightforward API for compiling, registering, and rendering templates, with support for helpers, partials, and custom data contexts. The project is open‑source and positioned as a versatile tool for any Rust application that needs flexible, server‑side templating.

### Русский

`handlebars-rust` — это открытая библиотека на Rust, реализующая шаблонизатор Handlebars. Она позволяет быстро генерировать текстовые представления (HTML, JSON и т.п.) с помощью знакомого синтаксиса {{…}} и поддерживает расширения, пользовательские хелперы и компиляцию шаблонов в байт‑код для повышения производительности. Проект находится в категории Misc и предназначен для удобного внедрения шаблонов Handlebars в любые Rust‑проекты.
