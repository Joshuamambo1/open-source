## 📝 Summary

### English

The project provides a lightweight tool for projecting high‑dimensional language‑model embeddings onto a 3‑D sphere, enabling interactive visual exploration of semantic relationships among tokens, sentences, or documents. By leveraging dimensionality‑reduction techniques and WebGL rendering, it lets users rotate, zoom, and cluster embeddings in real time, making it easier to interpret and debug large LLMs. The code is open‑source, well‑documented, and ready for integration into AI/ML research workflows.
