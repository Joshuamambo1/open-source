## 📝 Summary

### English

Boltffi is an open‑source, high‑performance bindings generator that lets developers expose Rust libraries to multiple languages—including Swift, Kotlin, and others—with speeds up to 1,000 × faster than UniFFI. By automating the creation of native‑looking APIs, it enables seamless integration of Rust code into mobile apps while preserving idiomatic language ergonomics. The project targets mobile developers who need fast, reliable cross‑language interoperability.

### Русский

Boltffi — это высокопроизводительный генератор привязок для Rust, позволяющий без труда экспортировать библиотеки Rust в нативные модули Swift, Kotlin и другие мобильные платформы. По сравнению с UniFFI он работает до 1000 раз быстрее, обеспечивая низкую задержку и небольшие размеры бинарных файлов. Проект ориентирован на разработчиков мобильных приложений, которым нужен быстрый и надёжный способ интеграции Rust‑кода в iOS и Android.
