Gemma Chat is an open‑source, offline‑first chatbot tailored for Apple‑silicon Macs, allowing developers to run large language models locally without an internet connection. Built with a clean, design‑focused UI, it streamlines the setup and interaction process so users can code, prototype, and experiment on‑device while keeping data private.
