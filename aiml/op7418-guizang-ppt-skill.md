## 📝 Summary

The guizang-ppt-skill project is an open-source, AI-powered tool that generates interactive, horizontal-swipe magazine-style HTML decks from user prompts. Developed using Claude Code, this skill offers 10 customizable layouts and 5 curated themes, along with WebGL hero backgrounds, making it suitable for presentations and storytelling. The project produces a single-file output, simplifying the deployment process.
