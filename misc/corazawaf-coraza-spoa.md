## 📝 Summary

Here's a brief summary for the open-source project:

Coraza SPOA is an open-source project that integrates the OWASP Coraza Web Application Firewall (WAF) with HAProxy's SPOE filters, enabling robust web application security and scalability. This wrapper allows for seamless integration of Coraza's security features with HAProxy's load balancing capabilities. By combining these tools, users can protect their web applications from common vulnerabilities and attacks.
