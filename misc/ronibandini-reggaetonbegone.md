## 📝 Summary

The reggaetonBeGone project is an open-source initiative that leverages machine learning to identify and detect reggaeton music. Once detected, it sends packets to disable Bluetooth speakers, aiming to provide a humorous solution to reggaeton enthusiasts. The project's primary function is to poke fun at the reggaeton genre through technical means.
