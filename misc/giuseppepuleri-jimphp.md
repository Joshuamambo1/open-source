## 📝 Summary

The project is a personal learning exercise in which the author builds a fully functional PHP interpreter from scratch using C++. By re‑implementing PHP’s parsing, compilation, and execution stages, the author explores language design, memory management, and runtime behavior while sharpening their C++ and systems‑programming skills. The repository serves as a reference implementation and a showcase of the challenges involved in creating a high‑level scripting language interpreter.
