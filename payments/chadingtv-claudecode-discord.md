## 📝 Summary

Here's a 2-3 sentence summary of the project:

ClaudeCode-Discord is an open-source project that enables users to control and manage their Claude Code multi-machine agent hub directly from their mobile devices through Discord. This platform leverages the user's existing Claude Pro/Max subscription, eliminating the need for an API key. With ClaudeCode-Discord, users can start new sessions and manage multiple machines from a unified interface on their phone.
