## 📝 Summary

Here's a 2-3 sentence summary of the project:

Ember is an open-source project that provides a real-time terminal dashboard for Caddy servers, offering a comprehensive and interactive interface for server monitoring and management. This project also includes built-in support for FrankenPHP, a PHP execution environment compatible with Caddy. Ember allows users to easily track server performance and configure settings in a user-friendly and efficient manner.
