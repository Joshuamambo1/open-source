## 📝 Summary

Here's a brief summary for the prysm project:

Prysm is an open-source Python library for physical optics, providing a comprehensive framework for integrated modeling and analysis of optical systems. It includes tools for phase retrieval, segmented systems, polynomial fitting, and sequential ray tracing, enabling users to simulate, optimize, and troubleshoot complex optical designs. With prysm, researchers and engineers can efficiently develop and refine optical systems for various applications.
