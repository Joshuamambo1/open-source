## 📝 Summary

Nocord HF is an open-source, Discord-style client for FT8 communication, built using the Golang programming language. This project provides a user-friendly interface for amateur radio operators to engage in FT8 digital mode communication. By leveraging Golang, Nocord HF offers a lightweight and efficient solution for FT8 enthusiasts.
