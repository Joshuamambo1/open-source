## 📝 Summary

### English

**Summary:**  
This repository provides all the supporting materials for the “Write a RISC‑V OS from Scratch” (《从头写一个RISC-V OS》) online course. It includes lecture notes, example code, build scripts, and other resources needed to develop a minimal operating system for the RISC‑V architecture, making it a practical learning platform for students and hobbyists.
