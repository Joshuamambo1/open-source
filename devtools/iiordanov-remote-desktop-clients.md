## 📝 Summary

This open-source project, remote-desktop-clients, provides a collection of desktop clients for various remote desktop protocols, including VNC, RDP, SPICE, and oVirt/RHEV/Proxmox. It supports both Android and BlackBerry 10 platforms, offering users a range of options for remote access from their mobile devices. The project aims to extend access to remote desktop environments across different mobile ecosystems.
