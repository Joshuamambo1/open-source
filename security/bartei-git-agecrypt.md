## 📝 Summary

Git-agecrypt is an open-source project that brings transparent file-level encryption to Git repositories. This tool enables developers to securely store sensitive data in their Git repositories while maintaining the ease of use and version control that Git provides. By encrypting files at the file level, Git-agecrypt ensures that even if a repository is compromised, sensitive information remains protected.
