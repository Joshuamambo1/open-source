## 📝 Summary

The hazmat project is an open-source solution for macOS containment of AI agents, providing user isolation, a kernel sandbox, and robust security features such as a pf firewall and DNS blocklist. This project aims to ensure the safe and controlled execution of AI agents, leveraging formal verification using TLA+ to guarantee its security. Hazmat also includes backup and rollback capabilities for added reliability.
