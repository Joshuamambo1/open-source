## 📝 Summary

Seance is an open-source, scrolling terminal multiplexer designed to track AI coding agents. It enables developers to monitor and manage multiple AI-powered coding tools simultaneously within a single terminal interface, enhancing productivity and streamlining the development process. By providing a centralized hub for AI coding agents, Seance simplifies the workflow and boosts collaboration between humans and AI in code development.
