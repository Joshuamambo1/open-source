### 中文

**项目简介**  
chatwoot/chatwoot-mobile-app 是一款基于 React Native 的开源移动客户端，专为 Chatwoot（开源客服系统）打造，帮助开发者快速交付用户界面，减少自定义 UI 的工作量。

**价值**  
- **加速 UI 开发**：提供即插即用的聊天、工单等界面组件，开发者可直接复用，显著缩短前端交付周期。  
- **统一体验**：与 Chatwoot 后端保持一致的交互逻辑和视觉风格，确保 Web 与移动端的用户体验一致。  
- **社区与生态**：拥有 961+ ⭐、486+ 🍴，活跃的贡献者和持续更新，适合作为生产级 OSS 组件。

**典型接入方式**  
1. **阅读 README**：确认兼容的 React Native 版本及必需的依赖（如 `@react-native-firebase/messaging`、`react-navigation` 等）。  
2. **创建小型 PoC**：在现有 RN 项目中通过 `npm i @chatwoot/mobile-app`（或克隆仓库）引入核心模块，配置 Chatwoot API URL 与访问令牌。  
3. **自定义 UI**：在需要的页面中使用提供的 `ChatwootProvider` 包裹根组件，随后直接使用 `<ConversationList/>`、`<ConversationScreen/>` 等预置组件，或在此基础上进行主题化定制。  
4. **本地调试 & CI**：运行 `yarn ios` / `yarn android` 验证功能，加入自动化测试后再推广到正式环境。

**生产可用性**  
- **活跃度**：最近一次提交于 2026‑07‑01，代码基于 TypeScript，维护频率高。
