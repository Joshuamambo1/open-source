## 📝 Summary

The dotlottie-rs project is an open-source, high-performance player for Lottie and dotLottie animations built with Rust. This universal player offers seamless rendering across various platforms, utilizing low resources while maintaining extensive compatibility with different development frameworks. It provides FFI bindings for several languages and platforms, facilitating integration in Android, iOS, and Web projects.
