Here's a brief summary of the project:

DevGuard is an open-source, compliance-as-code solution for securing software supply chains. It provides attestation-based compliance, seamless CVE management, and integrates with vulnerability scanners, making it easier to maintain security frameworks. As an OWASP Incubating Project, DevGuard aims to promote secure software development practices.
