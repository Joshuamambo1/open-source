## 📝 Summary

jinleili/wgpu-in-app is an open‑source library that demonstrates how to embed the **wgpu** graphics API into native iOS and Android applications. It provides sample code and integration guidelines for setting up wgpu contexts, handling surface creation, and rendering frames within existing mobile app projects. The project is aimed at developers who want to leverage modern GPU‑accelerated graphics on both platforms without rewriting their apps from scratch.
