## 📝 Summary

OpenSK is an open-source project that provides a secure implementation of security keys, enabling users to protect their online accounts with strong authentication. Built with Rust, this project supports both FIDO U2F and FIDO2 standards, providing robust security for various applications. By making this technology open-source, OpenSK promotes secure and private authentication solutions for users worldwide.
