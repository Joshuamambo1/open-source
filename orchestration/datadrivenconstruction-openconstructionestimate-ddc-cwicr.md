## 📝 Summary

### English

OpenConstructionEstimate‑DDC‑CWICR is an open, multilingual construction‑cost database designed for AI agents, containing over 55 000 work items, 27 000+ resource entries, and cost data for 30 geographic regions. The dataset is indexed in a Qdrant vector store, enabling fast semantic search and easy integration into orchestration, AI/ML, and data‑driven workflows. It provides a ready‑to‑use knowledge base for cost estimation, budgeting, and analytics in construction projects worldwide.
