## 📝 Summary

This open-source project provides a minimalist offline version of Wikipedia, featuring all text content and proper math rendering capabilities. The project allows users to access Wikipedia articles without an internet connection, making it a valuable resource for those in areas with limited connectivity. It leverages the power of open-source technology to bring Wikipedia's vast knowledge base to users in a self-contained package.
