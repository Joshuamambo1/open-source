## 📝 Summary

### English

A lightweight .NET library that creates QR codes from arbitrary text or binary data, offering simple APIs for generating PNG, JPEG, or SVG images. It supports customization of size, error correction level, and color, making it easy to integrate QR code generation into any .NET application.
