## 📝 Summary

### English

djc/tokio-imap is an open‑source library that provides a fully asynchronous IMAP client built on top of Tokio, Rust’s high‑performance async runtime. It implements the IMAP protocol (including support for IDLE, STARTTLS, and various extensions) allowing developers to interact with mail servers efficiently without blocking. The crate is designed for easy integration into Rust applications that require non‑blocking email handling.
