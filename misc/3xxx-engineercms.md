## 📝 Summary

Engineercms is a web‑based knowledge‑management platform for civil engineers, built with Go (Beego) and designed for distributed file storage via a MinIO cluster. It lets users organize personal and project documents, share files with access codes, collaborate on documents in real time through OnlyOffice, create books with Mindoc, and view PDFs, while also offering a companion mobile mini‑program accessible via WeChat.
