> 蓝鲸作业平台（Job）是一套运维基础操作管理系统，具备海量任务并发处理能力。除了支持脚本执行、文件分发、定时任务等一系列基础运维场景以外，还支持通过流程调度能力将零碎的单个任务组装成一个自动化作业流程；而每个作业都可做为一个原子节点，提供给上层或周边系统/平台使用，实现调度自动化。

## 📝 Summary

Here's a brief summary of the TencentBlueKing/bk-job project:

BlueKing Job is an open-source operations management system that enables high-performance concurrent task processing. It offers a range of basic operations management capabilities, including script execution, file distribution, and scheduling, as well as more advanced features like workflow orchestration and automation. This system allows for the composition of individual tasks into automated workflows, making it a powerful tool for automation and scheduling.
