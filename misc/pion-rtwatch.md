## 📝 Summary

Rtwatch is an open‑source web application that lets groups of friends synchronously watch online videos together using WebRTC for real‑time, peer‑to‑peer streaming and chat. It provides a simple interface to create rooms, share video URLs, and enjoy low‑latency playback without relying on a central server for media delivery. The project is lightweight, cross‑platform, and licensed for community contributions.
