## 📝 Summary

SwiftScript is an open-source project that provides a tree-walking interpreter, library, and command-line interface (CLI) for the Swift programming language. This project enables users to execute Swift code directly in the terminal, making it a powerful tool for developers, educators, and hobbyists alike. With SwiftScript, users can write and run Swift code on the fly, fostering experimentation, learning, and prototyping.
