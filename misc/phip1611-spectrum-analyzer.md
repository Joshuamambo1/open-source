## 📝 Summary

The spectrum-analyzer library is an open-source, no-std library that utilizes the alloc crate to efficiently compute the frequency spectrum of digital signals, such as audio, via the Fast Fourier Transform (FFT) algorithm. This library aims to provide a simple and fast solution for signal processing tasks. It can be easily integrated into various applications requiring real-time frequency analysis.
