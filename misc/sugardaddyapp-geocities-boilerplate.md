## 📝 Summary

A lightweight boilerplate that lets you quickly spin up a nostalgic 1990s‑style Geocities website, complete with retro fonts, bright background patterns, animated GIFs, and classic HTML table layouts. It includes ready‑made CSS, sample page templates, and a simple build script so developers can focus on adding their own quirky content while preserving the era’s unmistakable aesthetic.
