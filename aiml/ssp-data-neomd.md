## 📝 Summary

The ssp-data/neomd project is an open-source, minimal email Terminal User Interface (TUI) that allows users to read emails formatted in Markdown. Notably, it integrates with Neovim for composing emails, providing a seamless writing experience. This project combines the efficiency of a TUI with the power of a text editor, making email management more efficient and streamlined.
