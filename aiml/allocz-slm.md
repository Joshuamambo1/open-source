## 📝 Summary

SLM is an open-source, zero-dependency text-based user interface (TUI) chat application that utilizes a Large Language Model (LLM) to provide intelligent and interactive conversations. Developed as a simple yet powerful solution, SLM enables users to engage in natural language-based interactions without requiring any external dependencies. This project showcases a straightforward implementation of LLM-based chat functionality in a TUI environment.
