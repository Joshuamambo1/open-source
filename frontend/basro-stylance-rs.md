## 📝 Summary

### English

**stylance-rs** is an open‑source Rust library that enables scoped CSS style imports, allowing developers to attach CSS modules directly to Rust components with automatic name‑spacing to prevent style collisions. By parsing and bundling CSS at compile time, it delivers zero‑runtime overhead while preserving the ergonomics of modern frontend workflows. Ideal for Rust‑based UI frameworks, it streamlines styling without sacrificing performance or safety.
