## 📝 Summary

**Summary:**  
**graphify-dotnet** is an open‑source .NET 10 port of the original *graphify* tool, enabling developers to automatically generate AI‑driven knowledge graphs of their codebases. Leveraging the GitHub Copilot SDK and Microsoft.Extensions.AI, it extracts semantic relationships between components and visualizes them, making code navigation, documentation, and impact analysis faster and more intuitive.
