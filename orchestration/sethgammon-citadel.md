## 📝 Summary

The Citadel project is an open-source agent orchestration harness designed for Claude Code, enabling the creation of scalable and complex AI/ML workflows. It features a four-tier routing system, campaign persistence, and parallel agent execution, ensuring seamless integration and high performance. Citadel supports institutional-scale development, making it suitable for solo developers and large-scale teams alike.
