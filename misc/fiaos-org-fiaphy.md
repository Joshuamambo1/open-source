Derive any environmental variable with **FiaPhy** – a lightweight, open‑source utility that lets you compute new environment variables from existing ones using simple expressions and transformations. It parses the current process’s environment, applies user‑defined derivation rules, and outputs the resulting variables for seamless integration into scripts, CI pipelines, or container setups.
