## 📝 Summary

The brentp/echtvar project is an open-source initiative focused on rapidly annotating and filtering genetic variants using all available data. Its goal is to provide accurate and efficient processing of genomic information, enabling researchers to gain insights into the underlying mechanisms of genetic diseases. This project aims to streamline variant analysis and improve the understanding of variant effects.
