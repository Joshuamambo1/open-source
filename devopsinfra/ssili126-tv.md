## 📝 Summary

Here's a brief summary of the open-source project:

The tv project by ssili126 automatically collects and tests IPv4 hotel TV streaming sources, ensuring smooth playback. It includes a range of channels, including CCTV national channels and some local stations, with daily updates. This project can be run on OpenWRT or Gaoxin using Docker.
