> SirixDB is an an embeddable, bitemporal, append-only database system and event store, storing immutable lightweight snapshots. It keeps the full history of each resource. Every commit stores a space-efficient snapshot through structural sharing. It is log-structured and never overwrites data. SirixDB uses a novel page-level versioning approach.

## 📝 Summary

SirixDB is an open-source, embeddable database system that stores immutable snapshots of data, allowing for efficient management of historical data. This bitemporal database uses a novel page-level versioning approach and a log-structured design to ensure data integrity and prevent overwrites. It is particularly suitable for applications involving AI/ML, data storage, and event-driven systems.
