## 📝 Summary

Matrirc is an open-source project that enables users to run the popular IRC client, irssi, in 2026 and communicate with people on the Matrix network. This project bridges the gap between two distinct messaging protocols, allowing users to interact with individuals on both IRC and Matrix. By integrating irssi with Matrix, Matrirc provides a seamless experience for users to connect with others across different platforms.
