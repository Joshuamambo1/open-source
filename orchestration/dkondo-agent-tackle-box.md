## 📝 Summary

### English

The **agent‑tackle‑box** is an open‑source toolkit for building and debugging AI agents, featuring a powerful terminal‑based debugger that works with LangGraph and LangChain agents. It lets developers inspect agent state, monitor tool calls, set semantic breakpoints, and step through Python code—all within a single Textual UI—making the development, testing, and orchestration of LLM‑driven agents faster and more transparent.
