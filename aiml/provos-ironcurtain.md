## 📝 Summary

The ironcurtain project is an open-source initiative that aims to create a secure runtime environment for autonomous AI agents. This runtime is guided by plain-English constitutions, which serve as policies for the AI's behavior. By leveraging this framework, developers can build trustworthy and secure AI systems that adhere to human-defined values and principles.
