## 📝 Summary

Kairo is an open-source terminal task manager built with Go, designed for speed and keyboard-first interaction. It features offline-first SQLite storage, Git synchronization, and fuzzy search capabilities, making it a powerful tool for managing tasks. Additionally, Kairo supports Lua plugins for further customization and extension.
