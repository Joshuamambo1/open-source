## 📝 Summary

Cyrus is an open-source, high-performance programming language designed for mid-level applications. It boasts a statically-typed and manually memory-managed approach, providing developers with fine-grained control over memory allocation. The project is currently under active development, aiming to deliver a robust and efficient language for complex software development needs.
