## 📝 Summary

Castor6 / tactus is a browser extension that introduces the first AI Agent capable of leveraging “Agent Skills,” an extensible framework that lets the AI execute complex, multi‑step tasks directly within the browser. By modularizing functionality into reusable skills, the extension enables customizable, orchestrated workflows for a wide range of web‑based automation scenarios.
