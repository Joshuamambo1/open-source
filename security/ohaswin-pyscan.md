## 📝 Summary

The pyscan project is an open-source, Python dependency vulnerability scanner built in Rust. It scans Python projects to identify and report potential vulnerabilities in dependencies, helping developers maintain the security and integrity of their applications. With its Rust foundation, pyscan offers a robust and efficient scanning solution for Python projects.
