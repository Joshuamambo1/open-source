## 📝 Summary

Here's a brief summary for the open-source project:

The YooMoney API client is an open-source Python library providing a seamless integration with the YooMoney (ex-Yandex.Money) API, supporting both synchronous and asynchronous interactions. It utilizes Pydantic models to ensure type safety and facilitates secure, reliable transactions. This project empowers developers to easily build and manage payment solutions within their applications.
