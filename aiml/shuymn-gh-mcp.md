## 📝 Summary

### English

The **gh‑mcp** extension lets you start the `github-mcp-server` directly from the GitHub CLI, reusing the authentication already configured for `gh`. It automatically fetches your GitHub credentials (avoiding the need to create and manage a personal access token) and launches the MCP server with the proper auth, making local AI/ML workflows and backend integrations much smoother.
