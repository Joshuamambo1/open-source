## 📝 Summary

This open-source project provides a reproducible implementation of a Silver Medal-winning solution to the NeurIPS Open Polymer Prediction 2025 Kaggle competition, utilizing a combination of machine learning models and techniques. The project leverages SMILES canonicalization, molecular descriptors, and ensemble methods such as CatBoost and XGBoost, with the option to incorporate PyTorch Geometric GNNs for graph-based modeling. This implementation aims to facilitate reproducibility and improve the accuracy of polymer property predictions.
