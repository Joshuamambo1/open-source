## 📝 Summary

This open-source project provides a visually appealing Text User Interface (TUI) for analyzing Go Pprof data. The TUI offers an intuitive way to explore and understand profiling data, making it easier to identify performance bottlenecks in Go applications. By leveraging the power of Pprof, developers can now visualize and debug their Go code more effectively.
