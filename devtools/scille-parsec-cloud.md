## 📝 Summary

Scille/parsec-cloud is an open-source file sharing project that provides a Dropbox-like experience with end-to-end encryption, ensuring the security and confidentiality of shared files. This decentralized solution allows users to store and access their files securely, while maintaining full control over their data. By leveraging full client encryption, users can share files with confidence, knowing their data remains protected.
