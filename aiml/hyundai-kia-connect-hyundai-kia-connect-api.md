## 📝 Summary

Here's a brief summary for the open-source project:

Hyundai-Kia-Connect/hyundai_kia_connect_api is an open-source Python library that provides an interface to Kia UVO and Hyundai Bluelink APIs. It enables integration with home automation systems and other applications, serving as a foundation for projects like the Kia UVO Home Assistant implementation. This project combines knowledge from Bluelinky and contributions to the Kia UVO project to provide a comprehensive and community-driven API solution.
