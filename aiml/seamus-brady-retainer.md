Retainer is an open‑source autonomous agent framework designed to run for extended periods without human supervision. It combines advanced language‑model reasoning, tool‑use, and state‑management components so the agent can plan, execute, and adapt to changing goals over long‑term tasks. Ideal for developers building self‑sustaining AI assistants, Retainer offers plug‑and‑play modules and a lightweight runtime that can be deployed on any cloud or edge environment.
