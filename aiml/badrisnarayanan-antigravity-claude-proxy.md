## 📝 Summary

The Antigravity Claude Proxy is an open-source project that provides a proxy for accessing Antigravity's claude and gemini models. This allows developers to utilize these models in Claude code and other applications, such as OpenClaw (Clawdbot), simplifying their integration and usage. By leveraging this proxy, users can tap into the capabilities of Antigravity's AI models with greater ease and flexibility.
