## 📝 Summary

Claude-Replay is an open-source project that enables the conversion of AI coding agent sessions, including Claude Code, Cursor, Codex, Gemini, and OpenCode, into interactive HTML replays. This allows for the creation of self-contained, embeddable demonstrations of AI coding capabilities. The project caters to developers and researchers working with AI/ML tools, providing a valuable resource for sharing and exploring AI-generated code.
