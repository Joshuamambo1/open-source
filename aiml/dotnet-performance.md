The dotnet/performance project is an open-source repository that houses benchmarks used to evaluate the performance of all .NET runtimes. This collection of performance tests enables developers to assess and compare the efficiency of various .NET environments, driving improvements in the platform's overall performance. By leveraging these benchmarks, the .NET community can contribute to the optimization and refinement of the .NET ecosystem.
