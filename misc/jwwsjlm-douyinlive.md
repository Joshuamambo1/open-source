## 📝 Summary

The douyinLive project is an open-source tool for extracting live comments from Douyin (TikTok's Chinese counterpart). It aims to provide a convenient way to collect and analyze live stream data. This project can be useful for researchers, analysts, or enthusiasts interested in understanding user behavior and engagement patterns on Douyin.
