## 📝 Summary

### English

ClawProBench is an open‑source benchmarking harness designed to evaluate large language model (LLM) agents within the OpenClaw runtime. It provides live‑first testing with deterministic grading and supports repeated‑trial runs to ensure reliable, reproducible performance metrics. The tool is geared toward researchers and developers building AI/ML agents who need consistent, automated assessments of their models.
