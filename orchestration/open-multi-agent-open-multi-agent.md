### English

Here's a brief summary of the open-source project:

**Summary:** open-multi-agent is an open-source, TypeScript-based multi-agent orchestration framework that enables the creation of repeatable agent workflows by decomposing complex tasks into task DAGs (Directed Acyclic Graphs) that can be executed on various Large Language Models (LLMs) or local models. This framework helps turn isolated prompts and tools into standardized agent workflows, making it an essential tool for coordinating multi-agent workflows and standardizing agent memory.

**Value Proposition:** The primary value of open-multi-agent lies in its ability to streamline the process of creating and executing complex agent workflows by breaking them down into manageable, executable tasks. This reduces the effort required to integrate multiple tools and LLMs, making it easier to build and maintain sophisticated AI/ML applications.

2. **Setup**: Install and configure the framework, choosing the desired LLM or local model for task execution.
