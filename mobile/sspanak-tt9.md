sspanak/tt9 is an open‑source Android application that brings back the classic T9 predictive text input used on early 2000s Nokia phones. It provides a lightweight, customizable keyboard that lets users type quickly by pressing each key a few times, mimicking the familiar multi‑tap and word‑prediction experience. The project targets mobile users who prefer a nostalgic, efficient alternative to modern full‑QWERTY keyboards.
