## 📝 Summary

The bcoles/kasld project is an open-source tool designed to bypass Kernel Address Space Layout Randomization (KASLR) by mapping the Linux kernel's virtual and physical memory layout as an unprivileged local user. This allows for the determination of the kernel's base address, which can be exploited for various purposes. The project aims to provide a practical solution for defeating KASLR.
