## 📝 Summary

WfmOxide is an open‑source Rust library that parses the proprietary `.wfm` and `.isf` file formats—commonly used for waveform and timing data in electronic design automation—while providing a lightweight command‑line interface for quick inspection and manipulation. The tool extracts signal values and timestamps, builds a time axis, and outputs the data in human‑readable or machine‑friendly formats, making it a handy utility for developers and engineers who need to work with waveform files without proprietary software.
