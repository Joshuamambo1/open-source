## 📝 Summary

### English

OrchardCore.Commerce is an open‑source commerce module built for the Orchard Core content management framework. It provides a flexible foundation for creating e‑commerce sites, handling product catalogs, shopping carts, order processing, and integration with multiple payment providers. Designed as a plug‑in for Orchard Core, it lets developers extend and customize the full shopping experience while leveraging the platform’s modular architecture.
