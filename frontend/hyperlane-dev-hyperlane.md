## 📝 Summary

Hyperlane is a lightweight, high‑performance Rust library for building HTTP servers that runs on any platform supported by Tokio. It streamlines modern web service development with first‑class support for middleware, WebSocket, Server‑Sent Events (SSE), and raw TCP connections, making it suitable for both frontend and backend applications.
