## 📝 Summary

### English

OtterWiki is a lightweight, open‑source wiki engine written in Python that stores each page as a plain Markdown file in a Git repository. By leveraging Git for version control and Markdown for content formatting, it provides a simple, file‑based workflow for collaborative editing, history tracking, and easy deployment. The project targets users who need a minimalistic, self‑hosted documentation solution without the overhead of a full‑blown wiki platform.
