## 📝 Summary

Borsapy is an open-source Python library providing access to data and analytics for the Turkish financial markets, including BIST stocks, forex, crypto, funds, and inflation. It aims to facilitate the integration of Turkish financial data into various applications, such as trading platforms, data visualizations, and machine learning models. By leveraging Borsapy, developers can efficiently tap into a comprehensive range of Turkish market data.
