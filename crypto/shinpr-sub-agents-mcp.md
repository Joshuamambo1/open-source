## 📝 Summary

shinpr/sub-agents-mcp is an open‑source framework that lets developers describe task‑specific AI sub‑agents using simple Markdown files, which are then automatically translated into executable agents for any MCP‑compatible tool. By separating agent definitions from code, the project streamlines the creation, orchestration, and deployment of specialized AI components across crypto, orchestration, and broader AI/ML workflows.
