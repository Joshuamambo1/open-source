## 📝 Summary

The Kai project is an open-source, locally-run AI coding assistant for Telegram, designed to provide users with maximum control and security. It leverages multiple persistent backends, offering features like shell access, automated PR reviews, layered security, and scheduled jobs. By running on a user's machine, Kai ensures that sensitive data remains private and under the user's control.
