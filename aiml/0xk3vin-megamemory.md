## 📝 Summary

The MegaMemory project is an open-source knowledge graph for coding agents that enables persistent storage and retrieval of project knowledge. It features an MCP (MegaMemory Connector Protocol) server with semantic search capabilities, in-process embeddings, and a web explorer for easy navigation and visualization. This project aims to facilitate more efficient and effective knowledge sharing and management among coding agents.
