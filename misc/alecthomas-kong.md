## 📝 Summary

### English

Kong is a lightweight, extensible command‑line parser library for the Go programming language. It enables developers to define commands, flags, and arguments declaratively through struct tags, automatically generating usage help, validation, and subcommand handling. The project is open‑source and aims to simplify the creation of robust CLI applications.

### Русский

Kong — это легковесный и расширяемый парсер командной строки, написанный на Go. Он предоставляет декларативный API для описания флагов, подкоманд и их вложенной структуры, автоматически генерируя справку и поддерживая валидацию вводимых аргументов. Проект подходит для любых Go‑приложений, где требуется надёжный и удобный CLI.
