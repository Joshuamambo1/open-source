## 📝 Summary

Diesel-Guard is an open-source linter tool designed to prevent downtime caused by unsafe schema changes in PostgreSQL migrations using Diesel and SQLx. It identifies and flags dangerous patterns in migration code, ensuring a safer and more predictable database environment. By using Diesel-Guard, developers can catch potential issues before they cause downtime and data loss.
