## 📝 Summary

### English

Statewave is an open‑source memory runtime designed for AI agents, providing durable episode storage, compiled memories, and a ranked context‑retrieval system. It runs self‑hosted on PostgreSQL with the pgvector extension, enabling fast vector similarity searches and flexible querying of agent memories. Ideal for developers building intelligent applications that need persistent, searchable context.
