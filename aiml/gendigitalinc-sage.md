## 📝 Summary

The Sage project is an open-source, lightweight Agent Detection and Response (ADR) layer designed to safeguard AI agents from malicious activities. It provides real-time monitoring and protection for commands, files, and web requests, ensuring the integrity and trustworthiness of AI systems. As part of the Gen Agent Trust Hub, Sage offers a robust security solution for AI/ML applications.
