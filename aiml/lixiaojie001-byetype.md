> ByeType 是一个 Markdown 驱动的 AI 语音输入工具，用多模态大模型直接处理原始音频。编辑 Markdown 文件即可自定义专有词汇、转录规则和格式化策略，人名、术语、口水词、数字格式一次转录到位，减少二次修改。支持 macOS、Windows 和 iOS。

## 📝 Summary

Here's a brief summary of the project in English:

ByeType is an open-source AI-powered voice input tool driven by Markdown, allowing users to customize specific vocabulary, transcription rules, and formatting strategies directly within their Markdown files. This innovative tool streamlines the transcription process, accurately converting names, technical terms, colloquialisms, and numerical formats with a single pass, reducing the need for subsequent edits. ByeType supports macOS, Windows, and iOS platforms.
