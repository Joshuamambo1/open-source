## 📝 Summary

Here's a brief summary for the kanvibe project:

Kanvibe is a self-hosted Kanban board that allows users to manage AI coding agents with browser terminals. It features hook-driven auto-tracking, enabling seamless integration with tmux/zellij sessions and git worktrees from a single board. This powerful tool streamlines the development process for AI/ML projects.
