## 📝 Summary

### English

skywind3000/vim-quickui provides a set of lightweight, cross‑compatible UI components that extend Vim 9 and Neovim with modern dialog boxes, menus, input prompts, and floating windows. Designed for developers who need quick, scriptable interfaces without leaving the editor, the plugin offers a simple API and works seamlessly on both Vim and Neovim. It fills the gap of missing native UI features, making interactive workflows faster and more ergonomic.

### Русский

**skywind3000/vim-quickui** – набор недостающих UI‑расширений для Vim 9 и NeoVim, позволяющих быстро создавать и управлять всплывающими окнами, меню, диалогами и другими интерактивными элементами прямо в редакторе. Проект написан на чистом Vimscript, не требует внешних зависимостей и легко интегрируется в любые плагины, улучшая пользовательский интерфейс без ущерба производительности. Он подходит как для разработки фронтенда, так и для создания удобных продуктов внутри Vim‑окружения.
