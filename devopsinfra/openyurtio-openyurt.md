## 📝 Summary

OpenYurt is an open-source project that extends native Kubernetes to edge computing environments, enabling seamless deployment and management of applications across data centers and edge devices. As a project under the Cloud Native Computing Foundation (CNCF), OpenYurt provides a scalable and flexible solution for edge computing use cases. It allows users to leverage existing Kubernetes skills to manage and orchestrate applications at the edge.
