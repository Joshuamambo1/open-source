The modelcontextprotocol/go-sdk is an open-source project that provides an official Go SDK for both Model Context Protocol servers and clients. Developed in collaboration with Google, this SDK enables seamless interaction between AI/ML applications and Model Context Protocol infrastructure. It serves as a powerful tool for developers building backend applications that integrate with Model Context Protocol.
