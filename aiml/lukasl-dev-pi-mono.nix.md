## 📝 Summary

### English

**pi-mono.nix** is an open‑source Nix flake that packages **pi**, a terminal‑based coding assistant powered by AI/ML. By providing a reproducible, declarative Nix configuration, it makes installing and running pi’s interactive coding agent simple and reliable on any Nix‑enabled system.
