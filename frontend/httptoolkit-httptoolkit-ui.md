## 📝 Summary

### English

HTTP Toolkit UI is the front‑end interface for the open‑source HTTP Toolkit, providing developers with a visual, interactive environment to intercept, inspect, and debug HTTP(S) traffic. Built with modern web technologies, it offers real‑time request/response inspection, editing, and replay features, making it a powerful dev‑tool for web and API development. The project is maintained as a standalone UI component, ready to be integrated or extended within custom workflows.

### Русский

HTTP Toolkit — это открытый набор инструментов для отладки и анализа HTTP‑трафика, а пакет **httptoolkit/httptoolkit-ui** представляет собой современный клиентский интерфейс, реализованный на React/TypeScript. UI предоставляет интерактивные панели для перехвата, просмотра и модификации запросов и ответов в реальном времени, а также удобные средства для работы с WebSocket, GraphQL и другими протоколами. Проект ориентирован на разработчиков и DevOps‑специалистов, которым нужен лёгкий, расширяемый и полностью открытый фронтенд‑инструмент для веб‑отладки.
