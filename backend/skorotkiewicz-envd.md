## 📝 Summary

Enve is an open-source, tiny server designed to store environment variables specific to each project. It allows developers to keep sensitive information isolated and organized on a per-project basis, improving project security and management. This lightweight solution streamlines environment variable management, making it easier to work on multiple projects.
