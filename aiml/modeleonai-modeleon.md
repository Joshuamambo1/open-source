## 📝 Summary

Modeleon is an open-source Python Domain-Specific Language (DSL) that allows users to write code and compile it into live Excel formulas. This innovative tool enables developers to leverage Python's simplicity and power to create dynamic, formula-based calculations within Microsoft Excel. By bridging the gap between Python and Excel, Modeleon streamlines data manipulation and analysis.
