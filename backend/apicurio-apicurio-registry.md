## 📝 Summary

### English

Apicurio Registry is an open‑source backend service that provides a centralized store for API specifications and data schemas (such as OpenAPI, AsyncAPI, Avro, Protobuf, and JSON Schema). It enables teams to version, validate, and retrieve contracts consistently across microservices and event‑driven architectures, supporting multiple storage backends and offering a RESTful API, UI, and client libraries for seamless integration.
