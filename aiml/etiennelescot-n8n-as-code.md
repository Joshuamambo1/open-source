## 📝 Summary

EtienneLescot/n8n-as-code is an open‑source toolkit that enhances AI agents by integrating them with n8n’s powerful workflow automation platform. It ships with over 537 fully‑typed nodes, more than 7,700 ready‑to‑use templates, Git‑style synchronization, and TypeScript‑based workflow definitions, making it easy to create, version‑control, and deploy complex AI‑driven automations. This project is ideal for developers who want to combine AI/ML capabilities with robust, code‑first workflow orchestration.
