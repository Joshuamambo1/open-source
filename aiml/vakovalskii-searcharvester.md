## 📝 Summary

Here's a brief summary of the project:

SearchArvester is an open-source, self-hosted search and markdown harvesting tool designed for AI agents. It combines SearXNG's extensive search engine support (over 100 engines) with FastAPI and trafilatura for efficient data retrieval and processing. This project allows users to run a customizable search engine and markdown extractor on their own infrastructure with ease, using a single command Docker Compose setup.
