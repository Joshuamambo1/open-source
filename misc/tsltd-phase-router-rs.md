## 📝 Summary

Here's a brief summary of the Phase Router project:

Phase Router is an open-source project that offers capacity-aware routing for Multi-Model or Multi-Expert (MoE) architectures. This routing approach aims to optimize performance by dynamically allocating capacity to individual models based on their current workload. By doing so, it can improve the overall efficiency and scalability of MoE systems.
