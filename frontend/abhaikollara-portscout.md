## 📝 Summary

PortScout is an open-source project that provides a TUI (Text User Interface) to identify and eliminate processes occupying specific ports on a system. This tool is designed to help users troubleshoot port conflicts and free up resources, making it a valuable resource for system administrators and developers. By leveraging PortScout, users can quickly and efficiently manage their system's ports and resolve issues related to port occupation.
