## 📝 Summary

Here's a brief summary for the open-source project:

The briqt/agent-usage project is an open-source, lightweight tool for tracking AI coding agent usage and costs across various platforms. It offers a single binary executable, utilizing SQLite for storage and a web dashboard for easy monitoring. This tool is designed to simplify the management of AI/ML costs and usage.
