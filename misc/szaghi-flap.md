## 📝 Summary

szaghi/FLAP is a lightweight, pure‑Fortran library that simplifies handling command‑line arguments in scientific and engineering programs. It provides a straightforward API for defining options, parsing flags, and retrieving values without external dependencies, making it ideal for developers who need a minimal, easy‑to‑integrate solution. The project is positioned as a “poor‑people’s” alternative to more heavyweight parsers, targeting miscellaneous utility and research codebases.
