## 📝 Summary

Threatcl is an open-source project that allows users to document their threat models using HashiCorp Configuration Language (HCL). This tool enables developers and security teams to create and manage comprehensive threat models in a standardized format. By leveraging HCL, threatcl streamlines the threat modeling process and facilitates collaboration across teams.
