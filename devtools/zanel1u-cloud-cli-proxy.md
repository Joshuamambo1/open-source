## 📝 Summary

Here's a brief summary of the open-source project:

The ZaneL1u/cloud-cli-proxy project is an open-source command-line tool that enables users to quickly obtain a dedicated cloud instance pre-installed with Claude Code and route all traffic through a specified exit IP address. Built on top of Docker and a full-tunnel WireGuard/sing-box setup, this solution ensures zero leakage of sensitive data. It simplifies the process of setting up a secure and isolated cloud environment.
