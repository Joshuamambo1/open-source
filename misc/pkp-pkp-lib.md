## 📝 Summary

pkp/pkp-lib is the core PHP library that powers the Public Knowledge Project’s suite of open‑source scholarly publishing platforms—Open Journal Systems (OJS), Open Monograph Press (OMP) and Open Preprint Systems (OPS). It provides reusable components for managing submissions, peer review, publishing workflows, metadata, and user permissions, enabling developers to extend or customize the PKP applications while maintaining a consistent, standards‑compliant foundation.
