**Summary:**  
This open‑source project implements the fourth edition of the “Rinha de Backend” challenge, focusing on fraud detection using vector search techniques. It provides a complete backend solution—including data ingestion, indexing, and query APIs—built with modern Go services and a vector database, showcasing how to design scalable, low‑latency systems for real‑time anomaly detection.
