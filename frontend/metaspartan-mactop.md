## 📝 Summary

The mactop project is an open-source Apple Silicon Monitor Top application that provides a customizable and visually appealing display of system information. This frontend application is designed to be user-friendly and visually pleasing, offering a unique way to monitor system metrics on Apple Silicon devices. It aims to offer a more flexible alternative to the default system monitoring tools.
