## 📝 Summary

Razzia is a lightweight, self‑hosted quiz platform designed for small‑scale events such as meetups, workshops, or classroom activities. Built with a modern frontend stack, it lets organizers create, manage, and run interactive quizzes without relying on external services, while participants join via a simple web interface. The project is fully open source, making it easy to customize and extend for any niche quiz‑based experience.
