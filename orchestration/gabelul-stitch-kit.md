## 📝 Summary

Here's a 2-3 sentence summary of the gabelul/stitch-kit project:

The gabelul/stitch-kit is an open-source project that empowers AI coding agents with advanced design capabilities, comprising 35 skills and 5 layers that can be applied across 7 framework targets. This project leverages Google's Stitch MCP and integrates with Claude Code and Codex CLI to facilitate ideation, batch generation, design systems, iteration loops, and production conversion. By providing a design intelligence layer, the project enables AI agents to take their coding capabilities to the next level.
