## 📝 Summary

### English

Vite DevTools is an open‑source extension that provides a visual interface for inspecting and debugging Vite (and its Rollup‑compatible successor, Rolldown) projects. It offers real‑time insights into module graphs, dependency trees, server status, and build performance, helping developers diagnose issues and optimize their Vite‑based workflows.

### Русский

**vitejs/devtools** — это набор инструментов разработчика (DevTools) для Vite (и совместимого с ним bundler’а Rolldown), позволяющий в режиме реального времени отслеживать состояние проекта, анализировать модули, их зависимости и производительность сборки. Интегрируется в браузерные DevTools, предоставляя удобный интерфейс для отладки, профилирования и визуализации процесса разработки.
