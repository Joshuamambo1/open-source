## 📝 Summary

The pinnacle project is an open-source Wayland compositor built on top of Smithay, offering a lightweight and customizable desktop experience. Inspired by AwesomeWM, it allows users to configure the environment using either Lua or Rust scripts, providing a flexible and extensible platform. This compositor aims to provide a simple yet powerful alternative for those seeking a modern Wayland-based desktop solution.
