The Reliquary Archiver is an open-source tool designed to extract and export in-game data from network packets of a specific turn-based anime game. This project utilizes AI/ML techniques to analyze and reconstruct game data, providing a valuable resource for fans and researchers interested in the game's mechanics and lore. By automating the process of data extraction, the Reliquary Archiver streamlines the process of collecting and organizing game data.
