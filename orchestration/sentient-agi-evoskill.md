## 📝 Summary

EvoSkill is an open-source framework that utilizes machine learning to enhance the performance of coding agents by automatically discovering and synthesizing reusable skills from failed trajectories. This innovative tool helps improve agent capabilities by repurposing past experiences, reducing the need for manual skill development. By leveraging automation and AI, EvoSkill streamlines the process of agent improvement.
