## 📝 Summary

Here's a brief summary of the Decohererk/DecoTV project:

Decohererk/DecoTV is an open-source, cross-platform media aggregator and player built upon the latest version of LunaTV. It offers an intuitive and user-friendly experience, providing access to a wide range of video content. Originally developed as KatelyaTV, this project has been rebranded and is now available for use on multiple platforms.
