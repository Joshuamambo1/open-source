## 📝 Summary

### English

Lavalink is a standalone audio sending node that leverages the Lavaplayer library to provide high‑performance, low‑latency music streaming for Discord bots and other voice applications. It runs as an independent service, handling audio decoding, transcoding, and network delivery, allowing developers to offload intensive audio processing from their main application servers. The project includes Docker support and extensive configuration options, making it easy to deploy and scale in modern DevOps and infrastructure environments.
