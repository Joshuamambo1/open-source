## 📝 Summary

### English

Wayfire is a lightweight, modular Wayland compositor designed for flexibility and extensibility. Built with a plugin architecture, it lets users customize window management, effects, and input handling by adding or swapping out components without modifying the core. The project targets developers and power users who want a highly configurable compositor for Linux desktops.
