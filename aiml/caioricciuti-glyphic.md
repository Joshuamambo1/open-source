## 📝 Summary

Glyphic is an open-source project that provides a user-friendly, graphical interface for configuring and utilizing Claude Code, an AI coding assistant from Anthropic. By offering a modern desktop app experience, Glyphic simplifies the management and customization of Claude Code, eliminating the need for manual editing of JSON files and markdown. This project aims to enhance the accessibility and usability of Claude Code for developers.
