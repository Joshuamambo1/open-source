## 📝 Summary

This is an open-source project that provides a Persian (Farsi) calendar implementation for Android devices. It allows users to view and navigate through the Persian calendar, enabling them to stay connected to their cultural and linguistic heritage. The calendar is fully localized to support the Persian language and follows the traditional Persian calendar system.
