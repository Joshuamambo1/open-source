**项目简介**  
InvadingOctopus/comedot 是一个基于 Godot 的 2D 游戏模板与组件化框架，提供完整的实现模式示例，帮助开发者快速上手并遵循业界成熟的架构。

**价值**  
- **学习成熟实现模式**：通过可运行的示例代码，直接观察并复用经过实战验证的组件交互、系统组织和资源管理方式。  
- **加速教学与培训**：可作为教程、工作坊或团队内部技术栈培训的教材，降低新人上手门槛。  
- **提升开发效率**：提供即插即用的组件、统一的 API/SDK 与 CLI，减少从零搭建框架的时间成本。

**典型接入方式**  
1. **克隆仓库**：`git clone https://github.com/InvadingOctopus/comedot.git`。  
2. **使用 Godot 打开**：在 Godot 编辑器中直接打开 `project.godot`，即可看到完整的项目结构与示例场景。  
3. **通过 CLI 生成项目**：运行 `./comedot-cli new MyGame`（或对应的 GDScript 脚本），自动生成基于模板的空白项目，随后在编辑器中继续开发。  
4. **集成组件**：在自定义项目中通过 `extends ComedotComponent` 继承框架组件，或在 `autoload` 中加载框架提供的全局服务（如事件总线、资源管理器）。
