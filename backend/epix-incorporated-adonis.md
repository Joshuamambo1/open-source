## 📝 Summary

### English

Epix‑Incorporated/Adonis is an open‑source server administration framework for Roblox games, providing a comprehensive suite of tools for moderation, permission management, and gameplay utilities. Built with a modular backend architecture, it enables developers to easily customize commands, automate admin actions, and integrate additional features while maintaining high performance and security.

### Русский

Epix‑Incorporated/Adonis — это открытая серверная система администрирования для Roblox, предоставляющая набор мощных инструментов для управления игроками, защиты от эксплойтов и автоматизации рутинных задач. Проект реализован на Lua и легко интегрируется в любой игровой сервер, позволяя администраторам быстро настраивать права доступа, логи и модерацию без необходимости писать собственный код. Благодаря активному сообществу и регулярным обновлениям, Adonis обеспечивает стабильную и безопасную работу Roblox‑проектов.
