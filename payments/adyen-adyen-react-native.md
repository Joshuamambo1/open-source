## 📝 Summary

### English

Adyen’s adyen-react-native library provides a lightweight, cross‑platform wrapper for integrating Adyen’s payment services into React Native mobile apps. It offers ready‑to‑use UI components and native modules for iOS and Android, handling tokenization, payment methods, and secure communication with the Adyen API. This open‑source solution streamlines the implementation of secure, PCI‑compliant payments in both iOS and Android front‑end projects.
