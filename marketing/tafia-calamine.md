## 📝 Summary

### English

Tafia / Calamine is a pure‑Rust library that reads Excel (XLSX, XLSB, XLS) and OpenDocument Spreadsheet (ODS) files without external dependencies, delivering fast, memory‑safe parsing directly in Rust. It abstracts the complexities of the underlying file formats into a simple, ergonomic API, making it ideal for developers who need reliable spreadsheet import functionality in their applications.
