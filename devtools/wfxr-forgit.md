## 📝 Summary

### English

ForGit is an open‑source command‑line utility that leverages **fzf** to provide an interactive, fuzzy‑search driven interface for common Git operations. By wrapping Git commands in a convenient, menu‑based workflow, it lets users browse branches, commits, stashes, and more, then execute actions (checkout, revert, reset, etc.) with just a few keystrokes. Ideal for developers who want a faster, more visual way to manage repositories from the terminal.
