### 中文

**项目简介**  
vericontext/vibeframe 是一款面向 AI 的视频编辑器，采用 CLI‑first、MCP（Model Context Protocol）就绪的设计思路，开发者可以直接在终端生成、编辑并发布视频，实现 AI 助手对真实工具和数据的无缝调用。

**价值**  
- **统一协议**：通过标准的 Model Context Protocol，将 AI 代理与本地工具、数据源进行可靠对接，降低集成成本。  
- **开发者友好**：提供完整的 API、SDK 与 CLI，支持 TypeScript/Node 环境，便于在脚本、CI/CD 或自动化工作流中调用。  
- **高效视频生产**：从文本提示到成片全流程自动化，帮助 AI 应用快速交付多媒体内容。

**典型接入方式**  
1. **CLI 调用**：在终端执行 `vibeframe generate ...`、`vibeframe edit ...` 等命令，适合快速原型或脚本化批处理。  
2. **SDK / API**：在 TypeScript 项目中引入 `@vibeframe/client`，使用 `createVideo`, `editVideo` 等函数，便于在自定义服务或 AI Agent 中直接调用。  
3. **MCP 服务器**：部署 Vibeframe 作为 Model Context Protocol 服务端，AI 代理通过标准的 MCP 请求（JSON‑RPC/HTTP）与其交互，实现“AI → 工具” 的统一入口。

- **技术成熟度**：使用 TypeScript 编写，提供完整类型定义；CLI 与 SDK 均已发布正式版本。  
- **生态兼容**：支持常见 CI/CD、容器化部署，可在内部私有化环境或云端直接运行。
