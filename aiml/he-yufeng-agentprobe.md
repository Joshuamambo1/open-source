## 📝 Summary

AgentProbe is a lightweight pytest plugin that lets you regression‑test AI agents by capturing snapshot baselines of their outputs and comparing them semantically rather than text‑for‑text. It includes utilities for mocking large language models, making it easy to run deterministic tests without external API calls. Ideal for developers who need reliable, automated verification of agent behavior in CI pipelines.
