Booksearch is an open-source project that enables fast and efficient search functionality for book collections through a text-based user interface (TUI). This project aims to provide a seamless browsing experience for users to find their desired books quickly and easily. Developed with a focus on user-friendliness, Booksearch streamlines the process of searching and discovering new books within a collection.
