## 📝 Summary

The phub-cli project is an open-source, terminal-based browser for PornHub that allows users to browse categories, search for videos, and stream them directly in mpv without the need for a browser. This project was inspired by ani-cli and serves as a convenient alternative for those who prefer a command-line interface. It supports instant streaming and offers a seamless browsing experience.
