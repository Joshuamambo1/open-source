**项目简介**  
PCGEx（PCGExtendedToolkit）为 Unreal Engine 的程序化内容生成（PCG）补全关键功能，提供图论运算、进阶路径寻找、空间操作、过滤、资产管理等模块，让复杂的生成流程可以直接在蓝图或 C++ 中调用。

**价值**  
- **功能完整**：弥补 Unreal 原生 PCG 在图结构、路径规划和空间查询上的缺失，适配多种生成场景（关卡布局、道路网络、资源分布等）。  
- **即插即用**：通过插件形式暴露 API/SDK，支持蓝图节点、C++ 接口以及命令行工具，开发者可根据项目需求灵活选用。  
- **社区与维护**：拥有 663+ 星、87+ Fork，近期仍在活跃维护，代码基于 C++，文档和示例齐全，降低学习成本。

**典型接入方式**  
1. **插件安装**：将仓库克隆或通过 Unreal Marketplace 添加插件，启动项目后即可在编辑器的 “Plugins” 列表中启用。  
2. **蓝图调用**：在关卡蓝图或 PCG Graph 中使用插件提供的节点（如 “GraphShortestPath”、 “SpatialFilter”等），无需写代码即可完成复杂生成。  
3. **C++ 集成**：在自定义模块中 `#include "PCGEx/PCGEx.h"`，调用公开的类与函数，或通过插件提供的 SDK 接口实现高级逻辑。  
4. **CLI/脚本**：插件提供的命令行工具可在构建流水线或离线批处理时执行 PCG 任务，适配 CI/CD 场景。
