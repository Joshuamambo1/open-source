## 📝 Summary

Uni is an open-source, embedded database that integrates property graph, vector search, and columnar storage into a single engine, ideal for applications needing fast and multimodal data access. It leverages object storage for durability, ensuring seamless data management and retrieval. This unified database is designed for applications in AI/ML, frontend development, and data-driven projects.
