## 📝 Summary

Here's a 2-3 sentence summary for the VioletCranberry/coco-search open-source project:

coco-search is a local-first hybrid semantic code search tool that enables efficient code discovery within large codebases. This open-source project leverages PostgreSQL with pgvector embeddings and combines vector similarity search with keyword search using RRF fusion, supporting over 30 languages. Features include a CLI, MCP server, web dashboard, and interactive REPL for a seamless development experience.
