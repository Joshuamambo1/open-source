## 📝 Summary

### English

GoogleCloudPlatform/kubectl‑ai is an open‑source, AI‑driven assistant that extends the standard `kubectl` command‑line tool with natural‑language capabilities. By leveraging large language models, it lets users query, troubleshoot, and manage Kubernetes clusters using conversational prompts, automatically generating the appropriate `kubectl` commands and offering contextual insights. Ideal for developers and operators, it streamlines day‑to‑day DevOps workflows while integrating seamlessly with existing Kubernetes environments.
