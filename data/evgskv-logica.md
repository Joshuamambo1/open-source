## 📝 Summary

Logica is an open-source logic programming language that enables efficient and flexible data querying. By compiling to SQL, it can run on popular databases such as DuckDB, Google BigQuery, PostgreSQL, and SQLite. This allows users to leverage the strengths of logic programming in a scalable and database-agnostic manner.
