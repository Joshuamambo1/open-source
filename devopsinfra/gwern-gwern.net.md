## 📝 Summary

### English

gwern/gwern.net is the open‑source codebase that powers the gwern.net website, built with a custom Hakyll static‑site generator. It provides a rich content‑authoring toolkit—including unique link archiving, popup UI, transclusions, collapsible sections, bidirectional backlinks, and advanced typography features like sidenotes, drop caps, link icons, inflation‑adjusted figures, and subscripted citations—while also supporting dark and reader modes. The project serves as both a DevOps/infra example and a design showcase for highly interactive, typographically sophisticated static sites.
