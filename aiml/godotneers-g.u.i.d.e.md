## 📝 Summary

The G.U.I.D.E project is an open-source extension for the Godot game engine, designed to simplify and unify input detection across various devices and platforms. This streamlined input handling engine enables developers to create more seamless and intuitive user experiences in their games. By leveraging G.U.I.D.E, game developers can reduce complexity and improve input responsiveness.
