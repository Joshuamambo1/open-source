## 📝 Summary

MadAppGang / claudish is an open‑source AI coding assistant that leverages Claude’s capabilities to generate, explain, and refactor code across any programming language and model backend. It provides a unified command‑line and web interface for developers to interact with the model, offering features such as context‑aware suggestions, multi‑file project handling, and customizable prompts. The project aims to make powerful, language‑agnostic AI coding assistance freely accessible to the community.
