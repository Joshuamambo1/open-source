**项目简介**  
RocketSim（AvdLee/RocketSimApp）是一套面向 Xcode iOS Simulator 的 30 多个实用工具，涵盖测试、调试、网络监控、截图、可访问性检查、应用行为触发以及通过 RocketSim CLI 实现的 AI 代理自动化。已被 80k+ 开发者采用，帮助在 iOS 开发流程中消除大量重复的手工操作。

**价值**  
- **提升效率**：将手动的测试、日志抓取、网络请求监控等步骤自动化，显著缩短调试和验证周期。  
- **可编排**：提供统一的 CLI 与 API，便于把各工具串联成可重复、可调度的工作流。  
- **AI 助手**：内置 AI 代理，可在模拟器上执行复杂的交互脚本，实现无人值守的回归测试或 UI 场景演练。  

**典型接入方式**  
1. **CLI**：直接在终端调用 `rocketsim` 命令，启动指定工具或组合任务（如 `rocketsim capture --app MyApp`）。  
2. **API/SDK**：通过 Swift 包或 CocoaPods 引入 `RocketSimKit`，在自定义脚本或 CI/CD 流程中调用其公开的函数接口。  
3. **集成 CI**：在 GitHub Actions、Bitrise、Jenkins 等 CI 环境中配置步骤，利用 CLI 完成自动化测试、报告生成和结果上传。
