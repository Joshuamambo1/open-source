## 📝 Summary

### English

The **dibdot/DoH-IP-blocklists** repository provides curated lists of domain names and their corresponding IPv4 and IPv6 addresses for publicly available DNS‑over‑HTTPS (DoH) servers. These blocklists can be used by security tools, firewalls, or networking scripts to identify, filter, or monitor DoH traffic. The project is maintained as an open‑source resource for developers and administrators needing up‑to‑date DoH endpoint data.
