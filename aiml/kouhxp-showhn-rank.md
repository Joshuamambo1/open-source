Ranking 1k ShowHN posts by estimated merit using an LLM judge and TrueSkill is an open‑source experiment that crawls the latest 1,000 “Show HN” submissions on Hacker News, lets a large language model score each post on perceived quality, and then feeds those scores into a TrueSkill rating system to produce a ranked leaderboard. The project demonstrates how LLM‑based evaluation combined with probabilistic skill‑rating algorithms can automatically surface the most valuable community contributions. It serves as a reproducible template for merit‑based ranking of user‑generated content in AI/ML research and community‑driven platforms.
