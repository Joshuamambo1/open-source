## 📝 Summary

Here's a 2-3 sentence summary for the project:

Sidekick Agent Hub is an open-source project that provides a unified interface for multiple AI coding assistants, including Claude Code, OpenCode, and Codex. This project allows users to monitor and manage their agent sessions from both Visual Studio Code (VS Code) and the terminal, featuring inline completions, code transforms, and a full Text User Interface (TUI) dashboard. With Sidekick Agent Hub, developers can easily switch between multiple Claude Code accounts, streamlining their coding workflow.
