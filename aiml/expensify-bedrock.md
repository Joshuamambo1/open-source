## 📝 Summary

Bedrock is an open-source distributed database designed to provide high availability and scalability. It specializes in active/active automatic failover and WAN replication, allowing for seamless data synchronization across multiple sites. This feature-rich database is ideal for demanding applications that require low-latency, high-throughput data management.
