## 📝 Summary

The blamebot is an open-source AI-powered on-call agent designed to detect and respond to deploy failures in a production environment. It utilizes machine learning to identify the root cause of the issue, determine the responsible team, and initiate an automatic rollback to restore functionality. This proactive approach aims to minimize downtime and expedite resolution of critical system failures.
