## 📝 Summary

This open-source project provides a Makefile-based build system for Shrike-lite, a lightweight FPGA development board, and the Renesas SLG47910V FPGA. The project enables users to easily build and manage their FPGA projects using a Makefile, streamlining the development process. It caters to developers working with the Shrike-lite and Renesas SLG47910V FPGA, offering a convenient and standardized build framework.
