### English

BushidoUK’s **Ransomware‑Tool‑Matrix** is an open‑source repository that catalogues the malware development and deployment tools employed by various ransomware gangs. By organizing binaries, scripts, and utility packages into a searchable matrix, the project helps security analysts, incident responders, and researchers quickly identify the capabilities and provenance of ransomware toolsets. It serves as a reference point for threat‑intel sharing and defensive strategy planning within the security community.
