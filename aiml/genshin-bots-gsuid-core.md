## 📝 Summary

Here's a brief summary of the Genshin-bots/gsuid_core project:

GsCore is an open-source, asynchronous core framework that provides a unified platform for developing plugins and supports multiple bot platforms, including NoneBot2, HoshinoBot, ZeroBot, YunZaiBot, and Koishi. This core framework offers a robust set of features, including a core database, reusable game query logic, and a web-based control panel. By leveraging GsCore, developers can streamline their plugin development and focus on creating innovative features for various Genshin Impact bots.
