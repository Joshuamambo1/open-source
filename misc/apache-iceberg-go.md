## 📝 Summary

### English

Apache Iceberg Go is an open‑source Go library that provides native support for reading, writing, and managing Apache Iceberg tables. It enables Go applications to interact with Iceberg’s high‑performance, schema‑evolution‑aware data format, offering features such as partition pruning, snapshot isolation, and atomic commits. The project serves as the official Go implementation for working with Iceberg tables across cloud and on‑premise data lakes.

### Русский

Apache Iceberg — это открытый формат таблиц для аналитических данных, а репозиторий **apache/iceberg-go** предоставляет официальную реализацию этого формата на языке Go. Библиотека позволяет создавать, читать и модифицировать Iceberg‑таблицы, поддерживая такие функции, как атомарные коммиты, схемы эволюции, управление версиями и эффективный скан данных. Проект ориентирован на интеграцию с экосистемой Go‑инструментов для обработки больших данных и подходит как для построения аналитических хранилищ, так и для разработки кастомных ETL‑решений.
