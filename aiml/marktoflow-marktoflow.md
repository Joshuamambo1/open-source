## 📝 Summary

Here's a 2-3 sentence summary of the marktoflow project:

Marktoflow is an open-source automation framework that empowers users to create workflows using markdown+YAML, which can be executed with popular AI tools like Claude Code, GitHub Copilot, Codex, and OpenCode without requiring AI API keys. The framework offers 39 direct SDK integrations, a visual designer UI, and full compatibility with MCP (Marketo Personalized), making it a powerful and user-friendly solution for automation tasks. With its simplicity and versatility, Marktoflow streamlines workflow creation and execution, revolutionizing the way automation is done.
