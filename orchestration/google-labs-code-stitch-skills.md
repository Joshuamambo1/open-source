## 📝 Summary

### English

Stitch‑Skills is an open‑source library that provides a collection of “Agent Skills” built to run on the Stitch MCP server. Each skill conforms to the Agent Skills open standard, ensuring seamless integration with coding agents like Antigravity, Gemini CLI, Claude Code, and Cursor for orchestrating AI‑driven workflows. The project targets orchestration, AI/ML, and backend developers looking to extend their agents with reusable, server‑side capabilities.
