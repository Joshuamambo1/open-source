## 📝 Summary

The opencode-telegram-bot project is an open-source, mobile client for running and monitoring AI coding tasks via Telegram. This allows users to execute tasks on their local machine from their phone while receiving updates and notifications through the Telegram messaging app. The bot also supports scheduled tasks, making it a lightweight alternative to OpenClaw for AI-related development tasks.
