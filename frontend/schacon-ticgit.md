## 📝 Summary

### English

schacon/ticgit is an open‑source, Git‑based distributed ticketing system that leverages the git‑meta framework to store and manage issue data directly within a Git repository. By treating tickets as version‑controlled objects, it enables collaborative workflow, full history tracking, and easy synchronization across multiple clones without requiring a separate server. The project provides a lightweight, command‑line interface suitable for teams that prefer Git‑centric tooling for project management.
