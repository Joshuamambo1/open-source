## 📝 Summary

The Bigfoot71/r3d project is an open-source 3D extension library designed specifically for the raylib framework. This library enables developers to leverage raylib's 2D capabilities while adding support for 3D rendering and advanced graphics features. By extending raylib's functionality, r3d facilitates the creation of complex and immersive 3D applications.
