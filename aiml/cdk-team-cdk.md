## 📝 Summary

The CDK (Container Development Kit) is an open-source project that simplifies security testing for Kubernetes (K8s), Docker, and Containerd environments. This tool enables users to automate and streamline security assessments, making it easier to identify vulnerabilities and strengthen containerized systems. By providing a user-friendly interface for security testing, CDK facilitates more efficient and effective container security management.
