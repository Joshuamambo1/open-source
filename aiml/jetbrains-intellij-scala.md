## 📝 Summary

### English

JetBrains/intellij-scala is an open‑source plugin that brings comprehensive Scala support to the IntelliJ IDEA IDE. It offers advanced features such as code completion, syntax highlighting, refactoring, debugging, and integration with build tools (sbt, Maven, Gradle), helping developers write, navigate, and maintain Scala code more efficiently. The project is maintained by JetBrains and the Scala community, with contributions welcomed via GitHub.
