## 📝 Summary

### English

AgentEval is a full‑featured .NET toolkit that lets developers rigorously evaluate AI agents, covering tool‑usage validation, RAG quality metrics, stochastic testing, and model comparison. Originally built for the Microsoft Agent Framework (MAF) and Microsoft.Extensions.AI, it provides the same kind of comprehensive assessment capabilities that Python libraries like RAGAS, PromptFoo, and DeepEval offer, but natively for the .NET ecosystem.
