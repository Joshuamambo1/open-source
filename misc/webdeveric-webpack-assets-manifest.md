## 📝 Summary

### English

The **webpack‑assets‑manifest** plugin for Webpack automatically creates a JSON manifest that maps each original asset filename to its corresponding hashed version generated during the build. By emitting this manifest alongside your bundles, it simplifies cache‑busting and server‑side rendering workflows that need to reference the correct file names at runtime. The project is open‑source, lightweight, and works with any Webpack configuration.
