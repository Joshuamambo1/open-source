This project, simgine/bevy_enhanced_input, provides an enhanced input management system for the Bevy game engine, offering a more comprehensive and customizable solution for handling user input. It aims to improve upon Bevy's built-in input handling capabilities, providing developers with more flexibility and control over their game's input logic. This project is open-source, allowing the community to contribute and benefit from its features.
