## 📝 Summary

A lightweight utility that lets you capture the current clipboard contents and store them as editable note files, making quick note‑taking seamless for developers. It provides simple commands to save, open, and modify these notes, and integrates with GitLab for version‑controlled backups. Ideal for fast, minimalistic documentation directly from the command line.
