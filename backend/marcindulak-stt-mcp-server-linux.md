## 📝 Summary

This project, marcindulak/stt-mcp-server-linux, is an open-source local speech-to-text server designed specifically for use with Tmux on Linux systems. It enables users to control their terminal-based applications with voice commands, offering a convenient and accessible interface. This server can be utilized with the Claude Code or other compatible applications.
