## 📝 Summary

Here's a 2-3 sentence summary for the project:

Waha-TUI is an open-source Terminal User Interface for WhatsApp that utilizes the WAHA (WhatsApp HTTP API) to manage WhatsApp sessions, chats, and messages directly from the terminal. This project is powered by OpenTUI, providing an intuitive and user-friendly interface for interacting with WhatsApp. With Waha-TUI, users can enjoy a seamless WhatsApp experience from the comfort of their terminal.
