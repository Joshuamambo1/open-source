## 📝 Summary

The Messy Model Bench Tests project compares the performance of two open-source models, Qwen3.6-27B and Coder-Next, in a series of bench tests. This project aims to provide a transparent and community-driven evaluation of these models, helping developers make informed decisions about their choice of tools. The results of these tests will be made available for public review and discussion.
