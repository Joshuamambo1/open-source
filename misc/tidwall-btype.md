### English

B‑tree‑based collection types for Go provide high‑performance, ordered data structures (maps, sets, and sorted slices) implemented on top of Go’s standard library `container/tree` package. By leveraging B‑trees, these collections deliver fast logarithmic‑time inserts, deletions, and lookups while maintaining low memory overhead and excellent cache locality. The project is open‑source, well‑documented, and suitable for any Go application that needs scalable, ordered containers.
