## 📝 Summary

LITUATUI/M3UPT is an open‑source collection of Portuguese‑language IPTV playlists formatted as M3U, featuring both television and radio stations. It includes only publicly available, official streams, making it a reliable resource for users seeking legal, ready‑to‑play media links. The project is primarily a frontend asset that can be imported into any compatible media player.
