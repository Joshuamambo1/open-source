## 📝 Summary

### English

neutralinojs‑cli is the official command‑line interface for building and managing Neutralinojs applications. It provides developers with tools to quickly scaffold new projects, run and debug apps locally, and package them for distribution across Windows, macOS, and Linux. The lightweight CLI streamlines the entire development workflow for Neutralinojs‑based desktop applications.

### Русский

`neutralinojs-cli` — это официальная командная оболочка для платформы **Neutralinojs**, позволяющая быстро создавать, настраивать и управлять кроссплатформенными десктопными приложениями без тяжёлой инфраструктуры. CLI предоставляет готовые шаблоны проектов, удобные скрипты сборки, запуск и публикацию, а также интеграцию с популярными сборщиками (Vite, Webpack) и пакетными менеджерами. Инструмент ориентирован на разработчиков, желающих быстро прототипировать лёгкие, нативные UI‑приложения на JavaScript/HTML/CSS.
