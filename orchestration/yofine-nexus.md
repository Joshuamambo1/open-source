## 📝 Summary

Here's a brief summary of the project:

Mexus (formerly Nexus) is an open-source, local web console for managing multiple CLI AI Agent instances in parallel. It provides a centralized interface for orchestrating and monitoring AI and ML tasks, allowing users to streamline their workflows and enhance productivity. With Mexus, developers can easily manage and deploy multiple AI agents in a single, user-friendly interface.
