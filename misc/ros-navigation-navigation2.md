## 📝 Summary

### English

The **ros-navigation/navigation2** repository provides the ROS 2 Navigation Stack, a comprehensive framework for autonomous robot movement. It includes algorithms for path planning, obstacle avoidance, localization, and recovery behaviors, all designed to work seamlessly with the ROS 2 middleware and a variety of robot platforms. This open‑source project enables developers to quickly equip robots with robust, modular navigation capabilities out of the box.
