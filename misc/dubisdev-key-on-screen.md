## 📝 Summary

The dubisdev/key-on-screen open-source project, also known as KoS, is an application that displays the keys being pressed on the screen, providing a visual representation of keyboard input. This tool is useful for demonstration purposes, educational settings, or anyone seeking to enhance their typing skills. By visualizing key presses, KoS helps users develop muscle memory and improve their typing accuracy.
