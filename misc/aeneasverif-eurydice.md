## 📝 Summary

Eurydice is an open-source project that enables the compilation of a subset of Rust code into C, allowing developers to seamlessly integrate their Rust programs into legacy environments that rely on C. This project, part of the AeneasVerif suite, aims to bridge the gap between modern Rust development and traditional C-based ecosystems. By leveraging Eurydice, developers can preserve the benefits of Rust while maintaining compatibility with existing infrastructure.
