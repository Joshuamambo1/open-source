## 📝 Summary

Here is a brief summary of the open-source project:

The UNLINEARITY/CLI-WeChat-Bridge project enables seamless integration of AI command-line tools into WeChat's ClawBot, supporting popular tools like Codex, Claude Code, and OpenCode. This integration allows for shared threads and bidirectional dialogue between WeChat and the local terminal, as well as file transfer capabilities.
