## 📝 Summary

The voxtype project is an open-source voice-to-text solution designed for Wayland compositors, enabling users to communicate hands-free. This AI-powered tool utilizes push-to-talk functionality, allowing users to effortlessly switch between voice input and other activities. By integrating with Wayland compositors, voxtype provides a seamless voice interface for users.
