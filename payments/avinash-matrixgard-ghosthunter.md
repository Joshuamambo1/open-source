## 📝 Summary

Ghost-hunter is an open-source AI-powered cloud cost investigator that provides insight into cloud expenses without requiring direct access to cloud infrastructure. This tool utilizes machine learning algorithms to analyze billing data, identifying areas of unnecessary expenditure and offering recommendations for cost optimization. By leveraging AI, Ghost-hunter enables users to make data-driven decisions and reduce their cloud costs without compromising security.
