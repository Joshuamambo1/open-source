## 📝 Summary

PgQue is a lightweight, open-source Postgres queue system that simplifies asynchronous task management with a single SQL file installation. Leveraging pg_cron for scheduling, it enables efficient and scalable job execution without adding bloat to your database. By providing a zero-bloat solution, PgQue makes it easy to manage tasks within your existing Postgres setup.
