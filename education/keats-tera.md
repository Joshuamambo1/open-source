### 中文

**简短介绍**  
Keats/tera 是一款基于 Jinja2/Django 语法的 Rust 模板引擎，提供高性能的 HTML 渲染能力，并且与 Rust 生态无缝集成。它的实现代码清晰、注释丰富，是学习模板引擎实现模式的理想教材。

**价值**  
- **学习实现模式**：通过阅读其成熟的代码库，开发者可以快速掌握模板解析、AST 构建和渲染流水线等核心概念。  
- **快速构建教学材料**：可直接用于编写教程、示例项目或内部培训教材，帮助团队快速上手 Rust 全栈开发。  
- **提升团队技术栈统一性**：在使用 Rust 开发后端服务时，统一使用 Tera 进行前端渲染，降低语言切换成本。

**典型接入方式**  
1. **作为库依赖**：在 `Cargo.toml` 中加入 `tera = "x.y"`，在代码中 `use tera::{Tera, Context};` 即可加载模板文件并渲染。  
2. **CLI 工具**：Tera 提供 `tera-cli`，可在 CI/CD 或本地脚本中直接执行模板渲染，适合生成静态站点或配置文件。  
3. **API/SDK**：通过 `Tera::render`、`Context::insert` 等公开 API，轻松在 Web 框架（如 Actix‑web、Rocket）中集成，实现服务端渲染。
