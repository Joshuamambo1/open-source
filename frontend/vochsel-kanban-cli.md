## 📝 Summary

Kanban-CLI is an open-source project that offers a web-based user interface for managing local Markdown todo lists. This project combines the simplicity of Markdown files with the interactive features of a web interface, allowing users to easily create, edit, and organize their tasks. By leveraging local Markdown files, users can maintain control over their data while still benefiting from a visually appealing and user-friendly UI.
