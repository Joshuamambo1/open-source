## 📝 Summary

The undercut-f1 project is an open-source, live F1 timing interface for all Formula 1 sessions. It provides a TUI (Text-based User Interface) experience with adjustable delay settings to synchronize with your TV, allowing for a seamless viewing experience. Additionally, the project enables replaying of previously recorded sessions, making it a comprehensive F1 live timing tool.
