## 📝 Summary

Pyinc is an open-source project that provides a pull-based incremental query kernel for Python. This kernel allows for efficient query computation by only updating the necessary data, reducing computational overhead and improving performance. By leveraging incremental computation, Pyinc enables faster and more scalable query execution in Python applications.
