### English

Chuchu is an open‑source Android SSH client that leverages the **libghostty** library to provide a fast, lightweight terminal experience on mobile devices. Designed for developers and power users, it offers a clean frontend UI, configurable key bindings, and essential SSH features such as port forwarding and agent forwarding. The project serves as both a handy mobile tool and a reference implementation for building terminal‑based applications on Android.
