## 📝 Summary

### English

dtkmn/mcp-zap-server is a Spring Boot‑based backend that wraps the OWASP ZAP security scanner behind the Model Context Protocol (MCP). By exposing ZAP’s core capabilities—spidering, active scanning, OpenAPI import, and report generation—as MCP endpoints, it enables any MCP‑compatible AI agent (such as Claude Desktop or Cursor) to orchestrate automated security testing directly from code. This open‑source project bridges AI‑driven workflows with web‑application security tooling.
