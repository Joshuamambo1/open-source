## 📝 Summary

### English

A lightweight tool that inspects a Rust crate’s public API to detect changes that could break semantic versioning guarantees. It parses the crate’s source, compares the current API against a previous release, and reports any added, removed, or altered items that would constitute a semver violation. This helps maintainers ensure version bumps are accurate before publishing.

### Русский

`cargo-semver-checks` — это open‑source утилита, позволяющая автоматически сканировать ваш Rust‑крейт на предмет нарушений семантической версии (semver). Инструмент интегрируется в процесс CI/CD, сравнивает текущие публичные API с предыдущими выпусками и выдаёт детальные отчёты о потенциально несовместимых изменениях. Подходит для любых проектов на Rust, где важна стабильность публичного интерфейса.
