## 📝 Summary

Fenster is an open-source project that enables users to run Chrome's local Gemini Nano as a command-line interface (CLI), providing a convenient way to interact with the browser's Gemini protocol. This tool simplifies the process of accessing and utilizing the Gemini protocol, making it more accessible to developers and users. By providing a CLI interface, Fenster streamlines the interaction with Gemini Nano, promoting easier experimentation and integration with other tools.
