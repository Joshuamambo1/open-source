## 📝 Summary

The dungngminh/simutil project is a cross-platform, open-source utility application that provides a TUI (Text User Interface) for managing iOS simulators, Android emulators, and physical devices. It also includes ADB (Android Debug Bridge) tools, making it a one-stop solution for mobile development. This project aims to simplify the process of launching and debugging mobile applications on various platforms.
