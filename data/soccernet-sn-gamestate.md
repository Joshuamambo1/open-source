## 📝 Summary

SoccerNet/sn-gamestate is an open-source project that provides a dataset and tools for reconstructing game states in soccer, enabling end-to-end athlete tracking and identification on a minimap. This project focuses on the CVPR24 - CVSports workshop and offers valuable resources for researchers and developers working in computer vision and sports analytics. The dataset and tools can facilitate the development of advanced soccer analysis and visualization systems.
