## 📝 Summary

TerraShark is an open-source Terraform skill for Claude Code and Codex, aimed at reducing hallucinations in Large Language Model (LLM) output. By promoting modular and secure code, TerraShark ensures that Infrastructure as Code (IaC) aligns with Hashicorp's official Terraform best practices. This tool helps developers improve the reliability and maintainability of their infrastructure configurations.
