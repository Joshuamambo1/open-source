## 📝 Summary

Here's a brief summary for the 86maid/ddddocr project:

dddocr is an open-source Optical Character Recognition (OCR) API server, implemented in Rust, that supports AI-powered image recognition and runs on multiple platforms without relying on OpenCV. This lightweight solution is easy to deploy and offers a simple yet efficient OCR experience. It also boasts support for AI MCP (Machine Print Character recognition), making it a versatile tool for various text recognition tasks.
