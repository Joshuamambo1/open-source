## 📝 Summary

Here's a brief summary of the project:

The tract project is an open-source, lightweight, and self-contained library for TensorFlow and ONNX inference. It aims to provide a simple and efficient way to perform machine learning model inference without the need for complex setup or dependencies. This makes it an ideal solution for resource-constrained environments or applications where model execution speed is critical.
