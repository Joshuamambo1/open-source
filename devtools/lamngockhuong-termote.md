## 📝 Summary

Termote is an open-source project that allows remote control of CLI tools, such as Claude Code and GitHub Copilot, from mobile or desktop devices via a Progressive Web Application (PWA). This enables seamless interaction with terminal-based applications across different platforms. By leveraging PWA technology, Termote provides a versatile and accessible solution for remote terminal control.
