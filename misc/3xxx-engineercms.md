## 📝 Summary

### English

Engineercms is a web‑based knowledge‑management platform for civil engineers, built with Go (Beego) and designed for distributed file storage via a MinIO cluster. It lets users organize personal and project documents, share files with access codes, collaborate on documents in real time through OnlyOffice, create books with Mindoc, and view PDFs, while also offering a companion mobile mini‑program accessible via WeChat.

### Русский

Engineercms — это веб‑ориентированная система управления знаниями для гражданских инженеров, написанная на Go (framework Beego) и поддерживающая распределённое файловое хранилище MinIO. Платформа позволяет централизованно хранить и обмениваться проектной документацией между заказчиком, проектировщиком, надзором и подрядчиком, предоставляет функции совместного редактирования через OnlyOffice, публикации книг в Mindoc, просмотр PDF и обмен файлами по коду доступа; мобильный клиент реализован в виде мини‑приложения WeChat («设计与管理»).
