## 📝 Summary

Raptor is an open-source project that leverages Claude Code, a general-purpose AI model, to create a security agent capable of both offensive and defensive operations. By configuring rules, sub-agents, and skills, users can orchestrate the agent's behavior to perform research, attacks, or defense operations with an adversarial mindset. This project combines AI and security expertise to provide a powerful tool for security research and operations.
