## 📝 Summary

ContextAtlas is an open-source project that provides a context infrastructure for AI coding agents, enabling hybrid retrieval, project memory, and retrieval observability. This framework integrates various technologies such as Tree-sitter indexing, LanceDB vector search, and FTS5 for advanced context management and search capabilities. ContextAtlas can be accessed through a CLI, MCP server, or embeddable library, making it a versatile tool for AI coding applications.
