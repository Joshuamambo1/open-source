## 📝 Summary

### English

Hazelcast Code Samples is an open‑source repository that provides concise, ready‑to‑run examples demonstrating how to use Hazelcast’s in‑memory data grid and streaming platform across various programming languages and APIs. The collection covers core features such as distributed maps, queues, topics, entry listeners, and client–server integration, helping developers quickly prototype and learn best practices for building scalable, fault‑tolerant applications with Hazelcast.
