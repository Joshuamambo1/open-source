## 📝 Summary

Here's a brief summary of the chenhg5/agencycli project:

AgencyCLI is an open-source, lightweight command-line interface for building self-managing AI agent teams. Users can define roles, skills, and projects in Markdown+YAML files, enabling agents to run autonomously on a set schedule and collaborate with each other through an inbox system. This innovative tool supports integration with various AI models, including Claude Code, Codex, Gemini, and Cursor.
