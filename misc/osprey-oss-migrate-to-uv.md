## 📝 Summary

### English

The **migrate-to-uv** tool automates the conversion of Python projects that use Poetry, Pipenv, pip‑tools, or plain pip into projects managed with the fast, modern **uv** package manager. It parses existing lock files and configuration, rewrites dependencies to uv‑compatible formats, and updates scripts and CI pipelines, allowing developers to switch to uv with minimal manual effort.

### Русский

Проект **osprey-oss/migrate-to-uv** предоставляет набор скриптов и инструкций для автоматизированного перехода существующего Python‑проекта с менеджеров зависимостей Poetry, Pipenv, pip‑tools или обычного pip на новый, более быстрый и удобный пакетный менеджер **uv**. Инструмент анализирует текущий `pyproject.toml`/`requirements.txt`, генерирует совместимый `uv.lock` и обновляет конфигурацию проекта, позволяя сразу начать использовать преимущества uv без потери совместимости.
