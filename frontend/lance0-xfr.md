## 📝 Summary

Here's a brief summary of the lance0/xfr project:

xfr is an open-source, modern alternative to iperf3 that offers a live text-based user interface (TUI), multi-client server capabilities, and support for the QUIC protocol. Built in Rust, xfr provides a high-performance and flexible solution for network benchmarking and testing. Its live TUI enables users to monitor and interact with their network tests in real-time.
