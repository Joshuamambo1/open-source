## 📝 Summary

The 24eme/signaturepdf project is an open-source web application that enables users to digitally sign PDF documents, either individually or with multiple signatories, as well as organize and manipulate their pages. Additionally, the software allows users to edit metadata and compress PDF files to reduce their file size. This free tool streamlines the PDF management process, providing users with a convenient and flexible solution.
