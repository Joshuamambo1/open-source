## 📝 Summary

ImTip is a lightweight (906 KB) open‑source desktop assistant that provides a “super hotkey” system, AI integration, and real‑time input tracking for Windows. It lets users define custom shortcuts, invoke AI-powered commands, and monitor keystrokes or mouse actions to automate workflows and enhance productivity. The project is built with a focus on minimal footprint while delivering powerful, scriptable automation features.
