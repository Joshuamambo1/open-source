## 📝 Summary

### English

The Neural Network Compression Framework (NNCF) is an open‑source toolkit that integrates with OpenVINO™ to automatically apply quantization, pruning, sparsity and other compression techniques to deep learning models, enabling faster and more memory‑efficient inference. It provides a flexible API and ready‑to‑use recipes for popular architectures, allowing developers to optimize AI/ML workloads without sacrificing accuracy.
