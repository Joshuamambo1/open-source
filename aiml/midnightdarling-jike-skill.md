## 📝 Summary

The Jike-skill project is an open-source social network client for both humans and AI agents, offering features such as QR login, feed browsing, post and comment management, and search capabilities. It also supports full history export in Markdown format. This project combines the capabilities of Claude Code skill with standalone scripts, catering to both human users and AI agents.
