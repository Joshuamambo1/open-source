## 📝 Summary

Epiq is an open-source, distributed issue tracker that utilizes Git as its backing store. This Command-Line Interface (CLI) native, Text User Interface (TUI) project provides a unique solution for managing issues in a decentralized and version-controlled manner. With its distributed architecture, Epiq allows for flexible and scalable issue tracking.
