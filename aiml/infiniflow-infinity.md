## 📝 Summary

Here's a brief summary for the infinity project:

Infinity is an open-source, AI-native database designed to optimize large language model (LLM) applications. It enables incredibly fast hybrid search across dense vector, sparse vector, tensor (multi-vector), and full-text data types. This database is tailored to meet the high-performance requirements of LLM applications.
