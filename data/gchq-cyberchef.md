## 📝 Summary

### English

CyberChef is an open‑source web application that functions as a “Swiss Army knife” for data handling, offering a wide range of built‑in operations for encryption, encoding, compression, and forensic data analysis. Users can chain together these operations via an intuitive drag‑and‑drop interface, making complex data transformations fast and repeatable without writing code. It’s widely used by security analysts, developers, and hobbyists for quick, visual processing of binary and textual data.
