## 📝 Summary

RsyncOSX / RsyncUI is a native macOS application built with SwiftUI that provides a clean, graphical interface for configuring and running rsync backups and synchronizations. It abstracts the complex command‑line syntax of rsync into intuitive dialogs, schedules, and visual progress indicators while still exposing advanced options for power users. Ideal for anyone who wants the reliability of rsync without the hassle of manual terminal commands.
