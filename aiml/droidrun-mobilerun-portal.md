## 📝 Summary

The mobilerun-portal is an open-source project that enables automation of mobile devices using natural language commands. This project serves as the counterpart Android app to the mobilerun framework, allowing users to control and interact with their mobile devices in a more intuitive and user-friendly way. By harnessing AI/ML capabilities, mobilerun-portal provides a seamless automation experience for mobile device users.
