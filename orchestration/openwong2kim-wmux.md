## 📝 Summary

Here's a brief summary of the open-source project:

The wmux project is an open-source AI Agent Terminal for Windows, designed to streamline AI development and automation workflows. This terminal allows users to run multiple AI tools, including Claude Code, Codex, and Gemini CLI, side by side in split terminals, while also providing features such as browser automation and smart notifications. With its seamless session persistence and MCP integration, wmux aims to simplify the process of building, testing, and deploying AI models.
