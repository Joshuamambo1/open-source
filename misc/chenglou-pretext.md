## 📝 Summary

chenglou / pretext is an open‑source JavaScript library that provides fast, precise, and feature‑rich utilities for measuring and laying out text. It supports a wide range of typographic calculations—including font metrics, line‑height, wrapping, truncation, and multi‑line layout—while handling complex scripts and diverse font families. The library is designed for high performance and ease of integration in web applications that require accurate text rendering and layout logic.
