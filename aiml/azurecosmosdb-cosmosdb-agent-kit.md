## 📝 Summary

The AzureCosmosDB/cosmosdb-agent-kit is an open-source project that provides a collection of skills for AI coding agents to extend their capabilities when working with Azure Cosmos DB, a globally distributed, multi-model database. This project enables AI agents to efficiently interact with Cosmos DB, streamlining the development and deployment of intelligent applications. By leveraging this kit, developers can empower their AI agents to perform complex tasks and operations on the Cosmos DB platform.
