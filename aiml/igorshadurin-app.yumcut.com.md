## 📝 Summary

Here's a brief summary:

YumCut is a free, open-source AI video generator that transforms user prompts into ready-to-share vertical videos for popular social media platforms like TikTok, Reels, and YouTube Shorts. Built with Next.js, this self-hosted solution allows creators to auto-generate video content, including scripts, scenes, voiceovers, subtitles, and watermarks, in multiple languages. With features like local-first pipeline, batch rendering, and API hooks, YumCut is designed to streamline the content creation process for creators and indie makers.
