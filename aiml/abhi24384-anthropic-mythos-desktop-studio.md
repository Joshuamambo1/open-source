**项目简介**  
Abhi24384/Anthropic-Mythos-Desktop-Studio 是一款面向 Windows、macOS 与 Linux 的 Claude 3.5 Sonnet 桌面客户端，提供即插即用的 UI 与本地 CLI/SDK，帮助开发者在 2026 年快速为桌面应用加入 Anthropic 大模型能力。

**价值**  
- **快速原型**：无需自行搭建模型堆栈，直接调用 Claude 3.5 Sonnet，实现对话、RAG、Agent 等功能的快速验证。  
- **统一入口**：统一的桌面 UI 与跨平台 CLI，让设计师、产品和工程师都能在同一环境下调试、评估模型输出。  
- **降低门槛**：通过封装好的 API/SDK，开发者只需关注业务逻辑，模型调用细节由项目处理。

**典型接入方式**  
1. **CLI**：在终端执行 `mythos-cli chat --model=claude-3.5-sonnet`，即可交互式对话或跑批处理。  
2. **SDK**：在代码中引入 `import mythos`（Python 示例），使用 `mythos.Client(api_key).completion(prompt)` 完成单次或流式请求。  
3. **桌面 UI**：启动 `MythosStudio`，在图形界面配置 API Key、模型参数、知识库路径，即可进行 RAG/Agent 流程的可视化编辑与调试。
