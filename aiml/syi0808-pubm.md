## 📝 Summary

A lightweight AI‑powered bot monitors GitHub issue threads for spam comments and surfaces them for manual review, preventing noisy discussions from spilling over into external chat platforms like Telegram. By leveraging natural‑language classification and GitHub’s webhook API, the tool automatically flags suspicious messages, lets maintainers approve or dismiss them, and keeps the conversation tidy without needing to migrate it elsewhere. The project showcases a practical integration of machine learning with a simple frontend UI for managing spam‑review workflows.
