## 📝 Summary

### English

Elodin is an open‑source monorepo that houses both the simulation environment and the flight software for the Elodin spacecraft project. It provides a unified codebase for developing, testing, and deploying the vehicle’s onboard systems, complete with tools for hardware‑in‑the‑loop simulation and integration. The repository is organized for collaborative development across various subsystems, making it a versatile platform for aerospace research and experimentation.
