## 📝 Summary

The tafl library is an open-source project designed to facilitate the development of Hnefatafl games and simulations. It provides a set of tools for move generation, validation, piece placement, movement, and game over detection, making it a useful resource for creating AI agents that can play against each other. This library can be used to build and test AI agents for the strategic Viking board game, Hnefatafl.
