## 📝 Summary

Squix is an open-source, minimal CLI tool designed to simplify SQL query management across multiple databases. Developed with Go and beautified using BubbleTea, Squix streamlines database interactions by providing a unified interface for query execution. This tool empowers developers to efficiently manage and execute SQL queries in a hassle-free manner.
