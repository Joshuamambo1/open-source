## 📝 Summary

Blissd/fotema is a lightweight, open‑source photo gallery application designed specifically for Linux environments. It provides an intuitive interface for browsing, organizing, and viewing image collections, supporting common formats and offering basic slideshow and thumbnail features. Ideal for users who need a simple yet functional gallery tool without the overhead of larger multimedia suites.
