## 📝 Summary

kucherenko/jscpd is an open‑source copy‑and‑paste detection tool that scans programming source files to identify duplicated code fragments across projects. It supports a wide range of languages, generates detailed reports, and can be integrated into CI pipelines to help developers reduce technical debt and improve code quality.
