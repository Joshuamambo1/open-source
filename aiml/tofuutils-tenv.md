## 📝 Summary

Here's a brief summary for the open-source project:

TENV is an open-source version manager designed for Terraform, Terragrunt, Terramate, and Atmos. This project provides a convenient way to manage and switch between different versions of these infrastructure-as-code tools, streamlining the development process for DevOps and infrastructure teams. By centralizing version control, TENV aims to improve collaboration and reduce configuration errors.
