## 📝 Summary

### English

finos/git-proxy is an open‑source tool that sits in front of a Git server to enforce custom push‑time protections and policy checks, helping organizations prevent unwanted or non‑compliant code from entering their repositories. By intercepting Git operations, it enables fine‑grained security controls—such as commit‑message validation, secret scanning, and custom rule enforcement—while remaining compatible with existing Git workflows.

### Русский

finos/git-proxy — это open‑source решение, позволяющее внедрять кастомные политики и механизмы защиты при выполнении push‑операций в Git‑репозитории. Проект работает как прокси‑сервер, перехватывая запросы и применяя пользовательские правила (например, проверку подписей, сканирование уязвимостей или контроль доступа), тем самым повышая безопасность процесса разработки. Он легко интегрируется в существующие CI/CD‑конвейеры и поддерживает расширяемую конфигурацию правил.
