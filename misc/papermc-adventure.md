## 📝 Summary

### English

PaperMC Adventure is a lightweight, platform‑agnostic UI toolkit designed for Minecraft: Java Edition servers and mods. It provides a flexible API for building rich text components, handling colors, styles, click/hover events, and serialization to Minecraft’s native chat format. By abstracting the underlying chat system, Adventure enables developers to create consistent, expressive user interfaces across different Minecraft server implementations.
