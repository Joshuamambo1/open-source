## 📝 Summary

The llm-server project is an open-source tool designed to optimize the performance of LLaMA-based AI models. It automatically detects available GPUs, strategically optimizes model-occupancy estimation (MoE) placement, and provides crash recovery features. This smart launcher aims to streamline the deployment and management of LLaMA models for efficient AI development and research.
