## 📝 Summary

The VKD3D-Proton project is an open-source fork of the VKD3D engine, specifically tailored to support Direct3D 12 implementation within the Proton compatibility layer. This fork focuses on extending VKD3D's capabilities to enhance compatibility with Windows games on Linux platforms. By doing so, it aims to provide a smoother gaming experience for users running Windows games on Linux via Proton.
