## 📝 Summary

Tantivy-py is an open-source Python wrapper for the Tantivy search engine. It provides a seamless interface to leverage Tantivy's full-text search capabilities within Python applications. This project enables developers to build efficient and scalable search functionalities with the ease of a Pythonic API.
