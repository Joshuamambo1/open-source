## 📝 Summary

Here's a 2-3 sentence summary for the UML-MCP Server project:

The UML-MCP Server is an open-source tool that utilizes the Model Context Protocol (MCP) to generate various types of UML diagrams. It enables users to create diagrams through natural language descriptions or by writing syntax for popular diagramming languages like PlantUML, Mermaid, and Kroki. This versatile server facilitates efficient UML diagram creation and collaboration.
