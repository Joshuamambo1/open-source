## 📝 Summary

DMontgomery40’s **mcp-canvas-lms** is a lightweight MCP (Microservice Control Protocol) server that wraps the Canvas LMS API, providing a unified backend for managing Canvas resources. Version 2.2 bundles 54 ready‑to‑use tools for creating, updating, and querying courses, assignments, enrollments, and grades, making it easy to automate and integrate Canvas workflows in your applications. It’s ideal for developers looking for a programmatic, open‑source solution to interact with Canvas without dealing directly with the raw API.
