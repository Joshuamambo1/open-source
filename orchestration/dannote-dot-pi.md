### 中文

**项目简介**  
dannote/dot‑pi 是一个精选的 Pi 生态包，提供扩展插件、技能、提示快捷方式和规则集合，帮助把零散的 Prompt 与工具编排成可重复、可组合的智能体工作流。

**价值**  
- **工作流标准化**：将多智能体的交互、工具调用和记忆管理抽象为统一的规则和接口，降低重复实现成本。  
- **快速原型**：内置常用 Prompt 模板和技能库，可在几行代码或 CLI 命令内搭建完整的多智能体协作流程。  
- **可扩展性**：通过 TypeScript SDK、REST API 和 CLI 三种接入方式，便于在现有系统中嵌入或自行扩展新工具。

**典型接入方式**  
1. **SDK**：`npm i @dot-pi/sdk` → 在 TypeScript/JavaScript 项目中直接调用 `DotPi.createAgent()`、`DotPi.runWorkflow()` 等方法。  
2. **CLI**：`npx dot-pi run <workflow.yaml>` → 适合 CI/CD、脚本化任务或本地调试。  
3. **REST API**：启动 `dot-pi server`，通过 `POST /workflows`、`GET /agents/:id` 等端点与任意语言的服务交互。
