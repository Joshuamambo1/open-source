## 📝 Summary

Here's a brief summary of the open-source project:

DevoxxGenie is a plugin for IntelliJ IDEA that leverages local and cloud-based Large Language Models (LLMs) to enhance code review, testing, and explanation. With its latest version, it also supports Spec-Driven Development with Command Line Interface (CLI) Runners. This innovative tool streamlines the development process by utilizing AI and ML capabilities to improve project code quality and efficiency.
