## 📝 Summary

Here's a brief summary of the project:

Graph Explorer is an open-source, React-based web application that enables users to visually explore complex data relationships in property graph and RDF data without requiring graph query knowledge. This intuitive tool facilitates connections between data, empowering users to uncover hidden insights and navigate intricate data structures with ease. By leveraging a user-friendly interface, Graph Explorer bridges the gap between data visualization and graph querying.
