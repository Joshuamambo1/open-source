## 📝 Summary

### English

Apache Hadoop is an open‑source framework that enables distributed storage and processing of large data sets across clusters of commodity hardware. It provides scalable, fault‑tolerant solutions through its core components—HDFS for reliable file storage and MapReduce (or YARN) for parallel computation. The project serves as a foundational platform for big‑data analytics and ecosystem tools.

### Русский

Apache Hadoop — это открытая платформа для распределённого хранения и обработки больших объёмов данных, реализующая модель MapReduce и файловую систему HDFS. Она позволяет масштабировать вычислительные кластеры от нескольких узлов до тысяч, обеспечивая отказоустойчивость и высокую производительность при работе с массивными наборами данных.
