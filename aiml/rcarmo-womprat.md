## 📝 Summary

Womprat is an open-source, single-binary client for Windows that provides access to machines on a Tailnet, a decentralized, censorship-resistant network. This project is designed to facilitate secure communication and collaboration on the Tailnet platform. By utilizing Womprat, users can easily connect to and interact with other machines on the network.
