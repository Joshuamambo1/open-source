## 📝 Summary

Persian‑Calendar is an open‑source Android library that provides a fully localized Persian (Jalali) calendar widget and date‑handling utilities. It offers seamless integration with Android apps, supporting Persian numerals, right‑to‑left layouts, and conversion between Gregorian and Jalali dates. The project is lightweight, easy to customize, and ideal for developers needing native Persian calendar support on mobile devices.
