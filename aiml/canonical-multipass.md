## 📝 Summary

### English

Canonical Multipass is an open‑source tool that simplifies the creation, management, and orchestration of lightweight Ubuntu virtual machines across macOS, Windows, and Linux. It provides a command‑line interface for launching disposable or persistent instances, enabling developers to spin up reproducible development environments quickly. Although primarily a virtualization utility, its automation capabilities make it useful for AI/ML workflows that require isolated Ubuntu environments.
