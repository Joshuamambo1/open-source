ETPNav is an open-source project that implements Evolving Topological Planning for Vision-Language Navigation in Continuous Environments, a cutting-edge approach published in TPAMI 2024. This AI/ML-based solution enables efficient navigation in complex, dynamic environments by leveraging topological planning and evolving strategies. The project aims to advance the field of vision-language navigation and provide a valuable resource for researchers and developers.
