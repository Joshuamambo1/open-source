## 📝 Summary

### English

LFortran is an open‑source modern Fortran compiler that aims to provide a fast, interactive, and extensible development experience. It supports the latest Fortran standards, offers Python‑like scripting for metaprogramming, and integrates with AI/ML workflows through easy interfacing with scientific Python libraries. The project serves as the primary repository for contributors and users to build, test, and extend the compiler’s capabilities.
