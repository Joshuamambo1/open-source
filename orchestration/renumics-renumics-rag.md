## 📝 Summary

Renumics‑RAG is an open‑source visual interface designed to monitor and explore the inner workings of a Retrieval‑Augmented Generation (RAG) assistant. It provides real‑time dashboards, traceability of retrieved documents, and performance metrics, helping developers orchestrate and debug AI/ML pipelines that combine external knowledge sources with generative models.
