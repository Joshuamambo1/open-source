The crowdsec-web-ui project is an open-source web interface designed to provide a modern and responsive management experience for CrowdSec alerts and decisions. This project aims to simplify the process of monitoring and responding to security threats in real-time, making it easier for users to stay on top of their security posture. Developed with a focus on AI/ML, frontend, and backend technologies, it offers a comprehensive solution for CrowdSec users.
