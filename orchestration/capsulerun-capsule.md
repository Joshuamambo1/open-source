## 📝 Summary

Capsule is an open-source project that enables the safe execution of untrusted code within isolated WebAssembly sandboxes. This allows for secure and scalable hosting of AI/ML models, microservices, and other code, reducing the risk of malicious activity and data breaches. By leveraging WebAssembly, Capsule provides a flexible and portable way to orchestrate isolated code execution.
