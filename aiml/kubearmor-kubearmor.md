## 📝 Summary

KubeArmor is an open-source runtime security enforcement system that hardens and sandboxes workloads, providing least-permissive policies to prevent security breaches. Leveraging Linux Security Modules (LSMs) such as LSM-BPF and AppArmor, KubeArmor makes it easy to implement robust security controls. This project empowers DevOps teams to secure their cloud-native applications and workloads with ease.
