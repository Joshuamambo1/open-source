## 📝 Summary

Bforartists is an open-source 3D creation software forked from Blender, focusing on enhancing the user interface to provide a more intuitive and streamlined experience. This project aims to improve the usability and accessibility of Blender's UI, making it easier for users to create 3D content. By refining the UI, Bforartists seeks to bridge the gap between professional artists and beginners in the field of 3D modeling and animation.
