## 📝 Summary

afragen/git‑updater is an open‑source WordPress plugin that enables automatic updates for themes and plugins hosted on GitHub, Bitbucket, GitLab, and Gitea. It integrates with WordPress’s native update system, allowing site administrators to manage version control‑based extensions without manual downloads. The project is community‑maintained and suitable for developers who want seamless Git‑driven updates within WordPress.
