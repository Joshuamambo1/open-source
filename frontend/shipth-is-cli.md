## 📝 Summary

A lightweight command‑line tool that automates the entire build and publishing pipeline for Godot games targeting iOS App Store and Google Play, eliminating the need for Xcode or Android Studio. It streamlines asset packaging, code signing, and store submission directly from the terminal, making mobile deployment fast and developer‑friendly. Suitable for frontend and dev‑tool workflows, it works cross‑platform and is open‑source.
