## 📝 Summary

Here's a brief summary of the Agentica project:

Agentica is an open-source, lightweight asynchronous framework for building AI agents in Python. Designed to support a range of applications, including tool calls, Relational Action Graph (RAG), multi-agent systems, and Multi-Criteria Planning (MCP), Agentica enables efficient and scalable development of intelligent systems. This async-first framework is ideal for those seeking to create sophisticated AI agents with minimal overhead.
