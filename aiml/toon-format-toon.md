## 📝 Summary

Toon (Token‑Oriented Object Notation) is an open‑source, schema‑aware serialization format that packs JSON‑style data into a compact, human‑readable representation optimized for large‑language‑model prompts. The repository provides the full specification, benchmark results comparing TOON to plain JSON and other formats, and a TypeScript SDK for parsing, validating, and generating TOON data. It targets AI/ML developers who need efficient, token‑light data interchange for prompt engineering and model‑side processing.
