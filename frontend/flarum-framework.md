### English

Flarum Framework is a lightweight, modern PHP‑based platform for creating fast, extensible forums. It provides a clean, component‑driven frontend built with Mithril.js, offering a highly customizable UI and a robust API for developers to extend functionality. Ideal for building vibrant online communities with minimal overhead.
