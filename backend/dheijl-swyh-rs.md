## 📝 Summary

dheijl/swyh-rs is a Rust implementation of “Stream What You Hear,” a tool that captures live audio from a system’s output and streams it over the network. Built as a backend service, it leverages Rust’s performance and safety features while staying compatible with the original SWYH concept. The project is open‑source, making it easy to integrate, customize, or extend for audio‑streaming applications.
