## 📝 Summary

Ascii-chat is an open-source project that enables video conferencing directly in the terminal, offering a unique and innovative way to communicate remotely. This project combines the capabilities of a video chat with the familiarity of a command-line interface, providing users with a versatile option for virtual meetings and collaborations. By utilizing terminal-based functionality, it also aims to enhance security through its decentralized nature.
