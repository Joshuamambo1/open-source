## 📝 Summary

### English

MAVLink is a lightweight communication protocol widely used in UAV and robotics applications. The **rust-mavlink** project provides a pure‑Rust implementation of this protocol, offering tools for encoding, decoding, and handling MAVLink messages with idiomatic, type‑safe Rust APIs. It enables developers to integrate MAVLink support into Rust‑based ground stations, autopilots, and other embedded or desktop systems.

### Русский

Библиотека **rust-mavlink** реализует протокол MAVLink для языка Rust, предоставляя удобный и типобезопасный интерфейс для обмена телеметрией и командными сообщениями с беспилотными системами. Проект поддерживает как версии MAVLink 1, так и MAVLink 2, генерируя структуры сообщений из XML‑описаний и позволяя легко интегрировать их в асинхронные и синхронные приложения. Благодаря открытой лицензии и активному сообществу, библиотека подходит как для прототипирования, так и для производства надёжных решений в сфере робототехники и дронов.
