## 📝 Summary

Miri is an open-source project that enhances the miracle-wm window manager by making it scrollable. This plugin aims to improve the user experience by providing a more intuitive and navigable interface for managing windows. By adding scrolling functionality, users can more easily access and manage their applications, increasing overall productivity.
