**项目简介**  
keyuchen21/agentic‑engineering‑handbook 是一份面向 OpenAI、Claude、MCP、Harness、Evals 与生产级 Agent 系统的完整学习路线图，兼具实践代码与实现细节，帮助开发者快速原型化或审查区块链工作流。

**价值**  
- **快速落地 Web3 场景**：提供可直接运行的示例，支持钱包、DeFi、跨链等常见区块链业务的原型搭建与调试。  
- **透明实现细节**：公开 API/SDK/CLI 调用方式、语言元数据以及专题拆解，便于开发者审计和二次开发。

**典型接入方式**  
1. **克隆仓库**：`git clone https://github.com/keyuchen21/agentic-engineering-handbook.git`  
2. **安装依赖**（Python 环境）：`pip install -r requirements.txt`  
3. **调用示例**：使用仓库提供的 CLI 或 SDK，直接运行 `python examples/run_agent.py --config configs/web3_wallet.yaml`，即可启动一个完整的区块链 Agent 流程。  
4. **自定义扩展**：通过修改 `configs/*.yaml` 或在 `src/agents/` 下添加自定义模块，即可对接自己的链上业务或第三方服务。
