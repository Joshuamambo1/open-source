## 📝 Summary

ConorWilliams/libfork is an open-source, high-performance tasking library written in C++20, utilizing coroutines to provide a lock-free and wait-free framework for concurrent programming. This library is designed to optimize complex tasks, such as those found in AI and machine learning applications. It leverages continuation-stealing to minimize context switching and maximize efficiency.
