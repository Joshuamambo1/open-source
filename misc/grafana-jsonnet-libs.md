## 📝 Summary

### English

Grafana Labs' **jsonnet-libs** is a collection of reusable Jsonnet libraries that simplify the generation of Grafana dashboards, data sources, and related configuration files. By providing modular, well‑documented components, the project enables developers to programmatically compose, version‑control, and templatize Grafana assets across environments. It is released as open‑source software under the MIT license and can be integrated into any Jsonnet workflow.
