## 📝 Summary

The borsa-mcp project is an open-source, MCP (Market Data Protocol) server that provides real-time stock exchange and fund data for both the Turkish and American markets. This project enables developers to access a wide range of financial data, facilitating the development of trading applications and AI/ML models. By utilizing this MCP server, users can streamline their data retrieval and analysis processes.
