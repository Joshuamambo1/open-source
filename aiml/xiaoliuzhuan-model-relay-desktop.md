## 📝 Summary

Here's a brief summary for the open-source project:

The xiaoliuzhuan/model-relay-desktop project is an open-source local desktop relay that enables users to route OpenAI and Anthropic models through switchable proxy config groups, improving model access and flexibility. This desktop application serves as a local relay for Trae and Cursor, simplifying the process of interacting with these models. With model-relay-desktop, users can easily switch between different proxy configurations, enhancing their overall AI/ML development experience.
