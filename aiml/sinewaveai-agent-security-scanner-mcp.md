## 📝 Summary

The agent-security-scanner-mcp project is an open-source, comprehensive security scanner designed for AI coding agents. It leverages advanced features such as prompt injection firewall, package hallucination detection, and 1000+ vulnerability rules with integrated AST (Abstract Syntax Tree) and taint analysis capabilities for robust security checks. Additionally, it offers auto-fix functionality to streamline the remediation process.
