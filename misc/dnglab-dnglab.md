## 📝 Summary

### English

dnglab is an open‑source utility that converts raw camera files from a wide range of proprietary formats into the standardized DNG (Digital Negative) format. By preserving all original sensor data and metadata, it enables photographers and developers to work with a universal, archival‑friendly file type without relying on vendor‑specific software. The project is lightweight, command‑line driven, and suitable for integration into imaging pipelines or batch‑processing workflows.
