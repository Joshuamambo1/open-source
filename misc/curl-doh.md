## 📝 Summary

The curl/doh project is an open-source, stand-alone application designed for performing DNS-over-HTTPS (DoH) name resolutions and lookups. It allows users to query DNS information using HTTPS protocols, enhancing privacy and security by encrypting DNS queries. This application can be used for testing, development, or troubleshooting purposes.
