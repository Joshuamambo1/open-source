## 📝 Summary

### English

Symphonia is a pure‑Rust library that provides fast, safe demultiplexing of multimedia container formats, metadata (tag) extraction, and audio decoding. It supports a wide range of codecs and file types without relying on external C dependencies, making it ideal for building cross‑platform media applications in Rust. The project is open‑source and aims to offer a modular, extensible foundation for audio and video processing tasks.
