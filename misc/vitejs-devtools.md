## 📝 Summary

Vite DevTools is an open‑source extension that provides a visual interface for inspecting and debugging Vite (and its Rollup‑compatible successor, Rolldown) projects. It offers real‑time insights into module graphs, dependency trees, server status, and build performance, helping developers diagnose issues and optimize their Vite‑based workflows.
