## 📝 Summary

DeepEye is an open‑source autonomous data agent system developed by the HKUSTDial group. It combines advanced AI/ML techniques to automatically ingest, process, and analyze diverse datasets, enabling users to query and retrieve insights without manual data engineering. The platform is designed for extensibility, allowing developers to plug in custom models and connectors for a wide range of data sources.
