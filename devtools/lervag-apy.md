## 📝 Summary

The lervag/apy project is an open-source command-line interface (CLI) script designed to facilitate interaction with local Anki collections. This tool allows users to manage and manipulate their Anki data in a convenient and efficient manner. With apy, users can perform various tasks, such as importing and exporting cards, checking deck statistics, and more, all from the comfort of their terminal.
