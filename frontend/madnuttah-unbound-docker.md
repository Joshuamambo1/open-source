## 📝 Summary

### English

The **madnuttah/unbound-docker** project provides a minimal, distroless Docker image of the Unbound DNS resolver built on Alpine Linux, optimized for security, privacy, performance, and a tiny footprint. It is designed with Pi‑hole integrations in mind, making it ideal for lightweight, production‑grade DNS filtering and caching in DevOps and infrastructure environments.

### Русский

Проект **madnuttah/unbound-docker** представляет собой минимальный, «distroless» Docker‑образ Unbound, построенный на Alpine Linux и оптимизированный для высокой безопасности, конфиденциальности и производительности, при этом сохраняя небольшой размер. Образ специально адаптирован для работы совместно с Pi‑hole, что делает его удобным решением для DNS‑фильтрации в домашних и небольших корпоративных сетях. Подходит для DevOps‑инфраструктуры, где важны быстрый запуск, низкое потребление ресурсов и строгие требования к безопасности.
