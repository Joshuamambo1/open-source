## 📝 Summary

**Show HN: Agent Postmortem Skill** is an open‑source toolkit that lets AI coding agents generate a detailed “postmortem” of their own execution—explaining the steps they took, the decisions they made, and the code they produced. By forcing agents to document and justify their work, the project aims to improve transparency, debuggability, and trustworthiness of autonomous code‑generation systems. It’s positioned for developers and researchers working on AI/ML‑driven coding assistants.
