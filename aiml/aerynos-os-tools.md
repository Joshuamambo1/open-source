AerynOS/os-tools is an open-source project that provides a comprehensive set of tools for building and managing operating system environments. At its core, it includes moss, a system state and package manager, and boulder, a build tool that leverages moss-format build processes. This project is designed to support the development and deployment of AI/ML and frontend applications.
