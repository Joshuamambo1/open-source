## 📝 Summary

CryFS is an open‑source, user‑space file system that encrypts data on the fly, allowing you to store confidential files securely in the cloud or on any untrusted storage medium. It splits directories into small encrypted blocks, hides file sizes and metadata, and provides transparent, POSIX‑compatible access while ensuring strong cryptographic guarantees. Designed for ease of use, CryFS lets developers and end‑users protect sensitive information without altering existing workflows.
