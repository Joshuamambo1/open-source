## 📝 Summary

godot‑gdunit‑labs/gdUnit4 is an open‑source, embedded unit‑testing framework tailored for Godot 4 that supports both GDScript and C#. It provides a full test‑driven development workflow with an in‑engine test inspector, rich assertion libraries, mocking utilities, and dedicated support for testing scenes and game logic. Ideal for developers seeking integrated, automated testing tools within the Godot editor.
