## 📝 Summary

knowsuchagency/mcp2cli is a developer‑focused tool that instantly converts any MCP, OpenAPI, or GraphQL endpoint into a fully functional command‑line interface. It works at runtime without requiring code generation or manual scaffolding, letting you explore, test, and script API calls directly from the terminal. Ideal for backend developers and AI/ML engineers who need quick, code‑free access to service operations.
