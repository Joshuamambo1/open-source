## 📝 Summary

This open-source project provides a Karpathy-style personal wiki for Large Language Models (LLMs), designed to be used with Claude Code and compatible with any file-editing AI agent. The wiki utilizes markdown and frontmatter for organization and flexibility. It eliminates the need for a vector database (DB) or Retrieval Augmented Generator (RAG) setup.
