## 📝 Summary

"Terminal-Doom is an open-source project that brings the classic game DOOM to modern terminals, allowing users to experience the iconic first-person shooter in a text-based environment. This project utilizes ASCII art and simple keyboard input to recreate the intense gameplay of DOOM, making it accessible on any terminal-emulating device. With Terminal-Doom, gamers can relive the nostalgia of DOOM in a unique and minimalist way."
