## 📝 Summary

### English

shishkin/astro-pagefind is an open‑source plugin that seamlessly integrates the Pagefind static‑site search engine with Astro projects. It provides easy configuration and automatic indexing of Astro-generated pages, enabling fast, client‑side full‑text search without requiring a separate backend. This lightweight tool is ideal for developers looking to add powerful, zero‑maintenance search functionality to any Astro site.
