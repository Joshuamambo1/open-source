## 📝 Summary

Here's a 2-3 sentence summary for the open-source project:

CLAI is an open-source command line artificial intelligence tool that serves as a local Large Language Model (LLM) context-feeder. It provides a simple interface for interacting with LLMs, enabling users to feed context to these models from the comfort of their terminal. This project aims to make working with LLMs more accessible and convenient for developers and researchers.
