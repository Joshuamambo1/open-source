## 📝 Summary

The sdsrss/code-graph-mcp project is an open-source backend server that enables the creation and management of an Abstract Syntax Tree (AST) knowledge graph for Claude Code. This graph allows for advanced features such as semantic search, call graph traversal, HTTP route tracing, and impact analysis across 10 languages, which are automatically indexed using Tree-sitter.
