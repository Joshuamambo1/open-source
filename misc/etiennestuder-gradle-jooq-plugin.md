## 📝 Summary

### English

A lightweight Gradle plugin that seamlessly integrates jOOQ code generation and runtime support into your build process. It automates the configuration of jOOQ, handling dependency management, generation tasks, and optional integration with the Gradle Java plugin, making it easy to incorporate type‑safe SQL querying into Java projects.
