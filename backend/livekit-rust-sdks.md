## 📝 Summary

### English

This open-source project, livekit/rust-sdks, provides Realtime and Server SDKs for the Rust programming language, enabling developers to build scalable and efficient applications that utilize LiveKit. The SDKs facilitate the creation of real-time communication, video conferencing, and collaboration features in Rust-based backend applications. This project is ideal for developers looking to integrate LiveKit's functionality into their Rust-powered backend infrastructure.
