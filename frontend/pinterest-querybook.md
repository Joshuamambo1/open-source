## 📝 Summary

Querybook is an open-source project that simplifies big data querying by combining collocated table metadata with a user-friendly notebook interface. This intuitive tool enables data analysts and scientists to efficiently explore and analyze data, streamlining their workflow and enhancing productivity. With Querybook, users can easily access, manipulate, and visualize large datasets in a seamless and interactive environment.
