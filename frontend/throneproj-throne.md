## 📝 Summary

The Throne project is an open-source, cross-platform GUI proxy utility designed to provide a user-friendly interface for managing proxy settings. Empowered by the sing-box technology, Throne aims to simplify the process of configuring and utilizing proxies across various platforms. This frontend-focused tool offers an intuitive way to interact with proxy services.
