## 📝 Summary

Cc-bridge is an open-source project that enables two Claude Code windows to communicate and share data via a single JSONL file. This innovation allows users to integrate and synchronize information between multiple instances of the Claude Code interface, fostering collaboration and streamlined workflows. By leveraging a JSONL file, Cc-bridge simplifies data exchange and promotes efficient interactions between Claude Code windows.
