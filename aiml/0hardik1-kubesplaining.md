## 📝 Summary

Kubesplaining is an open-source CLI tool designed to map RBAC (Role-Based Access Control) privilege-escalation paths in Kubernetes environments. This tool helps identify potential security risks and vulnerabilities in Kubernetes clusters by visualizing the paths that users can take to escalate their privileges. By providing a clearer view of these paths, Kubesplaining facilitates better security management and compliance in Kubernetes deployments.
