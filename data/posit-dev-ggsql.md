## 📝 Summary

The ggsql project is an open-source SQL extension that enables declarative data visualization based on the Grammar of Graphics. This extension allows users to create data visualizations using standard SQL queries, simplifying the process of data visualization and exploration. By leveraging the Grammar of Graphics, ggsql makes it easy to create a wide range of visualization types.
