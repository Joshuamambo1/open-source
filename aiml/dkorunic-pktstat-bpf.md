## 📝 Summary

Here's a brief summary of the project:

pktstat-bpf is an open-source, eBPF-based tool that provides real-time monitoring and reporting of Ethernet interface traffic. It leverages TC, XDP, KProbe, and CGroup eBPF to offer a comprehensive view of network traffic, making it a valuable asset for DevOps and infrastructure professionals. This tool enables efficient network traffic analysis and optimization.
