## 📝 Summary

Here's a brief summary of the open-source project:

PerformanceMonitor is a free, open-source tool that provides comprehensive SQL Server performance monitoring capabilities, featuring 32 collectors, real-time alerts, and a graphical plan viewer. It also supports AI analysis with its MCP server, enabling users to leverage machine learning for informed decision-making. Compatible with SQL Server 2016-2025, Azure SQL, and AWS RDS, this tool helps users optimize database performance.
