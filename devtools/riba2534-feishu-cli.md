## 📝 Summary

Here is a brief summary of the feishu-cli project:

Feishu-cli is an open-source command-line tool for the Feishu open platform, providing a comprehensive interface to manage Feishu documents, knowledge bases, spreadsheets, messages, calendars, and tasks. Its core feature is the seamless conversion between Markdown and Feishu documents without losing any data. This tool simplifies interactions with Feishu through a simple and intuitive command-line interface.
