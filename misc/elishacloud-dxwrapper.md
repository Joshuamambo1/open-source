## 📝 Summary

The dxwrapper project is an open-source solution designed to resolve compatibility issues with older games on Windows 10 and 11 by wrapping DirectX dynamic link libraries (dlls). This wrapper also enables the loading of custom libraries with the .asi file extension into game processes, providing a flexible and customizable solution for gamers. By addressing compatibility issues, dxwrapper aims to improve the gaming experience for users of classic titles on modern Windows operating systems.
