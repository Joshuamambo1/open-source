## 📝 Summary

A lightweight Windows batch file that automatically discovers Android devices on the local network and establishes an ADB (Android Debug Bridge) connection over LAN. By scanning the subnet for devices with ADB enabled, the script streamlines wireless debugging without manual IP entry, making it a handy tool for developers and power users.
