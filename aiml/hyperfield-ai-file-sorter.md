## 📝 Summary

Here's a brief summary for the hyperfield/ai-file-sorter project:

The hyperfield/ai-file-sorter is an open-source, cross-platform desktop application that leverages artificial intelligence (AI) to efficiently organize and rename files based on their content. This innovative tool enables users to take control of their file management, supporting both local and remote Large Language Models (LLMs) for accurate file categorization. With its preview-based workflow, users can visually verify changes before committing them.
