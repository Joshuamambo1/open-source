## 📝 Summary

Trunks is an open-source project that allows users to store and manage their Git repositories on their own storage solutions, rather than relying on external services. This project aims to provide a flexible and customizable alternative for Git repository hosting, giving users more control over their data. By leveraging self-hosted storage, users can maintain the security and ownership of their repositories.
