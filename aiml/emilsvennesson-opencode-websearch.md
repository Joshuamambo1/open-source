## 📝 Summary

This project, emilsvennesson/opencode-websearch, is an open-source native web search tool for OpenCode that leverages the model's built-in search capabilities. It eliminates the need for extra API keys or search services, making it a convenient and self-contained solution. Inspired by Claude Code's WebSearch tool, this project empowers users to perform web searches directly within the OpenCode environment.
