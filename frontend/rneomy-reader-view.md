## 📝 Summary

The rNeomy/reader-view project is an open-source extension that allows users to access Firefox's built-in reader view functionality through a convenient right-click context menu option. This enables users to quickly and easily strip away distracting web page elements and focus on the article's content. By integrating reader view into the context menu, this extension streamlines the reading experience in Firefox.
