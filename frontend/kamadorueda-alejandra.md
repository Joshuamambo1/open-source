## 📝 Summary

Alejandra is an uncompromising Nix code formatter that ensures consistent and readable Nix code across projects. This open-source project is designed to maintain code quality by automatically reformatting Nix code to adhere to a set of predefined rules. By enforcing a consistent coding style, Alejandra streamlines development and collaboration on Nix-based projects.
