## 📝 Summary

Here's a brief summary of the Lexy project:

Lexy is an open-source Command Line Interface (CLI) tool that fetches programming tutorials from "Learn X in Y Minutes" directly into your terminal. This lightweight tool enables quick searching, learning, and referencing of code examples without disrupting your workflow. With Lexy, developers can easily access a wealth of programming knowledge from within their terminal.
