## 📝 Summary

### English

Universal Tool Calling Protocol (UTCP) provides an open standard that enables AI agents to invoke any API directly, eliminating the need for custom middleware. This repository offers the official Python implementation of UTCP, delivering a lightweight, easy‑to‑integrate library for building AI‑driven back‑end services and developer tools. It’s designed for AI/ML applications that require seamless, secure, and language‑agnostic API calls.
