## 📝 Summary

Here's a brief summary of the OpenGUI project:

OpenGUI is an open-source, multi-platform command center that integrates with popular AI coding tools such as OpenCode, Claude Code, Codex, and Pi. This desktop and web application enables users to manage and conduct AI coding sessions across multiple projects, featuring real-time streaming chat, prompt queueing, and model switching capabilities. Additionally, it includes MCP (Machine Code Programming) tools for efficient AI coding and development.
