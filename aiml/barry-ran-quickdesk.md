## 📝 Summary

### English

QuickDesk is an open‑source, AI‑native remote‑desktop solution that lets any AI agent view and control a remote computer through its built‑in Model Context Protocol (MCP) server. By combining a lightweight frontend with a robust backend, it provides a free platform for developers to integrate AI‑driven interaction into desktop environments.
