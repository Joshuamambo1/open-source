stepfunc/dnp3 is a Rust library that provides a full implementation of the DNP3 (IEEE 1815) protocol, targeting reliable communication in SCADA and industrial automation systems. It offers idiomatic foreign‑function interfaces for C, .NET, C++, and Java, allowing developers to integrate the high‑performance Rust core into a wide range of existing codebases. The project is open‑source and positioned as a versatile, cross‑platform solution for building DNP3‑enabled applications.
