## 📝 Summary

GNOME/gparted is an open-source project that manages and edits disk partitions. It provides a user-friendly interface for users to view, create, delete, resize, and copy disk partitions, making it an essential tool for disk management and maintenance. This project is a mirror of the original gparted repository hosted on GitLab.
