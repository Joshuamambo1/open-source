## 📝 Summary

Neur0map’s **glazepkg** is a lightweight web‑based tool that aggregates and displays all of the packages installed on a system in a single, easy‑to‑navigate interface. Designed for developers, it provides quick insights into version numbers, dependencies, and package metadata, streamlining front‑end project maintenance and debugging.
