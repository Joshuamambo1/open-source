## 📝 Summary

Lekssays/codebadger is an open-source Model Context Protocol (MCP) server that offers static code analysis capabilities through Joern's Code Property Graph (CPG). This containerized server supports a wide range of programming languages, including Java, C/C++, JavaScript, and several others. By leveraging CPG, codebadger provides a robust and efficient solution for developers to analyze and improve their code.
