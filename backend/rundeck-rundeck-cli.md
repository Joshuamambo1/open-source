## 📝 Summary

### English

The **rundeck-cli** project provides a lightweight, cross‑platform command‑line interface for interacting with Rundeck servers. It enables users to execute jobs, manage projects, retrieve execution logs, and perform other Rundeck API actions directly from the terminal, making automation and scripting of Rundeck workflows straightforward. Ideal for developers and DevOps engineers, it serves as a powerful backend tool for integrating Rundeck into CI/CD pipelines and other tooling.
