## 📝 Summary

Here's a brief summary for the open-source project:

LagerNVS is an open-source implementation of the "LagerNVS Latent Geometry for Fully Neural Real-time Novel View Synthesis" method, presented at CVPR 2026. This project enables real-time novel view synthesis by leveraging latent geometry to generate photorealistic images from a single input view. By utilizing a fully neural approach, LagerNVS facilitates efficient and accurate novel view synthesis for a wide range of applications.
