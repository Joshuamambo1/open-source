## 📝 Summary

This open-source project aims to bring the classic first-person shooter game Doom to the GL-iNet GL-BE10000 Slate 7 Pro travel router. By porting the game to this unique device, users can now experience Doom on a platform that was not initially intended for gaming. This project showcases the potential for creative repurposing of hardware.
